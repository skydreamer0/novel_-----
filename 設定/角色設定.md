# 角色設定

## 主角：蘇晨

### 基本資料
| 項目 | 內容 |
|-----|------|
| 年齡 | 23歲 |
| 身份 | 臨海大學畢業生 / 天衡局特邀外聘人員（偽裝快遞員） |
| 性格 | 表面隨和、內心堅韌、護短果斷、實戰派、擅長藏拙 |
| 外貌 | 中等偏上，氣質因修煉變得出塵，皮膚有如青玉 |
| 家庭 | 孤兒，在福利院長大，視院長王奶奶為親人 |


### 性格特質
- **務實**：不會無腦衝動，先評估再行動
- **護短**：對在乎的人絕不退讓
- **不裝聖母**：敵人想殺他，他不會手軟
- **幽默感**：壓力下能開玩笑，保持理智
- **底線清晰**：不欺負弱者，但也不允許被人踩在腳下

### 金手指：萬界交易系統
- **核心能力**：與平行世界的存在進行等價交易
- **目前版本**：v2.0
- **新增功能**：
  - **存儲空間**：1立方米（不可存儲生命體）
  - **跨界搜索**：主動搜索特定需求的交易對象（冷卻24h）
  - **解析偵查**：探查目標靈氣結構與基礎資料
- **交易對象**：
  - 靈草世界：李小苗（好友）
  - 劍修世界：葉孤楓（目前受困於萬劍崖，急需生機物品）
- **獨特之處**：現實世界的某些「普通物品」在異世界極為珍貴（例如：可樂是「仙露」，泡麵是「龍鬚百味羹」，五金工具是「開山神物」）

### 修煉與戰鬥
- **境界**：二階·凝氣（初階）
- **功法**：長青吐納術（系統優化版，修煉速度+300%，自動吸納靈氣）
- **靈技**：
  - **感知**：一階覺醒能力，範圍偵測。
  - **長青指**：單點貫穿技，穿透力極強。
- **戰績**：單殺二階巔峰「三首幽冥犬」。
- **保密等級**：天衡局「二等搜查權」。

### 成長路線
```
快遞員 → 覺醒者（二階） → 天衡局外聘（獲二等搜查權） → 實戰天才 → 獨立勢力...
```
