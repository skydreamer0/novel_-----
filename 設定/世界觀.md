# 世界觀設定：《萬界交易商》

## 核心概念

現代都市表面之下，隱藏著一個由「覺醒者」構成的暗面世界。少數人類因未知原因覺醒了超凡能力，被國家秘密機構「天衡局」管理。而在這一切之上，還存在著更高維度的「萬界」——無數平行世界組成的龐大宇宙。

- 地標：臨海大學、東港金融區、舊城區「蛇巷」、臨海第一醫院

## 萬界設定

主角的系統能連結的世界會隨等級逐步解鎖：

| 等級 | 可連結世界 | 特色資源 |
|-----|-----------|---------|
| Lv.1-10 | 靈草世界 | 低階靈藥、基礎功法 |
| Lv.11-30 | 劍修世界 | 劍法、劍意碎片、靈石 |
| Lv.31-50 | 妖獸大陸 | 妖核、血脈精華、馴獸術、**基因藥劑原材料** |
| Lv.51-80 | 符籙仙域 | 符籙、陣法、仙器毛坯 |
| Lv.81-100 | 混沌古界 | 上古功法、天材地寶、神器碎片 |
| Lv.100+ | ??? | 未知 |

### 妖獸大陸（當前探索中）
- **環境**：原始森林、弱肉強食，空氣中含有狂暴的「妖氣」，普通人類難以生存。
- **勢力**：以部落形式存在的土著人類（如黑岩部），以及無數強大的妖獸族群。
- **特殊地點**：
  - **迷霧森林**：黑岩部所在地，外圍區域。
  - **亡靈沼澤**：禁地，隱藏著二十年前「聯合科考隊」的遺跡（疑似避難所/實驗室）。

每個世界都有自己的文明、種族和規則，主角可以通過系統與該世界的存在進行交易。

## 特殊物品與現象

### 源質膠囊（Source Quality Capsule）
- **來源**：「蛻變者」組織開發的人工作品。
- **效果**：植入妖獸或人體內，通過燃燒生命潛力強行提升戰力，使其產生瘋狂的進攻性。
- **特徵**：幽暗紅光的結晶，死後化作黑霧消失（或被收回）。

### 自動化防禦體系（Automated Defense System）
- **型號**：「守望者-II 型」靈能哨戒炮。
- **原理**：結合地球「金屬風暴」技術與劍修世界「連珠弩」原理。
- **能源**：壓縮靈石 / 符文強化。
- **火力**：每分鐘 6000 發鎢合金尖刺（破甲符文附魔），可輕易撕碎五階以下妖獸。
- **製造者**：蘇晨設計，林若溪（天衡局）技術支援/定製。

### 聯合科考隊（20年前 / 2025年）
- **官方名稱**：可能為「聯合國異常生物調查局」前身或某秘密部隊。
- **標誌**：DNA 雙螺旋 + 北斗七星。
- **領導者**：蘇天策（蘇晨之父，代號「少校」）。
- **足跡**：曾在妖獸大陸「亡靈沼澤」建立高科技基地（代號「燈塔」），擁有能夠識別基因的機器人守衛。
- **發現**：確認妖獸大陸是某高維文明的「生物兵器培養皿」。
- **黑星計畫（Project Black Star）**：相關絕密檔案，涉及封印某種「汙染源」。

### 異界源質
- 來自空間裂縫對面的未知能量物質，具備極強的侵蝕性與催化性，是系統升級的重要資源。
