# 共創準則：《萬界交易商》

## 一、核心原則

> **爽是唯一的正義。** 所有設定、劇情、角色都為「讀者爽感」服務。

### 1.1 爽感三秒法則
- 每翻一頁（約500字）讀者就應該能感受到一個微爽點
- 每章（3000字）至少有一個中型爽點
- 每卷至少有一個爆炸級大爽點

### 1.2 壓抑-爆發守恆
- 壓抑鋪墊的章數 ≤ 爆發回報的爆炸程度
- 壓抑不超過3章，爆發至少回報2倍爽感
- 短壓抑 → 中爆發；長壓抑 → 大爆發

---

## 二、章節格式

### 2.1 標準格式
```
# 第X章 · 章名

正文內容...
```

### 2.2 字數要求
- 每章 ≥ 3000 中文字
- 上限不硬性規定，但建議 ≤ 5000 字以維持節奏

### 2.3 檔案規範
- 所有章節內容必須寫入對應的 `.md` 檔案中。
- 路徑格式：`正文/卷X/第XXX章·章名.md`。
- 嚴禁僅在對話框中輸出正文而不寫入檔案。

### 2.4 視角規範
- 主視角：第三人稱·蘇晨
- 允許短暫切換反派/配角視角（≤ 500字/次），用於製造反差
- 切換視角時用分隔線標記

### 2.5 段落規範
- 段落不超過200字
- 對白佔比約 40-50%
- 戰鬥場景用短句（≤ 30字/句）
- 高頻換段保持閱讀呼吸感

### 2.5 守護第四面牆
- 嚴禁在章末使用「章末提示」、「作者有話說」或任何元數據（Meta-data）形式的預告。
- 所有的劇情鉤子、神祕線索或系統提示必須完全融入正文敘述中，透過角色行動、感官或系統面板呈現，保持故事的純粹性與沉浸感。

---

## 三、角色塑造規則

### 3.1 主角（蘇晨）
- ✅ 果斷行動，不猶豫
- ✅ 該狠就狠，不聖母
- ✅ 幽默感，壓力下能吐槽
- ✅ 護短，對朋友絕不退讓
- ❌ 不得無故降智
- ❌ 不得對敵人手下留情（除非有明確戰略目的）
- ❌ 不得長篇心理獨白超過200字

### 3.2 反派設計
- 必須足夠囂張（囂張度 = 打臉爽度）
- 必須有合理動機（不能純粹為了壞而壞）
- 打臉節奏：小反派1-2章解決，中反派5-8章解決，大反派一卷

### 3.3 女性角色
- 有獨立人格和能力
- 不做花瓶不拖後腿
- 感情線自然發展，不強行推進

---

## 四、系統交易寫作規則

### 4.1 交易場景模板
```
【萬界交易系統】
─────────────
交易對象：[世界名]·[對象名]
對方需求：[物品名]
您可獲得：[物品名]
交易評級：★☆☆☆☆ ~ ★★★★★
─────────────
是否確認交易？ [是/否]
```

### 4.2 趣味性原則
- 現實世界的普通物品在異世界要有「反差萌」（如可樂=仙露）
- 交易過程要有對話互動，展現異世界角色的個性
- 偶爾出現「虧本交易」或「撿漏交易」增加驚喜感

### 4.3 平衡性原則
- 系統提供的資源與主角當前實力匹配
- 每次升級後交易收益也要升級，保持新鮮感
- 不能一次交易就獲得碾壓一切的力量

---

## 五、寫作禁忌清單

| # | 禁忌 | 原因 |
|---|------|------|
| 1 | 主角降智 | 為了劇情轉折讓主角犯低級錯誤 |
| 2 | 水字數 | 無意義的環境描寫或重複心理活動 |
| 3 | 說教段落 | 大段人生道理或社會評論 |
| 4 | 爽點斷流 | 連續3章以上無爽感 |
| 5 | 退師門 | 主角無故退步或失去已有能力 |
| 6 | 工具人女角 | 女角色只為襯托男主而存在 |
| 7 | 過度解說 | 大段設定講解打斷節奏 |
| 8 | 聖母行為 | 對死敵仁慈放手 |
| 9 | 突破第四面牆 | 在章末使用元數據預告（如「章末提示」） |

---

## 六、協作流程

### 6.1 寫作前確認
每次開始新章節前，確認：
1. 本章預定爽點是什麼？
2. 章末鉤子是什麼？
3. 與大綱的對應關係
4. **確認目標寫入檔案的路徑（必須寫入 .md 檔案）**

### 6.2 寫作後檢查
每章完成後確認：
- [ ] 字數 ≥ 3000
- [ ] 至少1個爽點
- [ ] 章末有鉤子
- [ ] 主角沒有降智
- [ ] 無水字數段落

### 6.3 定期回顧
- 每10章整體回顧一次節奏
- 確認爽感曲線健康（持續上升，無長期低谷）
- 可根據需要調整後續大綱
