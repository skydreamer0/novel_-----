# 《萬界交易商》 (Universal Trader)

> **核心主題**：都市爽文 | 系統流 | 萬界交易 | 現代修真
> 
> 一個普通的快遞員蘇晨，意外覺醒了「萬界交易系統」，從此開啟了用廉價物資換取異界逆天資源、在現代都市橫推一切的傳奇故事。

---

## 📖 故事簡介

現代都市表面之下，隱藏著一個由「覺醒者」構成的暗面世界。蘇晨獲得的系統能連結無數平行世界，從低階靈草世界到混沌古界，他只需用泡麵、能量飲料甚至玻璃珠，就能換取靈丹妙藥、絕世功法與神兵利器。

在天衡局的管理與世家大族的夾縫中，蘇晨如何利用「跨界交易」降維打擊，重組秩序，進而探尋空間裂縫背後的真相？

---

## 📂 項目結構

本項目不僅包含小說創作內容，還整合了一個輕量級的 Web 閱讀器。

```
.
├── 大綱/               # 劇情大綱與章節規劃
│   └── 卷一·覺醒.md     # 卷一 1-30 章詳細規劃
├── 正文/               # 小說正式章節內容
├── 設定/               # 世界觀與技術細節
│   ├── 世界觀.md       # 宏觀背景、組織與地理
│   ├── 力量體系.md     # 覺醒者等級與修行細節
│   ├── 角色設定.md     # 主要人物小傳與性格
│   └── 共創準則.md     # 寫作規範與 AI 協作指南
├── index.html          # 閱讀器前段界面 (廢星牧魂者)
├── styles.css          # 閱讀器樣式表
├── app.js              # 閱讀器邏輯與章節加載
└── 字數統計.csv         # 自動化生成的小說字數統計
```

## 🛠️ 技術棧與工具

### 1. 閱讀器 (Reader App)
*   **前端**: 原生 HTML5, Vanilla CSS, JavaScript (ES6+)
*   **解析**: [Marked.js](https://github.com/markedjs/marked) (用於渲染 Markdown 格式的章節內容)
*   **功能**:
    *   🌙 深色模式 / 淺色模式切換
    *   📱 響應式佈局 (適配手機與桌面)
    *   🔍 章節搜索與側邊欄導航
    *   🔡 字體大小與容器寬度調節
    *   💾 本地進度自動保存 (LocalStorage)

### 2. 自動化工作流
*   **Agent 工作流**: 利用定制的 AI Agent 進行內容協作、風格審計與字數統計。
*   **字數統計**: 通過腳本自動掃描 `正文/` 目錄並產出統計數據。

## 🚀 快速開始

### 閱讀小說
1.  在本機環境下使用任何靜態文件服務器啟動（如 VS Code 的 `Live Server` 或 `python -m http.server`）。
2.  打開瀏覽器訪問 `index.html`。
3.  從左側菜單選擇章節開始閱讀。

## 🤝 創作規範

如果您參與本項目的共同創作，請務必先閱讀 `設定/共創準則.md`。
*   **章節字數**: 建議每章不低於 3,000 字。
*   **風格**: 保持快節奏、強反差的爽文敘事模式。
*   **存儲**: 確保所有章節文件以 `.md` 格式存放在 `正文/` 對應卷目錄下。
