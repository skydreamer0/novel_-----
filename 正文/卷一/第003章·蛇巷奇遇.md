# 第三章 · 蛇巷奇遇

送完最後三單快遞的時候，天已經暗了。

臨海市的夜晚比白天更活躍——至少對蘇晨的新感知來說是這樣。他能感受到城市地底深處流淌著的一條肉眼不可見的「河流」。那是靈氣的脈絡，像血管一樣在城市的地下蜿蜒。

大部分地方的靈氣都很稀薄，像是漏水的水龍頭，滴滴答答。但有一個方向例外。

西南方。靈氣濃度是其他地方的三到四倍。

蘇晨把電動車停在路邊，看了一眼手機上的地圖。

西南方——那裡是舊城區最深處的「蛇巷」。

蛇巷之所以叫蛇巷，不是因為有蛇，而是因為這條巷子窄得像蛇一樣彎彎繞繞，據說清朝時候就有了。兩側的老房子擠在一起，頭頂只露出一線天空。白天都暗，晚上更是伸手不見五指。

蘇晨在這一帶長大，自然不怕。

他循著靈氣的方向走進了蛇巷。

巷子裡瀰漫著老城區特有的氣味——陳年木料、潮溼的磚牆，和隱約的滷肉香。幾盞昏黃的路燈散落在拐角處，把光暈打在斑駁的牆壁上。

走了大約兩百米。

蘇晨停下腳步。

靈氣的源頭在他的正前方——一間看起來毫不起眼的雜貨店。

店面不大，門口掛著一塊已經褪色的木牌，上面寫著「趙記雜貨」四個字。字跡歪歪扭扭的，像是用手指頭蘸墨汁寫的。玻璃門裡透出暖黃色的燈光，隱約能看到裡面的貨架上擺滿了各種零食、日用品和瓶瓶罐罐。

看上去就是一家普普通通的老巷雜貨店。

但蘇晨的感知告訴他，這間店的地下——至少三米深的地方——有一股極其沉穩、深厚的靈氣在緩慢流動。

像是一口沉睡的深井。

他猶豫了一下。

正常來說，一個剛覺醒不到六小時的新手，不應該主動去接觸未知的靈氣源頭。但好奇心是人類的本能，更何況——他在這裡聞到了滷肉的味道，而他從早上到現在還沒吃過飯。

蘇晨推開了玻璃門。

門上掛的銅鈴發出清脆的一聲響。

「歡迎光……」

店裡只有一個人。

一個中年男人坐在收銀台後面的藤椅上，翹著二郎腿，手裡拿著一份已經過期三天的報紙。他大約四十五六歲的樣子，圓臉，微胖，頭頂有點禿，穿著一件洗得發白的背心，腳上踩著一雙人字拖。

標準的巷子口大叔形象。

但他的「歡迎光臨」只說了一半就停住了。

他放下報紙，緩緩抬頭，瞇著一雙不大的眼睛看了蘇晨足足五秒鐘。

然後，他的表情從職業化的微笑，變成了一種微妙的警惕。

「小子，」他開口了，聲音不高，「你怎麼找到這裡的？」

蘇晨正準備說「我路過的」，但他的直覺告訴他不要撒謊。

「我感覺到了。」他說，「靈氣。你這個店的下面有很強的靈氣。」

中年男人的眼睛瞇得更細了。

他上下打量蘇晨，目光銳利得不像一個雜貨店老闆。幾秒鐘後，他開口了，語氣帶著一絲驚訝：

「你剛覺醒的？」

「嗯。今天下午。」

「今天下午覺醒，晚上就能循著靈氣找到這裡……」男人嘖了一聲，「感知力倒是不錯。可惜了。」

「可惜什麼？」

「可惜你的資質嘛。」男人重新拿起報紙，語氣懶洋洋的，「我看了一下你的靈氣波動，頂多D級。一階·感知，體內靈氣含量少得可憐。就這個底子，練一輩子也未必能突破二階。」

蘇晨沒有生氣。

D級。這和他預期的差不多。他本來就是一個普通人，沒有什麼修煉傳承，也沒有所謂的天賦異稟。他的覺醒完全是靠清心草催化的，根底確實薄弱。

但他有系統。

「你是覺醒者？」蘇晨直接問。

男人翻了個白眼：「廢話。」

「幾階？」

「你管我幾階。」男人把報紙攤在臉上，像是要睡覺的樣子，「買東西就買，不買就走。我要打烊了。」

蘇晨沒走。

他的目光掃過貨架——表面上擺的都是普通商品：泡麵、礦泉水、薯片、衛生紙。但他的感知力隱隱約約捕捉到了某些東西。

角落裡有幾個罐子，外面貼著「鹽漬梅子」的標籤，但裡面散發出的靈氣波動，絕對不是梅子該有的。

「你這個店……不只是賣雜貨吧？」

報紙下面傳來一聲嗤笑。

「行啊，小子，眼睛倒尖。」男人掀開報紙，坐直了身體，第一次認真地看著蘇晨。「你叫什麼？」

「蘇晨。」

「我姓趙，你叫我老趙就行。」男人從櫃檯下面摸出一個紫砂茶壺，自顧自地倒了杯茶，「這間店確實不只賣雜貨。但上面的東西不是你這個等級能碰的。你現在的身體，碰一下那罐子裡的東西，大概率會靈氣紊亂，嚴重的話經脈碎裂，直接變成廢人。」

蘇晨後退了一步。

老趙滿意地點點頭：「算你識趣。」

「那——」

蘇晨的話還沒說完，門外突然傳來一陣嘈雜的腳步聲和叫罵聲。

銅鈴被粗暴地撞響，門被推開。

五個紋身青年魚貫而入。為首的那個剃著平頭，脖子上掛著一條手指粗的金鏈子，嘴裡叼著半截沒點燃的香煙。

「趙老頭！」平頭青年把煙往地上一扔，「上個月的保護費還沒交呢。今天要麼交錢，要麼交店。你自己選。」

老趙看了他們一眼，臉上沒有任何表情。

「滾。」

就一個字。

平頭青年的臉一下子漲紅了：「媽的，敬酒不吃吃罰酒是吧？兄弟們，砸！」

五個人散開，開始推翻貨架、摔瓶子。玻璃碎裂的聲音在狹小的店裡格外刺耳。

蘇晨看了一眼老趙。

老趙端著茶杯，一口一口地喝，彷彿眼前發生的一切與他無關。

但蘇晨注意到了一個細節——老趙的眼神雖然懶散，但嘴角微微上翹。那不是害怕，也不是憤怒。

是一種「看猴戲」的表情。

蘇晨瞬間明白了。

這老頭至少是三四階的覺醒者，這幾個普通人的混混在他眼裡大概和螞蟻差不多。他不動手，純粹是懶得動。

但店裡的東西在被砸。

蘇晨不知道為什麼，就是看不下去。

或許是因為這間店讓他想到了福利院。王奶奶的福利院也被附近的混混騷擾過，那時候他才十四歲，拿著掃把把人趕走，被打了一頓。

不過現在，他不是十四歲了。

而且，他是覺醒者了。

蘇晨往前一步，一把抓住了平頭青年伸向貨架的手。

「幹嘛呢你？」平頭青年回頭，眼裡帶著兇光。

蘇晨沒說話。

他只是輕輕用力——

「嘎嘣——」

不是骨折的聲音。是平頭青年手腕裡某個關節被扭到了極限發出的聲響。

「啊啊啊啊啊！」平頭青年慘叫出聲，膝蓋一軟直接跪了下去。

蘇晨自己都嚇了一跳。

他只是想抓住對方的手，根本沒用多大力。但一階·感知帶來的體質增幅，讓他的力量遠超普通人。他的五根手指像鉗子一樣扣在對方的手腕上，那個平頭青年掙都掙不開。

其他四個人愣住了。

「你找死——」

一個紋身青年從背後撲上來。

蘇晨的身體比腦子快。他的感知力在對方出手的瞬間就捕捉到了攻擊的軌跡——右拳，目標是他的後腦勺。

他側身一閃。

那一拳打空了。

蘇晨順勢一肘，撞在紋身青年的肋骨上。不是很重，但足以讓對方一個踉蹌退出去三步。

三秒。

三秒鐘的時間裡，一個跪著，一個捂著肋骨彎腰。

剩下三個面面相覷，眼神裡的兇狠正在迅速被恐懼取代。

「你、你是練家子？」一個瘦小的青年往後退了一步。

蘇晨看著自己的手掌。

他沒練過任何格鬥術。他只是本能地根據感知力做出了反應。一階·感知對普通人而言，就已經是降維打擊了。

「走不走？」蘇晨抬頭看向剩下三個人，語氣平靜。

不是威脅。是真的在問。

三個人互相看了一眼，架起跪在地上的平頭青年和捂著肋骨的同伴，連滾帶爬地跑出了店。

「下次再來蛇巷老子弄死你！」

門口傳來一句色厲內荏的叫囂，然後腳步聲漸遠。

巷子裡恢復了安靜。

蘇晨轉身看向老趙。

老趙依然坐在藤椅上，茶杯端在手裡，但杯中的茶已經見底了。他看蘇晨的眼神，和剛才不太一樣了。

不再是「看一個D級廢柴」的眼神。

而是帶著幾分研究的意味。

「小子。」老趙把茶杯放下。

「嗯？」

「你剛覺醒不到一天，就敢對著五個人動手？」

「他們是普通人。」蘇晨實話實說。

「你怎麼知道他們是普通人？萬一裡面有覺醒者呢？」

蘇晨想了想：「我的感知沒有從他們身上偵測到任何靈氣波動。覺醒者就算再弱，身上也應該有靈氣的痕跡吧？」

老趙沉默了兩秒。

然後他哈哈大笑。

「有意思！」他笑得藤椅吱呀作響，「覺醒不到一天就知道用感知力辨別對手，不是誰教的，是自己悟出來的……你這腦子，比你的資質值錢多了。」

他站起身，走到被砸亂的貨架前，隨手一揮——

蘇晨瞪大了眼睛。

所有被砸翻的貨品、碎裂的玻璃瓶，像是被按下了倒放鍵。東西自動飛了起來，在空中翻轉、拼合，精準地歸位到各自的位置上。碎玻璃碴子凝聚成完整的瓶子，倒翻的貨架穩穩地立了回去。

三秒鐘。

整間店恢復如初。

像是什麼都沒發生過。

蘇晨的喉結動了動。

操控系靈技。而且是極其精細的操控——同時控制上百個物件的運動軌跡，每一個都精確到毫米。

這絕對不是一階兩階能做到的。

「你到底幾階？」蘇晨脫口而出。

老趙回頭看了他一眼，咧嘴一笑，露出一口略微發黃的牙齒。

「夠高了。高到你這個級別不需要知道。」

他走回收銀台，從下面翻出一碗用保鮮膜蓋著的東西，推到蘇晨面前。

「滷肉飯。今天剩的。」

蘇晨低頭看著那碗滷肉飯。

滷汁浸透了白飯，肉塊燉得軟爛，上面撒了一點蔥花。熱氣透過保鮮膜散發出誘人的香味。

他的胃發出了一聲不爭氣的咕嚕聲。

「吃吧。」老趙重新躺回藤椅上，「當是你幫我趕走那群蒼蠅的報酬。雖然我自己也能趕，但你替我省了動手的力氣。我這個人最討厭浪費力氣。」

蘇晨沒客氣，撕開保鮮膜就開始吃。

他是真的餓壞了。

在他大口扒飯的時候，老趙靠在椅背上，看著天花板，用一種漫不經心的語氣說道：

「小子，你知道覺醒者的世界是什麼樣的嗎？」

「不太清楚。」蘇晨嘴裡塞滿了飯，含糊地回答。

「簡單來說，」老趙伸出三根手指，「三股勢力。第一，天衡局。政府的秘密機構，管理全國所有登記在冊的覺醒者。第二，世家。傳承了幾百年的覺醒者家族，有錢有人有功法。第三，黑市。不受管控的地下世界，什麼都能買到，也什麼都敢賣。」

「你屬於哪一類？」

「我？」老趙笑了，「我三類都不是。我就是一個退了休的老頭，開個雜貨店混日子。」

蘇晨把最後一口飯塞進嘴裡。

他不信。

一個能在城市靈脈節點上開店的退休老頭，絕不可能只是「混日子」。但對方不想說，他也不會追問。至少不是現在。

「那我應該怎麼辦？」蘇晨把碗放下，「去天衡局登記嗎？」

「這要看你自己。」老趙伸了個懶腰，「登記的好處是有組織保護，有任務能接，有資源能拿。壞處嘛——你的一切都在別人的監控之下。」

他頓了頓，又說：「不過以你D級的資質，天衡局大概不會太重視你就是了。」

又是D級。

蘇晨咧嘴笑了笑，沒有反駁。

他走到門口，推開門。夜風吹進來，帶著舊城區的潮溼氣息。巷子裡黑漆漆的，只有遠處幾盞路燈發出昏黃的光。

「老趙。」

「嗯？」

「謝謝滷肉飯。」

「……少來這套。下次來記得帶錢。」

蘇晨笑著走了出去。

在他踏出店門的瞬間，老趙的聲音從背後傳來。語氣和剛才完全不同——不再懶散，而是帶著一絲認真。

「小子。」

「嗯？」

「這個世界比你想的危險得多。你今天能用拳頭嚇跑幾個混混，但覺醒者的世界裡，有些東西是拳頭解決不了的。」

蘇晨停下腳步，回頭看了他一眼。

老趙坐在昏黃的燈光裡，影子被拉得很長。

「城市東北方向的靈氣異常，你也感覺到了吧？」

蘇晨瞳孔微縮：「你也感覺到了？」

「廢話，半個城的覺醒者都感覺到了。」老趙的語氣沉了下來，「那裡出了東西。不是好東西。天衡局的人已經去了。」

他看著蘇晨的眼睛，一字一句地說：

「在你有足夠的實力之前，不要靠近那個方向。聽到了嗎？」

蘇晨點了點頭。

「記住了。」

他轉身走進夜色裡。

電動車的電瓶已經不太夠了，他推著車慢慢走在回家的路上。手機上，主管老劉又發來了一條語音訊息。他沒聽，直接刪了。

回到自己租的那間十平米的單間公寓，蘇晨洗了個澡，坐在了床邊。

他掏出口袋裡用工作服包著的兩株清心草，小心地放在枕頭旁邊。

然後，他召出了系統光幕。

```
══════════════════════════
  【萬界交易系統 v1.0】
══════════════════════════
  宿主：蘇晨
  等級：Lv.1
  境界：一階·感知
  靈技：未覺醒
  交易積分：10
  今日交易次數：0/3（已重置）
  已解鎖世界：靈草世界（Lv.1）
  
  可用功能：
  ① 發起交易
  ② 物品鑑定（基礎）
  ③ 交易記錄
══════════════════════════
```

他還有兩株清心草沒用。今天的交易次數已經重置，又有三次機會。

最重要的是——他現在知道了這個世界的規則。天衡局、世家、黑市。還有那個城市東北方向的異常。

蘇晨閉上眼睛，深呼一口氣。

他需要變強。

快速地、安靜地、在所有人的視線之外變強。

「系統。」

「在。」

「靈草世界除了清心草之外，有沒有能幫助修煉的功法類物品？」

```
  回答：有。
  靈草世界存在少量低階功法。
  但功法交易需要更高價值的籌碼。
  建議交易物品：加工食品、
  調味料、糖類製品。
  
  提示：靈草世界物資匱乏，
  宿主世界的食品類物品
  在該世界普遍具有較高價值。
```

蘇晨看了一眼窗外。

樓下就是一家24小時平價超市。

他站起身，從床底下翻出了存著全部積蓄的鐵盒子。打開。

裡面有：

三百七十二塊錢。

一張福利院的合照。

和一張王奶奶塞給他但他偷偷放回去又被奶奶偷偷塞回來的存摺。

蘇晨把三百七十二塊錢拿了出來。

然後他穿上拖鞋，走向了樓下的超市。

十五分鐘後，他提著兩大袋東西回來了。

泡麵，十二包。辣條，二十袋。可樂，六瓶。白砂糖，兩公斤。方塊酥，四盒。

十四歲的小藥農李小苗不知道的是，她的長期交易夥伴正在為她準備一場前所未有的「美食轟炸」。

而她手裡那些「滿山都是、不值錢」的靈草和功法——

才是真正的無價之寶。

蘇晨坐在堆滿零食的床上，嘴角緩緩上揚。

三百七十二塊錢的投資。

回報率——

無限。

就在蘇晨躺在狹小的單人床上，準備進入冥想狀態時，窗外的夜空突然被一道刺眼的冰藍色光芒點亮。

那光芒熾烈如白晝，卻冷得讓人脊背發涼，僅僅持續了三秒便消失在東北郊區的方向。

與此同時，整個臨海市無數正處於深度修煉中的覺醒者同時噴出一口鮮血，從驚愕中醒來。

第二天清晨，當蘇晨推著那台電動車走出蛇巷時，發現巷子口那堵斑駁的紅磚牆上，不知何時已經貼出了一張落款為「天衡局臨海分局」的醒目告示。

「臨海分局招募預備調查員、特邀外聘人員，覺醒者優先……」

蘇晨看著告示，心跳微微加速。

這個世界，終於要開始展露它的猙獰，而他，也終於拿到了入場券。
