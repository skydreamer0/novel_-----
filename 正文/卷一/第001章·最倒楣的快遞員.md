# 第一章 · 最倒楣的快遞員

臨海市的六月，熱得像是有人在天上架了口鍋，把整座城市放在裡面蒸。

蘇晨騎著電動車穿梭在大街小巷，後座堆了六個快遞箱，用繩子綁得搖搖欲墜。安全帽帶子斷了，他只能用下巴夾著，看起來像個嘴裡叼著半個西瓜的傻子。

「叮——您有新的訂單。」

手機導航的聲音從褲兜裡傳來，蘇晨咬著牙把車龍頭一扭，拐進了東港金融區那條被稱為「索命三公里」的上坡路。

之所以叫這個名字，是因為這條路上住的客戶，十個有八個投訴狂，剩下兩個是投訴狂的親戚。

「蘇晨！你的KPI這個月又墊底了！再被投訴一次，你就別幹了！」

主管老劉的語音訊息卡在耳機裡迴盪，蘇晨伸手想按掉，手一滑，耳機掉進了下水道的鐵柵欄縫裡。

他低頭看了一眼那個狹窄的縫隙，然後平靜地把視線收回來。

算了。那副耳機是九塊九包郵的。

兩分鐘後，他氣喘吁吁地按響了金港灣小區7棟1601的門鈴。

門開了一條縫，露出一張保養得極好的中年女人的臉。她掃了一眼蘇晨滿身的汗漬，皺眉的幅度精確到毫米。

「快遞？放門口。」

「姐，麻煩簽收一下。」蘇晨舉起手機，螢幕上是簽收頁面。

「我說放門口就放門口，聽不懂？」

蘇晨嘴角動了動，把東西放在門口的地墊上。他很想說這是易碎品需要當面驗收，但他更珍惜自己的KPI。

他剛轉身，身後傳來門開的聲音和一聲尖叫。

「你！你看你怎麼放的！盒子壓扁了！」

蘇晨回頭。那個「壓扁」的程度，大概相當於一張A4紙被折了一下。

「我要投訴！」女人已經掏出手機開始拍照，「態度惡劣，暴力投遞！」

蘇晨張了張嘴，最終什麼都沒說。

他太清楚這個流程了——投訴、扣錢、寫檢討。這套他已經經歷過不下二十次。

下樓的時候他算了一筆帳：這個月被扣了四百，加上耳機損失九塊九，再加上今天早上電動車被貼的違停罰單五十塊……

他這個月基本上是在做慈善。

電動車停在樓下，後座剩下的五個箱子在陽光下微微發燙。蘇晨灌了一口已經變得溫熱的礦泉水，繼續上路。

下一單在兩公里外的商業步行街。收件人是個年輕男人，開門的時候手裡拿著遊戲手柄，空調的冷氣從門縫裡溢出來，打在蘇晨滿是汗水的臉上。

「哦，快遞。等一下。」男人頭也不回地走了進去。

蘇晨在門口站了七分鐘。

七分鐘裡他聽到了遊戲裡的槍聲、爆炸聲、和這個男人罵隊友的聲音。走廊裡沒有空調，他感覺自己像是一塊正在脫水的牛肉乾。

男人終於簽了收。

「誒等一下，我順便寄個件。」男人指了指客廳角落裡一個巨大的紙箱，「幫我搬下去。」

蘇晨看了一眼那個箱子。目測至少三十公斤。電梯在維修。這裡是十二樓。

「好的。」他把箱子扛上了肩膀。

走到六樓的時候，他的腿開始發抖。走到三樓的時候，汗水已經把他的工作服浸透了，從衣角往下滴。他的視線有點模糊，胸口的悶熱感越來越強。

但他沒有停下來。

因為停下來就意味著——箱子會從肩膀上滑下去。他的身體已經處在臨界狀態，靠的全是慣性。

終於走到了一樓大堂。他把箱子放上電動車，喘得像條魚一樣張著嘴。

手機震了一下。又一條投訴通知。剛才那個金港灣的女人，果然投了。

罰款一百五。

蘇晨用袖子擦了一把臉上分不清是汗還是別的什麼液體，打開手機看了一眼。螢幕上還有主管老劉發來的新訊息：「你那個1601的投訴我壓不住了，客服那邊要你寫書面檢討。八百字，今晚交。」

八百字的書面檢討。

蘇晨上大學的時候寫畢業論文都沒這麼痛苦過。

他又灌了一口礦泉水，拿起下一單的訂單。目的地在三公里外的舊城區。

舊城區的路窄得只能容一輛車通過，兩側是上世紀八九十年代蓋的老樓。牆皮剝落、鐵鏽斑斑的防盜窗上晾滿了衣服。蘇晨在這裡長大——福安福利院就在這片區域的盡頭。

他一邊騎車一邊想著晚上要不要去看王奶奶。福利院的老院長今年八十二了，身體越來越差。他上次去的時候，老太太拉著他的手說了句「小晨啊，奶奶給你存了些錢，你拿去討個媳婦」。

那存摺上的數字是三萬二。老太太攢了大半輩子。

蘇晨沒拿，騎著車跑出去哭了一場。

「嘀嘀——」

一聲刺耳的喇叭從身後響起，一輛改裝過的越野車貼著蘇晨身邊擦過去，後視鏡刮掉了他一個快遞箱。

箱子摔在地上，裂開了。

蘇晨緊急剎車，電動車歪倒，剩下的箱子像下餃子一樣噼裡啪啦滾了一地。他右手肘著地，火辣辣地疼。

越野車沒停。

車尾燈消失在巷子拐角，留下一串囂張的改裝排氣管轟鳴聲。

蘇晨趴在地上，看著散落一地的快遞箱和自己手肘上滲出的血。

六月的太陽，烤得柏油路面都在冒泡。

他沒有動。不是不想動，是動不了。

早上五點半起床，連送了七個小時的快遞，一口飯沒吃。那瓶礦泉水是他今天攝入的全部水分。加上被刮倒的衝擊，他現在渾身發軟，耳朵裡嗡嗡作響。

舊城區的這條小巷幾乎沒有人經過。斑駁的牆壁上爬滿了歲月的痕跡，頭頂是密密麻麻的電線和偶爾飛過的鴿子。

蘇晨翻了個身，躺平了。

「要是我有超能力就好了。」他看著天空自言自語，嘴角扯出一個有點苦的笑。

這句話他說過很多次。每次被生活按在地上摩擦的時候，他都會這麼想。如果自己有漫畫裡那種超能力，這些爛事全都不是事。

當然，他知道那是不可能的。

就像他知道丟進下水道的耳機撈不回來，被投訴的差評消不掉，越野車的車牌號他根本沒看清。

這就是現實。

普通人的現實。

他閉上眼睛，準備休息三十秒再爬起來繼續送快遞。

然後——

腦海深處，忽然響起了一個聲音。

那不是幻覺。不是耳鳴。不是中暑的妄想。

那是一個精確的、冰冷的、帶著電子質感的女聲。

「叮——」

「檢測到宿主瀕臨崩潰值。」

「緊急啟動協議觸發。」

「萬界交易系統正在初始化……」

「初始化進度：10%……35%……72%……100%。」

「初始化完成。」

蘇晨猛地睜開眼睛。

他的眼前，憑空出現了一塊半透明的藍色光幕。

光幕上的文字清晰得不像是投影，它彷彿直接刻在了他的視網膜上，隨著視線移動而移動。

```
══════════════════════════
  【萬界交易系統 v1.0】
══════════════════════════
  宿主：蘇晨
  等級：Lv.0（未綁定）
  交易次數：0/3（日限）
  交易積分：0
  已解鎖世界：靈草世界（Lv.1）
──────────────────────────
  ！系統提示：
  請在24小時內完成首次交易
  以完成綁定程序。
  逾期未綁定，系統將自動解綁
  並尋找下一位宿主。
══════════════════════════
```

蘇晨盯著這塊光幕，一動不動地躺了整整十秒鐘。

他做了三件事：

第一，掐了一下自己的大腿。疼。

第二，閉眼再睜開。光幕還在。

第三，用受傷的手肘撐地坐起來。光幕跟著他移動，穩穩地懸浮在視野正前方。

「……我是中暑了嗎？」

光幕上多出了一行字。

```
  回答：宿主未中暑。
  當前體溫37.2℃，略高於正常值。
  建議補充水分。
```

蘇晨呆住了。

它聽得見。

他嚥了口口水，小心翼翼地開口：「你是什麼？」

```
  回答：萬界交易系統。
  功能：連接萬界，為宿主提供
  跨世界交易服務。
  簡單來說——
  宿主可以用現實物品，
  換取其他世界的資源。
```

蘇晨的腦子轉得飛快。

如果這是真的——哪怕只有百分之一的可能性是真的——

「其他世界的資源……是什麼意思？」

```
  回答：包括但不限於——
  靈藥、功法、丹藥、法器、
  靈石、秘技、天材地寶等。
  具體可交易物品取決於
  已解鎖世界與宿主等級。
```

蘇晨慢慢站了起來。

他的手還在抖。不是因為害怕。是因為——

靈藥？功法？法器？

這些詞從小說裡跳出來，光明正大地出現在他的眼前。如果這是惡作劇，那這個惡作劇的成本也太高了——憑空生成一塊能跟隨視線移動的全息光幕，還能即時回應語音——這種技術地球上不存在。

如果不是惡作劇呢？

蘇晨的心跳加速了。他能感覺到太陽穴處的血管在突突地跳動。他想起了自己說過的那句話——「要是我有超能力就好了」。

這句話他從十四歲開始說，十四歲那年他拿著掃把擋在福利院門口，被兩個比他高一個頭的混混揍得鼻血直流。他躺在地上的時候第一次這麼想。

後來，每一次被生活按在地上的時候，他都會這麼想。

高考差三分沒上一本的時候。

大學期間靠三份兼職養活自己的時候。

畢業後投了一百七十六封履歷只收到四個面試通知的時候。

最後做了快遞員，被客戶罵、被主管罵、被生活罵的每一天。

每一次，他都會在心裡說那句話。然後告訴自己，這不可能。

但現在，一塊不可能存在的光幕就懸浮在他的面前，冷靜地告訴他——可能。

他低頭看了看地上散落的快遞箱，又看了看手肘上已經結痂的傷口。

是因為，如果這一切是真的，他這二十三年來所有的操蛋日子，可能就要到頭了。

他深吸一口氣。又深吸了一口。

「你說要完成首次交易才能綁定？」

```
  正確。
  建議宿主準備一件物品，
  系統將自動匹配最適合的
  交易對象。
  提示：任何現實物品均可。
```

蘇晨環顧四周。

倒在地上的電動車，散落的快遞箱（別人的東西，不能用），破了個洞的工作背心……

最後，他的目光落在了車筐裡那瓶喝了兩口的可樂上面。

那是他今天早上用投幣機花三塊錢買的，還剩大半瓶。

三塊錢。

「……就用這個試試。」

他伸手拿起可樂。

光幕閃了一下，一個新的介面緩緩展開。

```
══════════════════════════
  【發起交易】
──────────────────────────
  提供物品：碳酸飲料（可口可樂）
  物品評級：☆（普通物品）
  
  正在掃描可交易世界……
  正在匹配交易對象……
  
  匹配成功！
──────────────────────────
  交易對象：靈草世界·青禾原
  對方身份：藥農·李小苗
  對方狀態：在線
  
  是否建立交易連結？
══════════════════════════
```

蘇晨握著那瓶可樂，站在舊城區空無一人的小巷裡。

六月的陽光把他的影子拉得很長。

他咧嘴笑了一下。

「建立。」

光幕猛地爆發出一道柔和的金光，竟然直接將蘇晨整個人包裹了進去。

就在這一瞬，一個怯生生的、帶著稚氣的小女孩聲音，突兀地在他腦海中響起：

「你……你是仙人嗎？手裡那個……那個會噴氣的神液是什麼？」

蘇晨愣住了。

在這一刻，二十三歲的快遞員蘇晨還不知道，自己按下的這個按鈕，將會徹底改寫整個萬界的命運軌跡。

包括他自己的。
