# 第十一章 · 反獵殺時刻

深夜，臨海市舊城區。

老舊的公寓樓像是一頭在暗夜中沉睡的斑駁巨獸，牆皮在夜風中發出細微的剝落聲。蘇晨剛從窗戶翻進那間不到十平米的租屋，氣都還沒喘勻，全身的汗毛突然像被針紮了一樣，根根立起。

一種被毒蛇盯上的滑膩感，瞬間爬滿了他的背脊。

「系統，解析偵查，覆蓋半徑五十米！」

蘇晨在識海中發出冷冽的指令。

```
  【能量解析偵查中...】
  ─────────────
  目標數量：4
  方位分布：
  - 10點鐘方向（走廊盡頭）：二階巔峰（氣息晦暗）
  - 1點鐘方向（隔壁空房）：二階中階（持有熱武器）
  - 6點鐘方向（窗外空調主機）：二階初階（潛行狀態）
  - 正下方（鄰居房間）：二階初階（生命指數異常）
  ─────────────
  威脅評級：A級（致命伏擊）
```

蘇晨眼中的瞳孔驟然鎖成了針尖大小。

他沒想到「蛻變者」的動作竟然這麼快。他在任務中拿走的那顆黑色結晶，顯然比他想像中還要重要。這四個人配合默契，呈死角包圍，顯然是不打算留活口。

「果然，實力提升太快，麻煩也會跟著升級。」

蘇晨冷笑一聲，沒有打算開燈，而是順手抓起了桌上一柄生鏽的裁紙刀——那是他平時用來拆快遞包裹最順手的工具。

此時的他，識海中正盤旋著葉孤楓傳授的那道「歸墟劍意」。那意境寂滅、孤絕，彷彿連虛空都能斬碎。

「正好，拿你們試試劍。」

砰！

房門幾乎在同一瞬間被一股蠻力硬生生轟開。

兩道戴著黑色獠牙面具的身影魚貫而入，他們手中的合金短弩噴射出紫色的毒箭，帶著淒厲的破風聲，瞬間將床上的被褥射得稀爛。

「目標失蹤！」

領頭的殺手低喝一聲，語氣中帶著一絲驚疑。

在這種死局下，一個剛突破二階的新手，怎麼可能消失得無聲無息？

「你們是在找我嗎？」

蘇晨平淡得如同在向客戶確認快遞單號的聲音，緩緩在門後的陰影中響起。

兩名殺手猛地回頭，卻只看見一道極其微弱、暗淡到幾乎看不見的光芒閃過。

「歸墟，瞬殺。」

沒有璀璨的特效，沒有震耳的嘶吼。

那一抹青黑色的光芒，快到了超越肉眼捕捉的極限。在兩名殺手的視界中，整個房間的空間彷彿在這一刻靜止了。

「咔……」

輕微的一聲脆響。

領頭殺手的喉管在瞬間被那柄生鏽的裁紙刀精準劃開。他甚至沒有感覺到疼痛，只是發現自己的視角開始不受控制地傾斜，黑色的液体瘋狂湧出，卻被一種奇異的引力束縛在一起，沒有噴濺出半點聲音。

他在意識消散前，最後看見的，是那個穿著快遞服的少年，眼神比他這個殺手還要冰冷。

第一殺！

「老二！」

另一名殺手眼眶欲裂，他猛地發動了覺醒能力，雙臂肌肉像是充氣般迅速膨脹，化作了一對佈滿紫色鱗片的利爪，瘋狂地抓向蘇晨的面門。

然而，蘇晨的身影卻像是紙片般在空中一個不可思議的高頻震動，直接避開了抓擊。

他右手握著的裁紙刀，在空中畫出一個完美的圓形弧線，以一種完全不符合物理常識的角度，刺入了對方心臟旁的能量源點。

「噗滋——」

靈氣炸裂。那名殺手發出一聲悶哼，整個人直接被自體崩塌的靈壓震飛出去，甚至沒能在大腦反應過來之前發出求救。

十秒，兩殺。

這就是萬界頂級劍意的恐怖。

「不對勁！」

躲在窗外空調主機上的三號殺手感應到屋內戰鬥的平息速度也太快了，他毫不猶豫地從懷裡掏出一管紫色的藥劑，直接扎進了自己的大腿。

「為了神聖的……進化！」

他的身體發出骨骼交替磨損的刺耳聲，四肢拉長，背部竟然刺破皮膚長出了幾根類似蜘蛛長矛的節肢。

「轟——！！」

整個公寓的三層外牆，在瞬間被一種強烈的紅光炸裂。

原來，四號殺手隱藏在正下方，他見伏擊失敗，竟然直接啟動了事先埋設在天花板層的「源質感應雷」。

蘇晨感覺整層樓的地板都在消失。

他身形如燕，在煙塵與火光中，腳尖精準地踏在一塊漂浮的斷磚上，借力一躍，整個人直接竄上了天台。

月光下。

那名異化後的「蜘蛛怪」早已在天台等候多時。他的雙手已經徹底變成了紫色的金屬鐮刃，口中噴吐著帶有腐蝕性的紫色毒氣。

「找到你了，卑微的母體材料。」殺手的聲音嘶啞，帶著一股不屬於人類的瘋狂。

二階巔峰，半妖化。

「母體材料？」蘇晨吐出一口帶灰的唾沫，看著已經毀了一半的公寓樓，眼中的怒火第一次燃燒了起來，「把我房子炸了，還想把我當材料？你們這組織的人，是不是發工資之前都要先切了腦葉？」

「死！」

怪物黑影一閃速度快到了極致，兩柄鐮刃交錯成十字，帶著斷金裂石的威力，將空氣都斬出了刺耳的爆鳴聲。

蘇晨沒有退，也沒有躲。

他體內的《長青吐納術》在此刻運轉到了極致，剛剛突破二階的靈氣如同怒濤般灌注進指尖。

「系統，暫行同步葉孤楓劍意加持！」

```
  【能量過載警報】
  ─────────────
  同步率：15%
  預計消耗：總靈氣值的 60%
  是否執行？ [是/否]
  ─────────────
```

「執行！」

那一瞬間，蘇晨感覺手中的裁紙刀消失了。或者說，那是因為他本身化作了一柄通天徹地的巨劍。

寂滅、荒涼、歸於虛無。

「瞬殺·寂滅！」

蘇晨腳下的天台水泥地發出不堪重負的崩裂聲。他整個人化作一道漆黑的細線，正面迎著那對紫色鐮刃撞了上去。

唰。

兩道身影交錯而過。

「蜘蛛怪」僵在了半空中，他的兩對鐮刃還保持著下劈的姿勢，但刃口卻在瞬間崩裂成了無數粉末。

他的喉間發出咯咯的聲音，一個細小的黑點從他的眉心緩緩擴散，最終覆蓋了整個臉部。

「這……是……什麼……」

砰。

殺手的身體像是一截枯木，從空中重重摔落在瓦礫堆中，生機全無。

一劍，瞬殺二階巔峰。

蘇晨拄著鋼管（裁紙刀已崩毀），大口大口地喘著粗氣，臉色慘白得嚇人。體內的靈氣被這一劍徹底抽乾，連骨頭縫裡都透著一股酸麻。

```
  【戰鬥總結】
  ─────────────
  目標：蛻變者先遣隊*4
  評價：SSS（完美反獵殺）
  獲得物資：源質殘片*2、破損的合金刀具*2
  系統經驗：+50
  ─────────────
```

就在這時，一陣密集的腳步聲和刺耳的剎車聲在樓下響起。

幾輛掛著天衡局特種號牌的黑色越野車狂飆而至，強烈的探照燈光束瞬間將天台照得亮如白晝。

「不許動！天衡局辦事！」

幾名全副武裝的調查員衝上天台。

林若溪背負著那柄冰藍色的長劍，一馬當先地從瓦礫堆中走出。當她看見水塔下站著的蘇晨，以及那一地被切得不成人形的怪物殘肢時，整個人徹底僵在了原地。

「蘇晨？」她的聲音帶著一抹難掩的顫抖和難以置信。

她原本接到加密警報，說是有一支專業的「蛻變者」刺客隊伍進入了舊城區目標點。她幾乎是帶著給蘇晨收屍的心情狂飆而來的。

可現在，她看見了什麼？

四具焦黑或被切斷的屍體，連半妖化的二階巔峰隊長都沒能逃脫，而那個只有「二階初階」的快遞員，雖然看起來虛弱，卻依舊站得筆直。

「隊長，你來晚了。」

蘇晨抬起頭，露出一個比哭還難看的憨厚笑容，指了指還在燃燒的公寓，「局裡管報銷嗎？我這房租還交了一年呢，房東太太非殺了我不可。」

林若溪看著他，又看了看那切面平整如鏡的水泥斷裂處，清冷的眼眸中第一次浮現出了深深的震撼與……一絲莫名的忌憚。

她知道，這個名不見經傳的「D級潛力」少年，身上隱藏的東西足以讓整個臨海市乃至省城都為之瘋狂。

「救援隊，封鎖現場！」

林若溪回過神，語氣變得嚴肅起來，她走到蘇晨面前，脫下自己的披風罩在蘇晨血跡斑斑的肩膀上，壓低聲音說道：

「這種實力……以後別在公眾面前展示。跟我走，現在這裡不安全了。」

蘇晨點了點頭，感覺眼皮越來越沉。

「獎金……別忘了加培……」

話音未落，他整個人直接栽進了這朵冰山雪蓮的懷中。

林若溪身體一僵，手下意識地扶住對方的腰。感受到蘇晨身上那股即便是力竭也經久不散的、銳利到極點的劍氣，她抿了抿唇，清冷的眸光直刺夜空。

「蛻變者，你們過界了。」
