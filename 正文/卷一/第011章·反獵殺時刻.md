# 第十一章 · 反獵殺時刻

深夜，臨海市舊城區，光影交錯的破敗小巷深處。

那座屹立了數十年的老舊公寓樓，在此刻像是一頭在窒息黑夜中沉睡的斑駁巨獸，外牆上那些鏽跡斑斑的鐵架在夜風的吹拂下，發出令人牙酸的細微摩擦聲，伴隨著牆皮一片片無聲脫落的脆響。

蘇晨動作輕捷地從那扇破舊的窗戶翻進自己那間不足十平米的出租屋，雙腳剛落地，氣都還沒來得及喘勻。就在這一剎那，他全身的汗毛像是感應到了某種極致的危險，如被萬根鋼針紮了一般根根倒立而起！

一種被潛伏在暗處的劇毒之吻死死盯上的粘稠滑膩感，順著他的腳踝，瞬間爬滿了他的整個背脊，令他的心跳猛然停滯了一拍。

「系統，立刻開啟全功率解析偵查，覆蓋半徑五十米內的所有生命信號！」

蘇晨在識海中發出了一聲如同萬載玄冰般的冷冽指令，眼神中原本的疲憊瞬間被一股凌厲的殺伐氣息所取代。

```
  【區域級能量解析偵查模塊啟動中...】
  ─────────────
  偵測到強敵目標：4名
  即時方位分布：
  - 10點鐘方向（走廊狹窄盡頭）：二階巔峰覺醒者（氣息晦暗且帶著腐敗味）。
  - 1點鐘方向（隔壁廢棄空房）：二階中階（持有非法高頻熱武器）。
  - 6點鐘方向（窗外空調壓縮機主機後）：二階初階（正處於高階潛行狀態）。
  - 正下方（三樓鄰居地板下方）：二階初階（生命指數呈現病態扭曲，疑似實驗體）。
  ─────────────
  綜合威脅評級：A級（極度致命的連鎖伏擊圈）
```

蘇晨原本清澈的瞳孔，在那一瞬間驟然縮成了如貓科動物般的針尖大小。

他是真沒料到，那個神祕的「蛻變者」組織，動作快得簡直像是一群嗅到了腐肉氣味的禿鷲。他在實戰任務中順手牽羊拿走的那顆疑似「進化母板」的黑色結晶，顯然比他想像中要重要千百倍。這四名殺手配合得天衣無縫，成斜十字死角包圍了這間小屋，顯然是接到了徹底抹殺、不留活口的死命令。

「果然，一旦實力提升得太扎眼，隨之而來的麻煩也會跟著跨維度升級啊。」

蘇晨嘴角勾起一抹自嘲的冷笑。他沒有選擇開啟那盞昏黃的電燈，而是憑藉著驚人的夜視能力，順手從桌上抓起了一柄生鏽、卻被他磨得極其銳利的裁紙刀。這曾是他平時用來拆高檔快遞包裹最順手的工具，如今卻成了他保命的唯一兵刃。

此時的他，整個識海中正瘋狂地盤旋著剛剛從葉孤楓那裡換來的、那道震撼諸天的「歸墟劍意」。那種意境寂滅、孤高且帶著毀滅萬物的決絕，彷彿連這片虛偽的時空都能被其一劍斬碎。

「正好，體內的靈氣剛回滿，便拿你們這幫見不得光的老鼠，試試我這跨位面的劍意。」

「砰——！」

幾乎是在蘇晨思緒落下的同一微秒，老舊的木質房門被一股難以想像的恐怖蠻力硬生生轟成了漫天飛舞的木屑。

兩道戴著猙獰黑色獠牙面具、渾身籠罩在陰影中的殺手身影魚貫而入。他們手中的制式合金短弩在空中噴射出三道帶有紫黑色幽光的毒箭。那箭矢帶著尖銳得足以撕裂耳膜的破風聲，瞬間將床鋪上那層厚厚的棉被射得稀爛，棉絮如白雪般在空中紛飛。

「目標消失！撤回交叉火力！」

領頭的殺手瞳孔一縮，低聲喝止，語氣中帶著一抹難以掩飾的驚疑。
在這種近乎死局的交叉伏擊下，一個資料顯示兩天前才晉升二階的「廢物學生」，怎麼可能在他們的感知鎖定中消失得如此無聲無息？

「諸位，是在找我這個延遲派送的件嗎？」

蘇晨那平淡得如同平時在電話裡向客戶核對快遞單號的慵懶聲音，在一片死寂中緩緩從門後的陰影死角處響起。

兩名殺手驚得頭皮發麻，猛地旋身回頭。可他們在那狹窄的視界中，卻只看見了一道極其微弱、暗淡到幾乎與黑暗融為一體的青灰色流光，一閃而逝。

「歸墟——第一式：瞬殺。」

沒有什麼驚天動地的特效，也沒有震破耳膜的怒吼。

那一抹帶著死亡氣味的青黑色鋒芒，快到了超越人類視覺捕捉極限的地步。在那兩名殺手的最後意識界裡，不僅僅是視線，甚至連整個公寓三層的空間流動，彷彿都在這一刻徹底陷落到了靜止的深淵。

「喀……嚓。」

一聲極其輕微、如同剪刀裁開絲綢的脆響。

領頭殺手的喉管在轉瞬間就被那柄廉價、生鏽的裁紙刀精準無比地劃開。他甚至沒能感覺到絲毫的疼痛，只是驚恐地發現自己的世界平衡感開始不受控制地崩塌，大口大口的黑色液體混合著扭曲的靈氣瘋狂從喉間湧出，卻被一種奇異的歸墟引力束縛在一掌範圍內，竟然沒有發出半點噴濺的聲響。

他在意識徹底墜入永恆黑暗前，最後看見的，是那個穿著亮紅色快遞制服的平凡少年。那少年的眼神，竟然比他這個專業殺手還要冷酷、還要空無一物。

這，就是所謂的「第一殺」。

「老二！！」

另一名殺手見此慘狀，雙目驚得幾乎要裂開。他猛地發動了體內隱藏的覺醒秘法，全身的肌肉像是充氣般迅速膨脹，雙臂瞬間長出了密密麻麻的紫色鱗片，化作了一對足以分金斷石的猙獰利爪，帶著狂暴的靈壓瘋狂地抓向蘇晨的面門。

然而，蘇晨的身影卻在此刻展現出了如紙片般輕盈的、在高頻震動中產生殘影的視覺欺騙技巧。他竟然在半空中做了一個不可思議的位移，與對方的利爪擦肩而過。

他右手原本握著的裁紙刀，在空中順勢劃出一個圓潤且完美的弧形。它以一種完全不符合地心引力與運動學常識的角度，悄無聲息地刺入了對方那厚重鱗片下方、唯一的一處靈力迴路匯聚點。

「噗滋——！」

伴隨著靈氣迴路的瞬間炸裂，那名殺手只來得及發出一聲含糊不清的悶哼。他整個人直接被自己體內那失控崩塌的靈壓震飛出了十幾米遠，連同背後的磚牆都被撞出了一個巨大的窟窿。

短短十秒鐘，兩名專業刺客，一死一殘。

這，就是來自高等文明位面的殺伐劍意，對於這個靈氣剛復甦的世界所產生的降維打擊。

「情況不妙！這小子在扮豬吃虎！」

原本蹲守在窗外、正試圖通過空調架潛入的三號殺手感應到屋內的戰鬥平息得有些詭異，他沒有絲毫猶豫，直接從戰術腰帶裡掏出一管紫紅色的基因藥劑，粗暴地扎進了自己的大腿動脈。

「為了神聖且終極的……進化！與主同在！」

他的身體內部不斷傳出令人牙酸的骨骼強行拉伸與錯位的嘎吱聲。他的四肢被異常拉長，背部竟然刺破血肉生長出了幾根類似於巨型蜘蛛的長矛節肢，整個人在那一刻徹底喪失了人類的形態。

「轟——！！」

就在此時，整個公寓三樓的外牆，在一陣震耳欲聾的紅光與火浪中徹底炸裂開來。

原來隱藏在蘇晨正下方三樓房間、一直按兵不動的四號殺手，見伏擊計畫徹底破產且隊友接連隕落，竟然喪心病狂地直接引爆了事先秘密埋設在天花板隔層內的「源質高頻感應雷」。

蘇晨感覺到腳底的地板在瞬間分崩離析。

他身形如燕，在漫天飛舞的煙塵與滾滾火光中，腳尖極其精準地踏在一塊正在墜落的斷裂紅磚上。他借著那微弱的一點反彈力猛然一躍，整個人化作一道殘影，直接沿著斷裂的牆壁竄上了空曠的公寓天台。

冷月如鉤。

然而，那名已經異化成了半人半妖形態的「蜘蛛怪人」，似乎早已憑藉著野獸般的直覺在那裡等候多時。他的雙腕已經徹底骨質化，變成了兩柄泛著金屬紫光的巨大鐮刃，口中更是不斷噴吐著帶有強烈腐蝕臭味的紫色孢子毒霧。

「找到你了……你這具口感絕佳的完美母體材料。」變異殺手的聲音嘶啞且層疊，帶著一種不屬於碳基生物的瘋狂與執著。

二階巔峰境界，且搭配了非人的生化異化增幅。

「母體材料？我都說過多少遍了，我是個送快遞的。」
蘇晨吐出一口混合著灰塵與血沫的唾沫，看著腳下已經毀掉了一大半、冒著滾滾濃煙的公寓樓，他心中那股積蓄已久的怒火第一次徹底燃燒了起來：
「把我辛辛苦苦攢錢租的房子給炸了，還想把我這塊材料塞進實驗室？你們組織在出任務前，是不是都得先去醫院切了腦下垂體？」

「給我……死！！」

變異怪物的身影在月光下如同鬼魅般消失，速度快到了極致。他的兩柄骨質鐮刃在半空中交錯成一個巨大的紫色十字，帶著足可斷金裂石的蠻橫威力，將周遭的空氣都斬裂出了刺耳的音爆聲。

這一次，蘇晨沒有選擇退，更沒有選擇像剛才那樣投機取巧地閃躲。

他體內的《長青吐納術》在此刻以前所未有的頻率瘋狂運轉。那剛突破二階、尚未完全平復的靈石靈氣如同決堤的怒濤，盡數灌注進了他的每一個毛孔，最終匯聚在他的指尖之上。

「系統，立刻開啟暫行同步模塊！強行加持葉孤楓那五成以上的歸墟劍意！」

```
  【能量超頻過載警報！！】
  ─────────────
  空間同步率：18%（已達宿主識海負荷極限）
  當前預計能量消耗：總靈氣蓄備值的 65%
  警告：施展後將進入長達 2 小時的虛弱期。是否執行？
  ─────────────
```

「別廢話了，給老子執行！！」

那一瞬間，蘇晨感覺自己手中的那截裁紙刀殘片消失了，或者說，是他整個人在這一刻徹底化作了一柄通天徹地、足以剖開這片蒼穹的寂滅巨劍。

周圍的一切喧囂似乎都遠去了，唯剩下一片無垠的、令人絕望的荒涼與虛無。

「瞬殺——寂滅大歸墟！」

蘇晨腳下的石質天台不堪重負地發出了如蛛網般的崩裂聲。他整個人在原地化作了一道漆黑得深不見底的細線，迎著那對如山崩般壓下的紫色鐮刃，正面撞了上去。

「唰——！」

兩道速度快到模糊的身影在天台上空一閃而過，交錯而立。

那尊原本咆哮著要將蘇晨切碎的「蜘蛛怪人」像是被按下了時間暫停鍵，僵在了半空中。他那兩對碩大的紫色鐮刃還保持著下劈收割的姿勢，但原本堅硬無比的刃口，卻在這一秒內無聲地崩解成了無數晶瑩的粉塵。

他的喉嚨發出「咯咯」的破碎聲，一個細微到若有若無的黑點，從他的眉心正中央緩緩擴散開來，最終將他整張臉都渲染成了死一般的灰敗。

「這……到底……是……什麼……」

「砰！」

殺手那巨大的、充滿了異化肌肉的殘軀，像是一截徹底腐朽的枯木，從空中重重摔落在破碎的瓦礫堆中，生機全無，連靈氣也一併被那道劍意徹底蒸發。

一劍出手，瞬殺一名二階巔峰的半妖精英。

蘇晨拄著一根從水塔旁隨手掰斷的鋼管（原本的裁紙刀早已承載不住那股劍意崩毀了），坐在滿是冷灰的天台上，大口大口地喘著帶著硝煙味的粗氣。他臉色慘白得如同白紙，體內的靈根像是被乾透的抹布，原本充盈的靈氣被那一劍抽掠得乾乾淨淨，連每一根骨頭縫裡都在透著一股鑽心的酸麻與虛空感。

```
  【戰後實時結算】
  ─────────────
  擊殺目標：[蛻變者]精英先遣小隊 * 4
  戰鬥評價：SSS（教科書級別的極致反獵殺）
  戰利品獲取：源質基因殘片 * 2、破損的合金外星科技刀具 * 2、黑市信標點 * 1
  系統升級進度：能量飽和，經驗值補正 +50
  ─────────────
```

就在這寂靜得只剩下火焰燃燒聲的時刻，一陣急促且整齊的腳步聲，伴隨著刺破夜空的輪胎磨擦聲在公寓樓下轟然響起。

十幾輛噴涂著天衡局特種徽標的黑色越野戰鬥車狂飆而至，雪白且刺眼的強光探照燈光束，在一瞬間將原本黑暗的天台照耀得亮如白晝。

「所有戰鬥人員注意！分段封鎖出口！不許動，天衡局辦事！」

幾名全副武裝、穿著戰術背心的精英調查員率先衝上天台。

林若溪背負著那柄象徵著極致寒氣的冰藍色寶劍，表情清冷得如一朵在冰原綻放的雪蓮。當她穿過滾滾濃煙，看見那個正精疲力竭地靠在破水塔下、手中提著一段彎曲鋼管的蘇晨，以及那一地橫七豎八、部分殘骸甚至被切得不成人形的怪物殘肢時，這位向來以冷靜著稱的局長接班人，整個人在原地徹底石化。

「蘇晨……你真的還活著？」她的聲音中第一次帶上了這種讓下屬們不敢置信的、明顯由於極度擔憂而產生的輕微顫抖。

她原本是在接到舊城區出現「蛻變者」秘密獵殺小隊的加密警報後，火速親自帶隊前來的。在她的預想中，面對那種成名已久的刺客精銳，蘇晨這個剛覺醒不久的「大白羊」，恐怕此時能剩個全屍都是奢望。她幾乎是帶著幫他收屍並報仇的一腔怒火狂飆而來的。

可現在，呈現在她眼前的是什麼？

是一支具備一名二階巔峰領頭、三名資深覺醒者組成的殺手團隊，全軍覆沒。
而那個她一直覺得需要呵護與引導、僅有「二階初階」實力的快遞少年。雖然看起來油盡燈枯、狼狽不堪，卻依舊像一柄永不折斷的斷劍般，站在這片廢墟之巔。

「隊長大姐，下次你要是再晚來三分鐘，我就真的得去跟閻王爺商量下一次『跨界配送』的清單了。」

蘇晨抬起滿是灰塵與血跡的臉，對著林若溪露出了一個比哭還要難看的憨厚老實人笑容。他頗有些心疼地指了指下方還在不斷噴吐火舌的公寓三樓：
「局裡那些經費……管報銷私人物品損失嗎？我那房子租金可是咬牙交了一整年的啊，隔壁的房東太太非把我切了掛晾衣架上不可。」

林若溪靜靜地看著他。那雙清冷的眼眸中，除了一抹釋然的放鬆外，第一次浮現出了深深的震撼，以及一絲極其隱秘的忌憚。

身為天衡局的高層，她敏銳地察覺到那被切得平滑如鏡的水泥斷口，以及殘留在空氣中那股即便是在爆炸風暴中也經久不散的、霸道到了極點的未知劍意。這絕對不是目前地球已知的、任何一門古武或覺醒靈技所能達成的效果。

她在那一刻終於意識到，這個看起來吊兒郎當、甚至有點守財奴性格的少年，其背後隱藏的那座名為「傳承」的冰山，足以讓整個臨海市、乃至此刻正處於旋渦中心的省城分局都為之瘋狂與顫慄。

「醫療組封鎖一號現場！任何人未經我允許，不得擅自接觸戰鬥殘核！」

林若溪猛然轉過身，她的動作冷酷且決斷。她大步流星地走到虛弱的蘇晨面前，一言不發地脫下自己身上那件象徵著指揮官身份、還帶著淡淡體溫與清冷香氣的白色披風，親手將其罩在了蘇晨那滿是傷痕、鮮血與黑灰交織的肩膀上，壓低聲音，只用兩個人能聽見的頻率快速叮囑道：

「你這種異於常理的底牌，在沒有徹底成長起來前，以後哪怕是生死關頭，也絕對不要在公共偵查儀器前輕易展示。現在局裡的局勢不穩，跟我走，換一個絕對安全的地點進行修復。」

蘇晨乖巧地點了點頭，感覺原本緊繃的那根神經在見到林若溪的剎那徹底鬆弛。那種如潮水般襲來的虛脫感瞬間將他吞沒，他的眼皮重得像是壓了兩塊鉛石。

「那獎金……記得幫我向局長申請……雙倍賠償……」

話還沒說完，他整個人就這樣直勾勾地失去了意識，一頭栽進了這朵平日裡生人勿近的「冰山雪蓮」那微涼且帶著顫意的懷中。

林若溪的身體如觸電般猛地一僵，那張即便面對妖獸潮也不曾變色的冷嬌臉龐上閃過一抹複雜的紅暈。她下意識地伸出手，輕輕托住了這個雖然瘦削卻彷彿扛起了整片黑夜的少年，雙手環過他的腰際。
感受到蘇晨身上那股即便陷入昏迷、也依然透著一股寧折不彎、銳利到極窄處的強大劍骨，她抿緊了嘴唇，冷冽的目光掃向夜空深處那幾處不懷好意的視線。

「蛻變者，你們的手伸得太長，這一次……確實過界了。」
