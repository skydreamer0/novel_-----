# 第二十九章 · 局長收場

三日後，龍池戰場的硝煙已散。

但這場戰鬥餘波，卻以海嘯之勢席捲了整個大夏國。

臨海第一人民醫院地底的秘密被曝光，周家與蛻變者勾結的證據鏈徹底完整。在省城總部，原本那些試圖打壓林家、奪取蘇晨功勞的政客們，此時全部噤若寒蟬。

因為他們面對的，是一個能在正面戰場擊殺半步五階主教的……怪物。

「蘇晨，你這是在……打包行李？」

分局辦公室內，秦老坐在蘇晨對面，那雙渾濁的眼中透著一股前所欲有的光彩。老爺子今天特意穿了一身嶄新的教官服，勳章掛滿了胸口。

「是啊，秦老。龍池那邊炸了，我的宿舍也沒了。我這人認床，還是想回蛇巷住。」蘇晨一邊往儲物袋裡塞著各種補給品，一邊隨口答道。

「你現在可是全大夏國最炙手可熱的功臣。」秦老敲了敲桌上的任命書，「總部那邊已經批了，特聘你為『大夏國第一巡察使』。級別……跟我持平。年薪、資源、美女，只要你開口，沒人敢說個不字。」

「秦老，您覺得我在意那些嗎？」

蘇晨停下手中的動作，轉過頭，眼神清澈而深邃，「對我來說，最好的獎賞就是世界太平，然後我能安安靜靜地送我的快遞。」

秦老愣了一下，隨即哈哈大笑。

「好！好一聲送快遞！真沒想到，我秦滄海活了七十載，最後看人看最準的一回，居然是看走眼的一回。」

秦老站起身，從懷裡掏出一枚紫金色的印章放在桌上。

「這是你應得的。即便你不想當巡察使，這印章你收著。在大夏境內，見此章如見我親臨。另外，周家的產業已經被查封，其中位於蛇巷周邊的那三條街，我做主，全部劃歸到你的名下。」

「三條街？」蘇晨眼睛亮了，「能收房租的那種？」

「想收多少收多少。」

「成交！」蘇晨毫不猶豫地收起印章，嘿嘿一笑，「秦老，您老人家果然懂我。」

就在這時，林若溪推門而入。

她已經換回了那身幹練的白色隊長制服，雖然臉色依舊清冷，但看向蘇晨時，眼神中分明多了一抹柔和。

「局長。護送蘇教官的回程車隊已經備好了。」

「不用護送。」蘇晨擺擺手，「我自己溜溜達達就回去了。對了，林隊長，你現在……官復原職了？」

「總部撤銷了對我的處分。並且，提拔我為……臨海分局代局長。」林若溪看了一眼秦老。

「我也該退休嘍。」秦老伸了個懶腰，「這位置坐了三十年，膩味了。以後這臨海市，就是你們年輕人的天下了。」

……

當晚。

蘇晨回到了熟悉的蛇巷。

趙記雜貨店的燈依然亮著。老趙還是那副懶散的模樣，正對著一台剛換上的、蘇晨送的新電視機看著肥皂劇。

「喲，巡察使大人回來了？」老趙頭也不回地刺了一句。

「趙叔，我還是那個送快遞的。」蘇晨一屁股坐在藤椅上，「話說，那三條街的房租，您以後幫我代收？」

「滾！老子不當管家。」老趙雖這麼說，嘴角卻勾起了一抹笑意。

蘇晨獨自走進地底密室。

他看著識海中那個已經閃成金色的系統圖標，深吸一口氣。

「系統，開啟 final 章節審核。卷一目標……全部達成。」

```
  【卷一：覺醒·完結審核中...】
  - 關鍵目標 1：覺醒系統（達成）
  - 關鍵目標 2：實力跨入三階（達成）
  - 關鍵目標 3：擊潰周家與源質主教（達成）
  - 關鍵目標 4：獲得位面門檻鑰匙（達成）
  
  【系統評價：S+】
  解鎖最終獎勵：[萬界傳送陣·錨點定位]。
```

蘇晨看著手中的那個散發著金色光暈的羅盤，心跳如雷。

他知道，臨海市的故事告一段落，但更廣闊、更瑰麗、也更危險的諸天萬界，正向他敞開大門。

下一個目的地：劍修世界。

他得去看看，葉孤楓那個瘋子，到底把他的快遞驛站建成什麼樣了。
