# 第二十五章 · 劍種入體

夜色已深，喧囂的臨海市在大雨過後透著一股泥土的芬芳。

蘇晨獨自坐在雜貨店的地底密室中。這兩天發生的事，像是快進的電影，讓他有些目不暇給。

「系統，我要跟葉孤楓進行一次深度交易。」

```
  【正在建立深度鎖定...】
  由於上次提供『上帝之杖』的人情，對象葉孤楓主動發起了一項『本命共享』。
  ─────────────
  交易內容：
  - 葉孤楓提供：九天劍意真傳·劍種。
  - 對方需求：協助他在劍修世界建立一座『萬界快遞驛站』。
  ─────────────
```

「驛站？他在劍修世界開快遞公司？」蘇晨有些哭笑不得。

「蘇兄。」光幕中的葉孤楓，此時盤坐在一座孤峰之上，手中正把玩著一罐沒拆封的可口可樂。「我已決定重建紫霄宮。但此界資源枯竭，尤其是那種能讓人神魂愉悅、暫忘修煉之苦的『美食』與『靈泉』。我想借你的手，讓這幫老古董也開開眼。」

「只要能保證我不被他們抓去當小白鼠，開驛站沒問題。」蘇晨咧嘴一笑，「不過，你說的『劍種』是什麼？」

「是我劍修一脈最珍貴的東西。」葉孤楓神色一正，指尖彈出一粒似虛似實、散發著無盡星光的種子。「它能寄宿在你的丹田中，將你全身的靈氣轉化為最純粹的劍氣。即便你遠在另一個世界，只要我的劍心不滅，這劍種便能賦予你越階斬敵的破壞力。」

```
  【系統檢測：SSS 級珍寶『無垢劍種』】
  融合成功率：75%。
  融合風險：經脈重組的劇痛。
  ─────────────
```

「來吧，劇痛什麼的，我早就習慣了。」

蘇晨大手一揮，同意交易。

那一瞬，他感覺胸口像是被一顆天外流星擊中。那枚半透明的劍種穿透虛空，直接扎進了他的丹田旋渦。

「轟——！！」

原本充斥在蘇晨體內的《長青吐納術》靈氣，在接觸到劍種的一剎那，竟然像是見到了君王的臣子，瘋狂地臣服、壓縮、重組。

「唔啊……」

蘇晨全身汗如雨下，他感覺自己的每一根血管、每一寸骨骼，都在被無數細小的利刃瘋狂切割、再修復。

三階鍛體的肉身在此刻發揮了巨大的作用。那強悍的恢復能力，與劍種的破壞力形成了一種微妙的動態平衡。

半個小時後。

蘇晨的身子猛地一震，雙眼中射出兩道長達半尺的吞吐劍芒。

他抬起手。

不再需要劍。

他指尖微動，一抹青翠中帶著寂滅黑色的劍氣，便在指尖縈繞。這股氣息，比之前的「瞬殺」更穩固、更霸道。

「這就是……劍氣？」

蘇晨對著牆角的一塊特種合金盾牌輕輕一點。

「嗤。」

盾牌像是豆腐般被無聲無息地洞穿，缺口圓潤如鏡面。

「蘇兄，感覺如何？」葉孤楓在對面舉起可樂，像是在慶祝。

「感覺……我現在能接下秦老三掌而不死。」蘇晨長舒一口氣，眼神中充滿了自信。

與此同時。

醫院後山的廢墟中。

原本已經被「上帝之杖」徹底抹除的坑洞裡，一隻慘白的手掌，突然從焦黑的泥土中探了出來。

那隻手掌上，長滿了紫色的膿包，五指之間，還夾著一塊神祕的青銅鏡碎片。

「蘇……晨……」

嘶啞的聲音在空曠的廢墟中迴盪，宛如來自幽冥的索命之聲。

而這一切，蘇晨並不知曉。

他正在系統商店裡瘋狂掃貨：大量太陽能發電機、高畫質投影儀、各類香辛料、甚至還有幾台退役的小型工業噴氣機。

「葉兄，既然要做，就做全臨海市……不，全劍修世界最豪華的快遞驛站。下一場交易，我給你帶點更有趣的——『會飛的鋼鐵坐騎』。」

蘇晨笑得像個狐狸。

一場跨越世界的商業版圖與實力飛躍，正式拉開帷幕。
