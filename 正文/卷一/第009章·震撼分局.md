# 第九章 · 震撼分局

天衡局臨海分局，深度醫療康復中心，三號醫務室。

空氣中瀰漫著一股刺鼻的消毒酒精味，以及一種唯有覺醒者才能嗅到的、由淡藍色靈能修復儀散發出的淡淡草木香氣。修復儀正發出規律且機械的嗡嗡顫鳴，柔和的半透明光幕如流水般輕輕拂過蘇晨赤裸的胸膛，修復著那些細微的肌肉拉傷。

「奇怪……這太不符合生命科學了。」

負責檢查的女醫官扶了扶厚重的黑框眼鏡，對著終端投影屏上那如過山車般劇烈波動的數據曲線反覆比對。她的眉頭鎖得死死的，隨後抬起頭，目光中充滿了審視：「蘇晨，就在剛才的五分鐘內，你的皮下細胞活性呈現出一種自毀式的下降，這通常是大規模消耗潛力的徵兆。但詭異的是，你的心跳和各項生命體徵卻穩固得像頭正值壯年的狂暴牛。你老實交代，你剛才是不是在戰場上偷吃了什麼高階的基因回復類禁藥？」

蘇晨此時正半靠在冰冷的金屬病床上，手裡抓著一塊乾硬的醫用維生素餅乾，咬得嘎吱作響。他含糊不清地嘟囔著：「醫生，您也太看得起我了。我就一送快遞的，平時連外賣都捨不得點，哪買得起那種動輒幾十萬一瓶的高級貨？可能是我平日裡跑樓梯跑得勤，心肺功能早就被生活給磨鍊出來了吧，這叫窮人的代謝快。」

「代謝快能讓那種三公分深的撕裂傷在半小時內結痂脫落？」女醫官沒好氣地斜了他一眼，顯然對這種滿是破綻的鬼話一個字也不信。但根據儀器多重檢測顯示，蘇晨體內的靈氣波動確實停留在極其不穩定的「二階初階」，這完全符合一個剛剛在生死邊緣強行突破、根基尚且虛浮的覺醒者特徵。

畢竟，在天衡局的內部檔案記錄裡，蘇晨這傢伙兩天前才剛剛覺醒。

兩天時間，從覺醒到逆境突破二階。這在臨海市這種靈氣貧瘠的二線城市，已經足以讓那些世家子弟驚掉下巴，勉強能掛上「一級天才」的標籤了。雖然比起省總局那些含著靈石出生的妖孽還差了一些，但考慮到他那被評定為「垃圾」的D級潛力值，這簡直是當代醫學的一場大型奇蹟。

「行了，別在我這兒裝可憐了，你可以出去了。林隊長在頂層辦公室等你，表情看起來不太好。」女醫官收起精密檢測儀，語氣中多了一抹難以掩飾的探究與好奇，「對了，順便告訴你一聲，隔壁那個周公子……他現在還在心理諮詢室吸著高級純氧呢。聽說是因為看見那三顆狗頭離他的脖子只有十公分，被嚇得靈壓紊亂，現在還在胡言亂語。」

蘇晨咧嘴一笑，露出整齊的白牙：「那是，畢竟那種三個腦袋、滿嘴腐臭的品種，確實挺費空氣的。」

走出醫務室，一路上遇到的那些身穿黑色作戰服的覺醒者鬥員們紛紛駐足側目，原本嘈雜的走廊瞬間安靜了不少。

「看見沒？就是那小子，穿著外賣服單殺了三首幽冥犬領主。」
「聽說還是臨場突破，那一指射出的指芒，嘖嘖，連軍用級的水泥承重柱都給射出了個對穿。」
「真假啊？他檔案上不是說潛力只有D級嗎？這年頭D級都這麼猛了？」
「誰知道呢，沒準是覺醒了什麼冷門到極點的自毀爆發系靈技，這輩子估計也就這一次高光時刻了。」

聽著四周那些夾雜著嫉妒、懷疑與震驚的竊竊私語，蘇晨的表情始終如一地平靜，步履間甚至還帶著點快遞員特有的輕盈與急促。但他的內心卻如同精密電腦一般在瘋狂運算著。

這次在倉庫裡的表現確實有些過火，甚至可以說是直接把「我是神祕高手」寫在了臉上。但如果當時不殺了那頭被改造過的畜生，不只是他，所有新人都得淪為飼料。既然已經把牌打出去了，那就得想辦法編織一個完美的、充滿「偶然性」和「不可複製性」的謊言，把這一切圓過去。

「咚咚。」

蘇晨敲響了林若溪辦公室那扇厚重的防彈大門。

推門而入，一股清冷、幽遠的檀香味夾雜著淡淡的冷香撲面而來。林若溪此刻早已換下了那件被火硝與黑灰弄髒的白裙，取而代之的是一套剪裁極其幹練、勾勒出她驚人身段的黑色高級調查員制服。她正靜靜地坐在真皮轉椅上，對著辦公桌上一塊正散發著不詳氣息的黑色結晶失神。

「坐。」

林若溪抬眼看向蘇晨，那雙向來冰冷如秋水的眸子裡，此時竟然多了一抹連她自己都未察覺的複雜情緒。

蘇晨倒是沒什麼覺悟，大大咧咧地一屁股坐在昂貴的牛皮沙發上，眼睛直勾勾地盯著桌上那個印有天衡局徽記的特製信封：「林隊長，咱們事先說好的那筆辛苦費……」

「五萬塊已經打入你的工資卡了。另外，鑒於你單殺二階巔峰妖獸，挽救了六名預備役覺醒者的生命，局裡特別批准了三萬塊的『特級戰鬥補償款』。」林若溪將信封輕輕推了過來，裡面裝著一枚二等功勳章和一張鑲嵌著金邊的特製儲值卡。

「謝謝隊長！隊長英明神武，大氣磅礴！」蘇晨雙眼發亮，這可是整整八萬塊啊！他送兩年快遞不吃不喝估計也就攢下這麼多。

「獎金拿了，但有些話你得聽進去。」林若溪伸出纖長的食指，輕輕點了點那顆死氣沉沉的黑色結晶。

結晶在室內燈光的折射下，翻湧著一種粘稠的、像是腐爛血液般的幽暗紅光。

「這顆結晶的鑑定報告就在剛才被加密傳輸到了我的終端。」林若溪的語氣變得前所未有的凝重，「這不是天然形成的妖核，而是一種人為干預、精確培育的『生命源質膠囊』。有人在那些妖犬還是幼崽時期，就將這種充滿惡意的種子植入了牠們體內。牠們透過瘋狂燃燒宿主的生命潛能，在短短數周內強行催生出遠超自然界規律的二階巔峰戰力。」

「人工培育？像種韭菜那樣？」蘇晨換上一副大受震撼、甚至帶著點滑稽色彩的驚恐表情，「誰會這麼瘋狂？養這種怪物除了拆家和嚇唬人，還能幹嘛？」

「殺人，以及動亂。」

林若溪雙眼微瞇，冷冽如霜，「昨晚如果不是你這變數出現，那六名具備成長潛力的新人會全部死在倉庫深處。而這，正是那個秘密組織想要看到的血腥祭禮——他們正在利用臨海市那處不穩定的空間裂隙，進行這種『速成突變技術』的實戰環境測試。我們暫且將這個隱藏在暗處的組織命名為……『蛻變者』。」

「『蛻變者』……這名字聽起來就充滿了那種反派組織的廉價感。」蘇晨小聲嘀咕。

「所以，局務委員會決定，即刻正式提升你的內部保密等級與行動權限。」林若溪站起身，緩步走到蘇晨面前，那股若有若無的清冷芬芳讓蘇晨的呼吸不由得屏住了一瞬。

「蘇晨，雖然數據說你潛力低下，但你展現出來的實戰意識、心理素質以及對危險的直覺，是我見過最強大的。這張卡裡除了獎金，還有一份特殊的通行證——從今天起，你持有臨海分局的『二等獨立搜查權』。擁有這份權限，你可以隨時調用本市二階以下所有妖獸及覺醒者的動向資料。

當然，這也意味著你正式踩進了這場深不見底的漩渦。周家那邊我也得提醒你，周家明雖然被你救了一命，但他自尊心極強且心胸狹隘，周家的那些老傢伙可能會以『感謝恩人』的名義來試探你的底細，你自己長個心眼。」

蘇晨隨意地擺擺手，站起身上下打量了一下那枚勳章：「周公子那是受了驚嚇，我理解。對了隊長，既然是立了功，那這塊『好看』的黑色石頭，應該不歸我私有了吧？」

「按局裡最高封印規定，這種危險源必須歸公。在完成所有光譜分析後，會由總局下達命令進行銷毀。」林若溪看著他，眼神犀利，「你似乎對這東西表現出了不尋常的熱情？」

「哪有，我就是覺得它紅通通的，想著拿回去磨個吊墜送給孤兒院的小朋友玩。」蘇晨嘿嘿一笑，撓撓頭轉身告辭，臉上的表情那叫一個憨厚老實。

走出天衡局那棟毫不起眼、實則戒備森嚴的灰色大樓，蘇晨感受著兜裡沉甸甸的金邊屬卡，嘴角在那無人的巷道口猛然拉開一個危險的弧度。

「系統，那結晶裡的那些髒東西，你搞定了沒？」

```
  【萬界交易系統·升級反饋】
  ─────────────
  偵測到高純度「異變源源質」能量採樣完畢。
  代碼解析進度：100%。
  反向吞噬能量反饋中...宿主體內積累的雜質已被初步淬煉。
  檢測到大量優質能量溢出，系統經驗池已達上限。
  
  正在進行核心架構升級...
  升級成功！
  【萬界交易系統 v2.0·正式版】全面開啟。
  ─────────────
```

「好戲終於開場了。」蘇晨心中暗爽不已。他無視了路邊那些昂貴的餐館，加快腳步回到了自己那間雖然侷促、卻佈滿了防盜隔離設置的小單間。

反鎖好房門，拉上厚厚的遮光窗簾，他第一次感受到與系統那種血脈相連的融合感。

```
  【萬界交易系統 v2.0】
  ─────────────
  宿主：蘇晨
  管理員權限：Lv.2
  當前真實修為：二階·凝氣（大圓滿）
  
  新版塊解鎖通知：
  1. 【跨界搜索】：宿主可根據具體需求，主動搜索諸天萬界中特定的交易對手（冷卻CD：24小時）。
  2. 【亞空間儲物間（初始1立方米）】：可實體化存儲任何無生命跡象的交易物資，且內部時間近乎凝固。
  3. 【解析偵查·真視之眼】：可透視現實世界中不高於宿主兩個小境界的所有目標基本靈氣結構與屬性。
  
  當前連線交易對象：
  - 靈草位面：李小苗（友誼度：生死之交，好評度：100%）
  - 劍修位面：葉孤楓（新搜索目標！正在定向校準中...）
  ─────────────
```

「亞空間储物間！這簡直是為我這『快遞員』量身定做的神器啊！」蘇晨激動得差點在床上打個滾。身為一個整天和包裹打交道的底層勞動者，他太清楚「隨身空間」這四個字代表了多麼恐怖的戰略價值和暴利空間。

不過，很快他的注意力就被那個新出現的人物名字吸引了。

「劍修世界？葉孤楓？」

蘇晨瞇起雙眼，舔了舔因為亢奮而略顯乾燥的嘴唇。在這個處處充滿危機、隨時可能有高級妖獸破牆而入的時代，唯有純粹的、極速的殺傷力才是安身立命的保障。《長青吐納術》雖然中正平和且根基穩固，但正如李小苗所說，這只是一門養生功法，在搏命時缺乏一錘定音的必殺技。

「開啟定向搜索。目標：劍修世界——葉孤楓。」

```
  【跨界信號傳輸中...】
  ─────────────
  搜索連線成功。
  目標當前生存狀態：極度悽慘。
  詳細說明：葉孤楓，曾是劍修大世界凌雲劍派的絕世劍子。因天賦過於妖孽遭同門陷害，現被挑斷全身經脈，囚禁於萬劍崖之底的禁靈魔窟。
  其目前極度渴望物資：一種能打破「禁靈鎖鏈」禁錮，並重新接續經脈、煥發肉體生機的『仙家神藥』。
  ─────────────
```

蘇晨看著光幕上的文字，腦袋轉得飛快。

在那個世界的所謂「神藥」，可能需要採集萬年雪參或者甚至要消耗一名大能的壽元。但在蘇晨眼裡，這種需求似乎存在著巨大的、廉價的替代方案。

「不對，單純的補充靈氣對他那種經脈盡碎的人來說反而是慢性毒藥。」蘇晨摸著下巴在房間裡來回走動，「李小苗之前提起過，她們藥田中生長在一種特殊的『地心玉髓』，那是吸收了大地精華的溫潤之物，最擅長修補破碎的經絡，而且對靈氣的要求並不苛刻。但重點是，那玉髓生長在幾千米深的毒瘴深溝裡，小苗那點修為根本下不去。」

想到這兒，蘇晨嘴角勾起一抹奸商般的弧度，他低頭看了一眼手中那張剛剛到手的八萬塊獎金卡。

「小苗下不去，那是因為她們那個世界的人太過依賴御劍飛行。在現代科技面前，幾千米深的懸崖算什麼？」

他猛地推開門，大步流星地衝向了樓下不遠處的一家大型五金機電城和專業戶外俱樂部。

「既然哥現在有了儲物空間，那就給異界的小朋友們，帶去一點來自二十一世紀真正的『工業革命震撼』吧。」

當晚，蘇晨在各家店舖豪擲了一萬多塊。
高強度航空級合金攀爬支架、鋰電池驅動的工業級脈衝衝擊鑽、碳纖維靜力繩索、多功能戰術折疊軍刀、甚至還有兩台亮度驚人的、足以晃瞎妖獸眼睛的超強光LED搜索探照燈。

「小苗，蘇大哥給你送『探險快遞』來了。這次的訂單大，咱們得好好的『合作一把』。」

在狹小的、充盈著靈氣的出租屋內，一場跨越不同次元、規模前所未有的「連鎖貿易戰」，正悄無聲息地拉開了帷幕。
