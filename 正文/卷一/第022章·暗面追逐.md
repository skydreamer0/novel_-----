# 第二十二章 · 暗面追逐

雨越來越大，像是要把整座臨海市淹沒。

「砰！砰！砰！」

三聲沉悶的槍響打破了舊城區的寧靜。蘇晨所在的越野車後窗瞬間粉碎，特製的合金彈頭擦著林若溪的耳際飛過，將擋風玻璃震出了蛛網狀的裂痕。

「這幫瘋子，居然連穿甲彈都用上了！」林若溪咬著牙，瘋狂轉動方向盤，在狹窄的巷弄間左右蛇行。

「林隊，低頭！」

蘇晨猛地起身，半個身子探出天窗。他的手中並沒有槍，而是握著三張剛從系統空間取出的、散發著淡淡火系靈氣的符籙。

那是他在東北區戰場「撿漏」換來的三階·爆炎符。

「雖然不是劍意，但這玩意的動靜，應該夠你們喝一壺了。」

蘇晨眼神一冷，指尖靈力注入。

「射！」

三張符籙化作三道紅色的殘影，在大雨中劃出詭異的弧度，精準地沒入了後方追兵的車底。

「轟——！！」

三團巨大的火球在雨夜中爆開，強烈的衝擊波竟然短暫地排開了方圓十米內的積水。兩台重型越野車當場被掀翻，在地上滑行了數十米，撞入了一間路邊的包子店，火光沖天。

然而，第三台車卻在爆炸的瞬間，車身竟然詭異地化作了液態般的紫色流體，輕而易舉地穿過了爆炎的中心，毫髮無傷。

「源質機甲？」林若溪透過後視鏡，聲音有些絕望。

那台越野車外殼迅速重組，頂部的旋轉機槍噴吐出三米長的火舌，瘋狂掃射。

「叮叮噹噹！」

金屬被撕裂的聲音在蘇晨耳邊炸響。

「系統，開啟解析偵查！鎖定駕駛員位置！」

```
  【目標鎖定：三點鐘方向駕駛艙】
  由於源質屏蔽，建議使用『瞬殺』穿透模式。
```

蘇晨一把拽住正要停車決命的林若溪。

「別停！衝向老趙的雜貨店，剩下的交給我！」

他右手五指虛握，體內的「歸墟劍意」在此刻發出了興奮的顫鳴。自從他在擂台大賽後，這股劍意似乎也因為吞噬了少許周家明的源質而發生了某種奇妙的進化。

「歸墟，瞬殺·影。」

蘇晨的身形在那一刻竟然與雨幕合而為一。他從急速行駛的車頂一躍而下。

在外人眼裡，他就像是一抹消失在雨中的影子。

唰。

那台紫色的越野車發出刺耳的剎車聲，駕駛員顯然也沒想到這個「快遞員」居然敢直接跳車反擊。

然而，當他透過感應器看到那個站在路中央的身影時，已經晚了。

蘇晨雙手合十，指尖凝聚出的劍芒在漆黑的雨夜中格外耀眼。那一劍，沒有華麗的招式，只有最純粹的速度與寂滅。

「噗！」

劍芒輕而易舉地切開了被源質加固過的特種合金裝甲，連同駕駛室內的那名「蛻變者」刺客，一齊斬成了兩半。

越野車失去平衡，在空中翻轉了三圈，重重砸在一根路燈杆上。

整片街道重歸死寂。只有雨水沖刷著金屬殘骸的聲音。

蘇晨半跪在积水中，大口喘氣，右手隱約有些顫抖。連續動用三階劍意對現在的他來說，負擔依舊巨大。

「走。」

林若溪將車倒了回來，伸手拉起蘇晨。

兩人棄車步入蛇巷。

此時的蛇巷，安靜得有些不正常。所有的燈光都熄滅了，唯有巷口那間「趙記雜貨店」的門縫裡，還透著一絲昏黃。

老趙依舊蹲在路邊，雨水順著他的雨帽滴落，他頭也不回地說：「殺了三個，跑了一個。蘇晨，你這收尾工作做得不太乾淨啊。」

蘇晨愣了一下，隨即苦笑：「趙叔，這都什麼時候了，您老還有心思開玩笑。」

「誰跟你開玩笑了？」老趙站起身，拍了拍手上的污泥，轉頭看向林若溪，「林教官，麻煩你在門口守著。接下來的內容，天衡局的人……不方便聽。」

林若溪看著老趙那雖然駝背、卻透著一股隱龍般氣勢的身架，眼神中閃過一抹複雜。

「好。但我只給你們三十分鐘。」

林若溪持劍立在巷口，像一尊安靜的守門石獅。

蘇晨跟著老趙進入了那間滿是陳舊氣息的雜貨店。

老趙走到收銀台後，從那個擺滿了「鹽漬梅子」的罐子後面，再次摸索。這一次，他不是拿東西，而是猛地按下了一個隱蔽的紅色按鈕。

「咔嚓。」

整個貨架突然向兩側緩緩分開，露出了背後一條漆黑深邃、一直通向地心深處的螺旋階梯。

「趙叔，這是？」

「這是當年我們那批老傢伙，為這座城市留下的最後一條生路。」老趙聲音變得無比嚴肅。

「蘇晨，你手裡那塊殘片，根本不是什麼神廟鑰匙。它是……一個信號發射器。」

蘇晨心中一震，猛地掏出那塊殘片，發現它的震動頻率確實在變快。

「它在呼喚外面的東西？」

「不。它在呼喚……地底下的東西。」老趙眼中閃過一抹痛苦的回憶，「臨海市自古以來就不是什麼繁華之地。它是……鎮壓一個魔神頭顱的祭壇。而現在，蛻變者想要喚醒它。」

「蘇晨，我知道你手裡有個神祕的門道，能換到不少好東西。」

老趙盯著蘇晨的眼睛，語氣中帶著一絲瘋狂，「給你十五分鐘，不管你用什麼辦法，給我召喚出足以毀掉整座地底祭壇的火力！否則，明天太陽升起的時候，臨海市將不再有活人。」

蘇晨感受著老趙手掌傳來的溫熱，那是長輩對後輩最後的孤注一擲。

他閉上眼，心中默念：「系統，給我接通所有高等級位面。我要開啟……萬界大甩賣！」

「這一次，我不要錢。我要……禁忌級別的，大規模殺傷性武器！」
