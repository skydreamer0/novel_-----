# 第四章 · 覺醒者的世界

深夜，臨海市舊城區的小公寓內。

蘇晨盤腿坐在床上，面前堆滿了從超市搜刮來的「戰略物資」。那幾瓶可樂在昏黃的燈光下折射著琥珀色的光澤，對他來說是消暑飲料，但在那個神祕的靈草世界，這可是能引起藥農驚呼的「仙露」。

「系統，呼叫靈草世界，李小苗。」

蘇晨在腦海中輕聲下令。

```
  【萬界交易系統】
  ─────────────
  連線中...
  連線成功。
  交易對象：靈草世界·李小苗（Lv.1）
  對象狀態：正在晾曬藥草，看起來很餓
  ─────────────
```

光幕閃爍，蘇晨再次看見了那個不過十四五歲、扎著雙馬尾的瘦弱女孩。李小苗正坐在一塊大石頭上，手裡拿著一塊乾巴巴的草根在啃，眼神中透著一絲疲憊。

「蘇大哥！」一看到蘇晨的身影出現在那虛幻的窗口中，李小苗的圓臉蛋立刻亮了起來，連手裡的草根都顧不得啃了，「你來啦！那天的『仙露』……我師父喝了以後，說那裡面蘊含了一種極其奇特的氣泡能量，竟然讓他的瓶頸都有了鬆動的跡象！」

蘇晨聽得嘴角一抽。可樂裡的二氧化碳居然被當成了氣泡能量？

「小苗，今天我給你帶了點新花樣。」蘇晨從袋子裡摸出一包紅燒牛肉麵，「這叫『龍鬚百味羹』，只要用開水一沖，香氣能傳出三里地。」

李小苗看著那精美的包裝袋，眼珠子都快掉出來了：「這……這這這……這包裝，難道是傳說中的水晶琉璃紙？」

「……差不多吧。」蘇晨臉不紅氣不喘地編著，「這東西在我們這兒也是稀罕貨。」

「那、那蘇大哥想要換什麼？」李小苗緊張地搓著手，「我這兒今天的清心草都賣給採藥商了，只剩下幾株還沒長成的……」

蘇晨笑了山一笑，進入正題：「我不換草藥。你們那兒有沒有什麼基礎的修煉功法？不需要大門派的祕傳，只要是能引導靈氣入體、淬煉經脈的就行。」

李小苗愣了一下，隨即有些洩氣地垂下頭：「基礎功法啊……我們這兒的人，十個有九個都會一點。我手裡就有一本《長青吐納術》，是採藥人的入門功夫。但是蘇大哥，這東西不值錢的，滿大街都是……用這麼珍貴的『百味羹』換，你太虧了。」

「沒事，我這人做生意講究眼緣。」蘇晨大手一揮，將一整箱泡麵和三瓶可樂推向了交易窗口，「這一箱都給你，再加上三瓶仙露。換你那本《長青吐納術》，換不換？」

李小苗的眼眶一下子就紅了。她覺得蘇大哥簡直是天底下最大的善人，肯定是看她修煉辛苦，才用這種藉口補貼她。

「換！換！蘇大哥你等等！」

李小苗從懷裡掏出一本泛黃的線裝書，虔誠地放進了虛空漣漪中。

```
  【交易成功】
  ─────────────
  交易對象：靈草世界·李小苗
  您付出：紅燒牛肉麵*12、可樂*3
  您獲得：【長青吐納術】（殘卷）
  交易評級：★★★★★（完美撿漏！）
  ─────────────
  系統提示：檢測到《長青吐納術》與宿主身體契合度極高，系統已自動優化並補完殘缺部分。
```

蘇晨手裡沉甸甸的，那本線裝書入手微涼，封面上寫著三個古樸的大字：吐納術。

他翻開第一頁，腦海中突然響起一聲清脆的嗡鳴。

那是系統的力量。

原本晦澀難懂的古文和經絡圖，在他眼裡瞬間活了過來。無數金色的光點在書頁上跳躍，最後化作一股龐大的信息流，直接灌注進他的識海。

```
  【長青吐納術·系統優化版】
  等級：一階基礎
  效果：自動感知周身靈氣，強化經脈韌性，修煉速度+300%
```

蘇晨倒吸一口涼氣。

普通覺醒者的修煉，是需要自己冥想、捕捉空氣中游離的靈氣，然後費勁地一點點納入體內。

但他這本「系統優化版」，竟然增加了「自動感知」和「300%修煉速度」！

他立刻按照腦海中的指引，盤腿坐好，雙手掐出一種奇異的印記。

一秒，兩秒……

「呼——」

長久的一口深呼吸後，蘇晨感覺四周的空氣都變了。

如果說之前的「一階·感知」是讓他穿透了濃霧看見世界，那麼現在，他就像是在濃霧中啟動了一台強力吸塵器。

那些星星點點的白色靈光從四壁鑽進來，爭先恐後地湧進他的毛孔。每一寸肌肉、每一塊骨骼都在這股暖流下發出愉悅的顫慄。

他感覺自己體內的那股微弱靈氣，正在以肉眼可見的速度壯大。

「難怪老趙說我是D級資質……」蘇晨一邊修煉一邊感慨，「普通人修煉這功法，估計像拿勺子喝水。但我現在像是在接高壓水龍頭。」

短短一小時，他體內的靈氣總量就翻了一倍。

更驚人的是，他感覺自己的皮膚表面隱約滲出了一些黑色的黏稠物質，散發著淡淡的腥味。

洗髓伐毛！

雖然只是最基礎的層次，但這說明他的身體素質正在發生質的飛躍。

……

第二天一早。

蘇晨是被一陣嘈雜的聲音吵醒的。

他睜開眼，第一感覺是眼前的世界變得空前清晰。遠處路燈上的蜘蛛網、樓下早餐攤蒸籠升起的白氣，甚至隔壁鄰居翻身的聲音，都纖毫畢現。

他看了一眼手臂，昨晚排出的雜質已經結成了乾皮。

洗完澡後，蘇晨看著鏡子裡的自己。他的身材並沒有像健身房教練那樣爆裂，但肌肉線條變得極其流暢，皮膚表層透著一種健康的青玉色，整個人精氣神飽滿到了頂點。

「這就是二階的感覺嗎？」他握住拳頭，感受著體內澎湃的力量。

雖然還沒正式突破到「二階·凝氣」，但他現在的靈氣儲存量，絕對已經觸摸到了那個門檻。

他穿好快遞制服，路過蛇巷時，發現那裡圍了不少人。

在一堵斑駁的紅磚牆上，一張落款為「天衡局臨海分局」的藍色告示格外醒目。

「臨海分局招募預備調查員、特邀外聘人員。」

蘇晨擠進人群瞄了一眼。

條件優渥：登記後月薪五萬起步，提供基礎修煉資源，購買各種靈能設施享八折優惠。

要求：一階以上覺醒者，無犯罪記錄。

「嘖，五萬塊啊……」旁邊的一個大叔感嘆道，「這年頭果然是覺醒者的天下。咱們這種老百姓，送死都沒資格。」

「可不是嘛，聽說昨晚東北區那道光，把好幾個高手都驚動了。天衡局這明顯是覺得人手不夠，想在民間招炮灰呢。」

蘇晨聽著周圍的議論，眼神微動。

五萬塊的月薪確實誘人，但他腦海中浮現出老趙昨晚的話——「登記的好處是有組織保護，壞處是你的一切都在別人的監控之下。」

他擁有萬界交易系統，這件事絕對不能暴露。

一旦進了天衡局，那些高階覺醒者的眼光毒得很，他這種「火箭式」的提升速度根本藏不住。

「還是先穩一手。」蘇晨搖搖頭，轉身走進了「趙記雜貨店」。

老趙還是在那裡喝茶，姿態和昨晚一模一樣。但這次蘇晨進門時，老趙的茶杯在半空中停頓了零點五秒。

他在看蘇晨。

眼神中透著一股見了鬼的驚愕，但很快就轉化為了深深的狐疑。

「小子，你昨晚……吃雞血了？」老趙放下茶杯，湊過來仔細盯著蘇晨的臉，「不對啊，你這靈氣濃度……怎麼比昨天厚實了一倍多？」

「可能是我厚積薄發？」蘇晨隨口扯淡。

「發你個頭！D級資質厚積薄發出這效果，那S級的還不得直接飛升？」老趙圍著蘇晨轉了兩圈，嘖嘖稱奇，「怪哉，真是怪哉。你該不會是偷吃了什麼禁藥吧？我警告你，那玩意兒副作用大得很，會燒壞腦子的。」

蘇晨笑了笑：「趙叔，你這兒有沒有更強一點的攻擊手段？總不能每次都跟人對拳頭吧。」

「攻擊手段？你還沒學會走就想跑？」老趙翻了個白眼，「你先把那點靈氣理順了再說。不過——」

他指了指對街的一個報刊亭，語氣變得有些陰沉。

「想學本事，先學會怎麼在這個世道活下去。」

蘇晨順著他的手指看去。

報刊亭的報紙頭條赫然寫著：【本市東北郊區發生多起人員失蹤案，警方提醒市民夜間減少外出】。

「失蹤案件？」

「沒那麼簡單。」老趙壓低聲音，「天衡局的人昨晚在那裡發現了殘留的妖氣。這說明裂縫不只是漏了點靈氣進來，還鑽進來一些不屬於這世界的壞胚子。」

蘇晨心中一凜。

這不就是昨天他在系統提示裡看到的那個「異常」嗎？

「小子，別怪我沒提醒你。」老趙重新躺在藤椅上，閉目養神，「現在這臨海市就是個裝滿炸藥的桶，東北區就是引信。你這點微末道行，千萬別去湊熱鬧。老老實實送你的快遞，多活幾年才是真理。」

蘇晨點了點頭，沒有說話。

但他心裡卻冒出了一個大膽的想法。

天衡局招募外聘人員，不一定要去分局坐班。如果是「臨時受委託」，是不是既能觀察這個世界的暗面，又能換取一些系統暫時提供不了的「現代覺醒者資料」？

就在這時，他的手機響了。

屏幕上閃爍著一個陌生的號碼。

蘇晨接起電話。

「喂，蘇晨嗎？我是天衡局臨海分局的林若溪。」

一個清冷好聽，卻帶著幾分公事公辦語氣的女聲從電話那頭傳來。

「我們查到你昨晚在蛇巷擊退了五名混混，身手不錯。現在我們這裡有個關於東北郊區的安全協助任務，缺個熟悉地形的當地人輔助，你有興趣聊聊嗎？」

蘇晨握著手機，回頭看了一眼正呼呼大睡的老趙。

趙叔啊趙叔，不是我想湊熱鬧。

是熱鬧自己找上門來了。

掛斷電話的瞬間，蘇晨感覺手機微微發熱。一條顯示為「未知來源」的加密信息自動在屏幕中央跳了出來。

附件是一張模糊的照片，背景是凌晨時分那片被封鎖的東北郊區。在漫天逸散的冰藍色流光中，隱約可以看見一尊晶瑩剔透的冰繭，而冰繭核心，似乎坐著一個長髮如銀、面容清冷至極的女子。

蘇晨的瞳孔縮成了針尖大小。

他死死盯著那個冰繭中的身影，雖然隔著屏幕，他卻感覺周身的靈氣都彷彿因為這張照片而流動得滯緩了一瞬。

這熱鬧，似乎比老趙說的還要大得多……

蘇晨深吸一口氣，收起手機，目光望向遠方漸深的天色，眼底閃過一抹決然。
