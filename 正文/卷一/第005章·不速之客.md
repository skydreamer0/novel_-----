# 第五章 · 不速之客

臨海市的夏日午後，空氣稠密得像是融化的柏油，帶著一股難言的焦灼感。柏油路面被烈日烤得微微變形，升騰起的熱浪讓遠處的景物都顯得有些扭曲。

蘇晨騎著他那台快散架的二手電動車，穿梭在縱橫交錯、彷彿永無止境的街道間。雖然昨晚剛學會了《長青吐納術》，體內的靈氣正像一條清涼的小溪緩緩流淌，但在這座鋼鐵與水泥鑄就的森林裡，即便貴為「修仙者」，也得為了下個月的房租和日常的三餐奔波勞碌。

不過，現在的他送快遞的效率高得驚人。五層樓的老舊樓梯，他以往爬上去總要喘口氣，現在卻能一口氣跨過三階，腳步輕盈得像是一隻在林間跳躍的羚羊，臉不紅氣不喘，連一滴汗都沒冒出來。路人只覺得一陣風颳過，回頭看去，只留下一個鮮紅色的快遞背影。

「叮。」

支架上那台螢幕碎裂的手機發出一聲清脆的提示音。蘇晨掃了一眼，導航顯示下一單的目標是——剛開發不久的「東港金融大廈」。

那是臨海市的新地標，一座通體由藍灰色鋼化玻璃幕牆包裹的摩天大樓，在烈日下反射著刺眼的光芒。它是所謂「社會菁英」的聚集地，進出其中的無不是西裝革履、步履匆匆的都市白領。

就在蘇晨等紅綠燈的時候，一台通體粉白的瑪莎拉蒂靜靜地停在了他旁邊。這台車的存在本身就像是一件昂貴的藝術品，流線型的車身在陽光下泛著一種如珍珠般的柔和光澤，與周圍那些灰頭土臉的代步車格格不入。

車窗緩緩降下，露出一張足以讓任何流量女星都感到自慚形穢的側臉。那女子戴著一副遮住半張臉的深色墨鏡，細碎的髮絲被微風吹亂，貼在她那如削出的鼻樑上。她的嘴唇薄而精緻，透著一股拒人於千里之外的清冷與孤傲。

最吸引蘇晨注意的，並非她的驚世美貌，而是她身上隱約散發的那股氣息。

冷。

那不是空調吹出的那種物理降溫，而是一種深入骨髓、彷彿能凍結時間流動的寒意。

蘇晨的「感知」本能地運作起來，在他眼裡，這台白色轎車周圍的靈氣竟然呈現出一種極其罕見的冰藍色結晶狀，細小的冰晶隨著她的呼吸在空氣中生滅。

「二階……不，這股壓迫感，至少是二階巔峰，甚至是快要突破三階的大手筆。」蘇晨心中暗暗吃驚，雙手不自覺地握緊了電動車的手把。

女子似乎察覺到了什麼不尋常的窺視，墨鏡後的目光微微轉動，隔著防爆隔熱紙，與蘇晨對視了一眼。

那一瞬間，蘇晨感覺自己的靈魂像是被一根極細的冰針刺穿，腦海中原本平靜的識海泛起陣陣漣漪，系統的光幕更是瘋狂地跳動起來。

```
  【緊急警告：檢測到高等級覺醒者精神掃描】
  自動防禦機制啟動...
  執行「大隱遁」模擬程序...
  當前偽裝狀態：D級雜系覺醒者（已鎖定）
  靈氣波動特徵：駁雜、微量、無威脅
```

坐在駕駛座上的林若溪微微皺起黛眉。

身為天衡局空降到臨海市的「天才調查員」，她肩負著追蹤昨晚那道沖天藍光來源的重任。剛才那一瞬，她確實感覺到旁邊那個看起來落魄不堪、甚至連鞋子都磨穿了底的快遞員身上，似乎藏著一絲極其純淨、甚至帶著某種古老荒涼感的氣息。

那種感覺，絕對不屬於這個已被污染的現代世界。

但當她下意識地釋放精神力進行二次探查時，對方的靈氣波動卻恢復了那種再普通不過的D級水平，就像是一潭死水，混濁、微弱，且毫無特色。

「是昨晚沒睡好導致的錯覺嗎？」林若溪低聲自語，收回了視線。

信號燈轉綠，瑪莎拉蒂發出一聲如猛獸復甦般低沉的轟鳴，如同一道銀色的閃電，瞬間消失在視線盡頭。

蘇晨這才長長地舒了一口氣，後背竟然在不知不覺間滲出了一層冷汗，涼颼颼的。

「這女人……好恐怖的洞察力。如果不是系統的自動隱匿，剛才那一下恐怕就得露餡。」

……

半小時後，東港金融大廈地下二層，一處偏僻的地下停車場。

這裡的燈光似乎有些年久失修，幾盞日光燈正發出惹人厭煩的滋滋聲，明明滅滅。空氣中瀰漫著一股淡淡的汽油味和長期不見太陽的霉味。

蘇晨把一份緊急公文包塞進客戶指定的智能儲物櫃，正準備跨上電動車離開，突然，他的耳朵動了動，捕捉到了一絲不和諧的噪音。

「咔嚓……咔嚓……」

那聲音像是利齒在啃食乾燥的骨頭，又像是利爪在水泥地上瘋狂抓撓。在空曠、甚至能聽見自己呼吸聲的地下室裡，這聲音顯得格外刺耳，令人毛骨悚然。

蘇晨的身體瞬間緊繃起來，右手按在了車把手上，目光如電般掃向四周。

在那排巨大的、漆著編號的水泥承重柱後方，一團濃郁得不正常的陰影正在瘋狂蠕動，彷彿有什麼黏糊糊的怪物正試圖從另一個維度擠進來。

「嗷嗚——！」

一聲沙啞而沉悶的低吼響徹停車場。

一道黑影猛地從陰影中竄出，快得只剩下一道模糊的殘線，目標直取蘇晨的喉嚨！

蘇晨憑藉著近段時間突飛猛進的反應力，本能地向右側身一閃。那黑影重重地撞在蘇晨身後的水泥柱上，發出一聲悶響，幾塊牆皮直接被撞得粉碎。

直到那東西落地，蘇晨才真正看清了這怪物的模樣。

那是一隻長約一米多、渾身上下長滿了鋼刺般褐色硬毛的「流浪狗」。但普通狗絕不會長著三排如鋸齒般的利牙，更不會在那張扭曲的臉上，嵌著一雙閃爍著幽綠色凶光、密密麻麻的複眼。

那些複眼正不停地旋轉著，死死鎖定了眼前的獵物。

「低階妖獸……難道是某個不穩定的次元裂縫鑽出來的？」

蘇晨感覺到自己的心臟在胸腔裡狂跳，呼吸變得急促起來。這還是他獲得傳送能力後，第一次真正面對來自異世界的危險怪物。那股從怪物身上散發出的、純粹的殺戮與毀滅氣息，讓他指尖微顫。

妖犬抖了抖身上的灰塵，喉嚨深處發出威脅的咕嚕聲，複眼中流露出嗜血的瘋狂。它四肢發力，指甲在水泥地上抓出幾道深深的痕跡，再次化作一道帶動勁風的殘影撲了上來。

「系統，別光看著，開啟戰鬥分析模式！」蘇晨在腦海中怒吼一聲，試圖壓制住內心的恐懼。

```
  【實時戰鬥分析啟動中...】
  識別目標：低階幽冥犬（幼生期）
  威脅級別：一階中階（對當前宿主而言：極高）
  偵測到弱點：小腹處皮毛較薄、右側第三隻複眼為神經中樞。
  系統建議：利用地形障礙進行規避，尋找一擊必殺的機會。
```

隨著系統提示的出現，蘇晨眼中的世界瞬間發生了奇妙的變化。

幽冥犬那原本快不可及的撲擊軌跡，在他眼中似乎被分解成了一幀幀連續的畫面。這是《長青吐納術》帶來的感官強化，讓他的大腦在極端環境下進入了超頻狀態。

「這一次，我不躲了！」

蘇晨雙眼微瞇，在幽冥犬那腥臭的大口即將觸碰到自己脖頸的瞬間，他沒有後退，反而猛地向前跨出了一大步。

那是福利院門口老趙平時教他的土法門——「進步即入身，退步必丟命」。

他精準地低頭避開了犬首的撕咬，能感受到那股灼熱的腐臭味擦著耳朵而過。同時，他全身的靈氣在此刻瘋狂地向右手匯聚，拳心彷彿包裹著一層淡淡的、肉眼難見的青芒。

「給我去死！」

一記勢大力沉的直拳，重重地轟在了幽冥犬柔軟的小腹上！

「嘭——！」

一聲如悶雷般的打擊聲在停車場內迴盪。

幽冥犬發出一聲刺耳的哀鳴，幾百斤重的身體竟然被這一拳的力量直接掀飛，半空中翻滾了幾圈，最後重重地砸在一輛黑色賓士車的引擎蓋上。

引擎蓋瞬間下陷，擋風玻璃碎成了晶瑩的粉末，防盜警報聲頓時響徹了整個地下空間。

蘇晨感覺自己的拳頭一陣酸麻生疼，那怪物的皮毛與肌肉結構遠超地球生物，簡直像是在搥打一塊實心的老樹皮。

然而，受創的幽冥犬並沒有就此倒下，劇痛反而激發了它體內的凶性。它發瘋似地甩動著腦袋，後肢在碎裂的引擎蓋上猛地一蹬，凌空旋轉起那乾枯卻充滿力量的身體。

那條長滿倒鉤、閃著烏光的尾巴猶如一條帶刺的鋼鞭，帶著尖銳的破空聲，狠狠抽向蘇晨的臉頰。

「刺啦——！」

蘇晨雖然盡力後仰，但還是慢了一瞬。

外賣制服的肩膀處被瞬間撕開了一道巨大的豁口，鮮紅的血液頓時滲了出來，火辣辣的疼痛直鑽心窩。

「馬德，給你臉了是吧！」

蘇晨心底隱藏多年的那股野性與狠勁也被徹底激了起來。在福利院長大的孩子，沒一個是好惹的，打架對他來說曾是家常便飯。

趁著幽冥犬落地後重心不穩、四肢打滑的空檔，他猛地抄起旁邊垃圾桶那沉重的合金蓋子。

蘇晨將全身剩餘的靈氣盡數灌注於雙臂，肌肉在此刻高高隆起，他像扔擲鐵餅一樣，將合金蓋子帶著嗚嗚的旋轉聲狠狠甩了過去。

「噹——！」

一聲金鐵交鳴。

飛旋的合金蓋精準地切中了幽冥犬右邊的主複眼。

綠色的腥臭汁液四處飛濺，幽冥犬發出了一聲幾乎要刺穿耳膜的慘叫，瘋狂地在地上打滾，那是最後的垂死掙扎。

「就是現在，結束這一切！」

蘇晨腳尖猛地點地，整個人猶如一支離弦的箭矢衝了上去。他一把抓住了幽冥犬那粗壯的後腿，全身肌肉緊繃到極限，像是古羅馬競技場上的大力士一般，將怪物那沉重的軀體猛地掄起，朝著堅硬的水泥柱狠狠地撞了過去！

「嘭！」「嘭！」「嘭！」

連續三聲沉悶的撞擊，幽冥犬的腦袋已經在劇烈的打擊下化作了一灘爛泥，徹底失去了生息。

蘇晨鬆開手，無力地癱坐在地板上。

他看著滿地狼藉，以及自己身上那些混合著綠液與鮮血的污跡，大口大口地呼吸著冰涼的空氣。

「呼……原來，這就是戰鬥。」

他能感覺到體內的靈氣已經枯竭，一種前所未有的虛脫感席捲全身，連手指頭都不想動彈一下。

就在這寂靜無聲、只有警報器還在徒勞鳴叫的時刻，一陣清脆、節奏穩定的高跟鞋聲從電梯轉角處悠悠傳來。

「嗒、嗒、嗒……」

這聲音在空曠的地下室裡迴盪，帶著一種掌控全局的壓迫感。

蘇晨瞳孔微縮，猛地回頭望去。

電梯口的方向，在那幽暗的燈光下，一襲白裙的林若溪正靜靜地在那裡。她的墨鏡已經摘下，露出一雙清冽如秋水的眸子。

她的右手之中，一柄由靈氣凝聚而成的、長約三尺的晶瑩冰劍正散發著絲絲白霧。

「是你？」

林若溪的聲音依舊是不帶溫度的清冷，但語氣中搜尋已久的疲憊感在此刻被一種掩飾不住的驚訝所取代。

她本是追蹤這股暴戾的妖氣而來，預想中會看到一幕血腥的殺戮現場。可她萬萬沒想到，看到的竟然是一個剛才還在街邊等紅綠燈、被她判定為「平庸」的D級快遞員，正以一種極其原始而狂暴的方式，親手錘殺了一頭一階妖獸。

這畫面所帶來的視覺衝擊力，讓這位見慣了大場面的天衡局天才也愣在了原地。

蘇晨看著眼前的「冰山女神」，腦子裡只剩下一個念頭：

完了，這下真的捂不住了。

他下意識地想要伸手遮擋住肩頭那凹陷的傷口，但奇蹟就在此刻發生了。

在林若溪那驚愕的注視下，蘇晨肩膀上原本深可見骨、還在流血的傷口，突然像是有了生命一般，在某種神祕力量的牽引下迅速癒合。不過短短三五秒的時間，新生的皮膚完整如初，甚至比周圍的皮膚還要細白幾分。

這根本不是D級覺醒者該有的自癒能力。

而更讓蘇晨感到不妙的是，在幽冥犬逐漸化作一縷黑霧回歸虛空的屍體下方，一顆指甲蓋大小、散發著幽暗紅光的菱形晶體正靜靜地躺在水泥地上。

「妖核……」

林若溪輕啟紅唇，聲音細不可聞。

她看著蘇晨，冰劍微微下垂，眼神中原本的審視與警惕，在那一刻轉化為了一種深不見底的、近乎執著的好奇與探索欲。

這局長親自交代的東區任務，似乎越來越有趣了。
