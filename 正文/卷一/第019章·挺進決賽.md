# 第十九章 · 挺進決賽

臨海市中心演武場，在東北區裂縫動亂之後的第三天，依然如期拉開了帷幕。

儘管這座城市還沒能完全從那場足以毀天滅地的震盪與火雨中恢復過來，街道上隨處可見正在清理廢墟的重型機械，但由天衡局總部親自下達的封口令與開幕令，卻如同不可撼動的鋼鐵意志，強行推動著「潛龍賽」的運轉。

整座體育館被數道高強度的靈能護盾所籠罩，半空中漂浮著數以百計的偵查無人機。這不僅僅是一場旨在選拔全市最強青年俊傑的選秀賽事，更是災後臨海市各大隱秘勢力在重新分配權力版圖、展示家族武力儲備與底蘊的血色戰場。

「聽說了嗎？在那晚東北區的廢墟核心，有個神祕的、穿著大紅快遞服的少年，僅憑一招就冰封了方圓三哩的烈焰，硬生生幫林隊長擋住了領主級青鸞的必殺一擊。」
「拉倒吧，你那是從哪兒聽來的地攤文學？我三表哥就在後勤部，那是總部派來的特種支援組幹的。快遞員？你咋不說他能送外賣砸死妖獸呢？」

演武場的休息區內，議論聲與猜疑聲此起彼伏，編織成一張密不透風的輿論網。

蘇晨此時正安靜地坐在一處不起眼的角落陰影裡，鼻樑上架著一副在雜貨店隨手買的寬大墨鏡，身上披著一件洗得發白的寬大休閒運動服。他右手手指上佩戴著的那枚「墨隱環」正穩定且安靜地散發著一圈圈微弱的冷光，將他體內那深邃如海的三階氣息精準地壓制在了再普通不過的一階巔峰水平。

在他身後，几个在那晚廢墟中被他順手救下命、卻被告知要「嚴守秘密」的新人調查員，此時正一臉崇拜加恭敬地站著，儼然已經在私底下把他當成了某種守護神靈。

「蘇教官……不，蘇大哥。這第一場海選，您打算幾招之內送那幫眼高於頂的豪門子弟下台？」那个留著寸頭、眼神犀利的男生壓低聲音，有些興奮地問道。

沒錯，蘇晨現在掛職的身分是：天衡局臨海分局特聘——備戰教練。
這是秦老在那晚之後親自提筆、連夜簽發的文件。雖然名義上他是教練，但根據「潛龍賽」那傳承百年的規矩：凡是二十五歲以下、且未踏入高階門檻的基層特勤人員，皆具備報名參賽權。顯然，這是秦老特意給他留的一扇大門。

「招數多少並不重要。」蘇晨輕輕用指尖推了推墨鏡，語氣淡然得像是在討論早餐吃什麼，「重要的是，動手的時候悠著點，別弄壞了這件運動服，這可是我在網上花了一百二十塊買的正牌貨，挺貴的。」

「哼，還是一如既往的喜歡裝模作樣。」

一道極度不和諧且充滿了粘稠惡意的聲音從側面橫插進來。

周家明坐在不遠處那裝飾奢華、被保鏢環繞的貴賓席上，此時的他，全身上下散發著一股令人感到脊背發涼、極其不安的紫色紊亂氣息。雖然他在東北區丟盡了臉面，但這短短兩天裡，他似乎透過某種極端途徑得到了周家禁庫中藥物的加持，修為竟然也詭異地跨進了三階，只是那種氣息顯得異常狂暴且失控，像是一枚隨時會引爆的人肉炸彈。

「蘇晨，我知道那晚你在廢墟裡耍了點不入流的旁門左道。但在這絕對公平、被萬人注視的擂台上，可沒有複雜的地形可以給你利用，更沒有林若溪能隨時衝出來幫你擋刀。」

周家明緩緩伸出一根手指，在自己的脖子下方做了一個優雅卻殘酷的切割動作。他獰笑道：「我在決賽等著你。到時候，在全臨海的高層面前，我會親手把你這隻跳梁小丑一塊塊撕碎。」

蘇晨連頭都沒抬一下，甚至連墨鏡都沒摘，只是在識海中輕輕呼喚。

```
  【解析偵查模塊·深度掃描啟動】
  ─────────────
  目標鎖定：周家明。
  生理狀態：呈現嚴重的[源質深度侵蝕]跡象，細胞活性呈透支性爆發。
  修為判定：三階初階（偽·藥物催化產物）。
  戰鬥缺陷：靈氣流動路徑混亂。爆發力雖強，但心肺負荷已達極限，持久力極差。
  ─────────────
```

「源質侵蝕……這帮瘋子難道把那種紫色藥劑當補藥喝了嗎？看來周家真的跟『蛻變者』走得太近，已經沒救了。」蘇晨在心底不入痕跡地冷笑了一聲。

「第一輪，第一場對陣：天衡局特別組蘇晨，對陣，城防部第一猛將趙猛！」

裁判那如雷鳴般的聲音透過擴音陣法響徹全場。

蘇晨緩緩起身，动作有些懶洋洋地揉了揉惺忪的睡眼，在萬眾注目之下走上了一號擂台。

對面的趙猛是一名三十歲、正處於體力巔峰的二階頂級「體修」。他赤裸著的上半身肌肉隆起如花崗岩，周身閃爍著沉穩的古銅色光芒，活像一尊活生生的金屬鐵塔。他看著蘇晨那略顯瘦弱的身影，鼻孔裡噴出一股濁氣，冷哼道：
「臭小子，聽說你是走秦局長的後門才掛了個教練頭銜的？今天，大爺我就讓現場所有人看清楚，什麼叫絕對的實力差距！跪下吧！」

「一。」

蘇晨站在原地，甚至連腳步都沒挪動一分，只是對著那尊「鐵塔」伸出了一根修長的手指。

「什麼一？你特麼被嚇傻了吧？！」
趙猛怒吼一聲，雙拳瞬間灌注了雄渾無比的土黃色靈光，猶如兩柄萬斤重錘般帶起刺耳的風壓，瘋狂地對著蘇晨的天靈蓋砸下。

「唰——！」

就在重錘即將觸及蘇晨髮絲的剎那，他的身形猛然一矮，就像是一抹徹底融入光影中的黑影，以一個不可思議的角度穿過了趙猛那看似嚴密的重錘縫隙。

蘇晨沒有出劍，甚至沒有動用哪怕一丁點的三階肉身原始力量。他只是在錯身而過的瞬間，精準無比、輕飄飄地遞出了一記側掌，切在了趙猛腰間氣門最為脆弱的那一各個靈壓節點上。

「嘭——！！」

一聲悶響過後，剛才還威風凜凜的趙猛，那巨大的身軀竟然像是一個被瞬間扎破放氣的皮球。他喉嚨裡出一聲淒慘的悶哼，整個人完全失控地倒飛出擂台邊緣，重重地撞在五十米外那厚實的水泥牆壁裡，整個人深埋其中，當場昏死了過去。

全場，陷入了死一般的寂靜。

一招。甚至是僅僅一次錯身。
除了二樓看台上的幾位大佬，在場的普通觀眾甚至都沒能看清蘇晨具體是怎麼出手的，只看到「鐵塔」突然就飛了。

「這……這就是所謂的特聘教練的身手？這也太離譜了吧！」
「太快了！那趙猛可是能徒手拆坦克的猛漢啊，連這小子的衣角都沒碰到就被秒了？」

演武場二樓那裝飾優雅的 VIP 看台上，林若溪正慵懶地靠在精緻的紅木欄杆上，一雙修長的美腿交疊在一起。她看著下方那個收手入兜、走得雲淡風輕且全程沒露半點氣息的少年，清冷的嘴角微微向上揚起一道不易察覺的弧度。

「這傢伙……裝模作樣的本事倒是越來越登峰造極了。」

接下來的整整三個小時裡，蘇晨在比賽中完美詮釋了什麼叫做「讓對手絕望的極致」。

第二輪：vs 周家護衛部精英。蘇晨僅僅用一根手指點在對方如龍般刺來的劍脊上，指力爆發間，百鍛精鋼劍應聲碎裂成無數殘片，人更是倒飛而出。消耗時間：3秒。
第三輪：vs 凌海大學校隊最強天才。蘇晨單手插兜，一掌拍散了對方引以為傲的漫天冰爆，隨即僅用肩膀來了記貼山靠。耗時：5秒。
準決賽：vs 一名疑似蛻變者潛伏進來的刺客。對方的袖箭淬毒，蘇晨眼神中第一次閃過一抹如刀般的冷冽，僅僅憑藉著一式「瞬殺」的起手殘影，就直接震碎了對方的奇經八脈。耗時：2秒。

他就像是一台精密的靈能收割機，無論面對什麼流派、什麼級別的二階對手，通通都是懶洋洋的一招解決，甚至連額頭上都沒冒出一滴汗珠。

夕陽西下，殘陽如血。

當演武場中央那巨大的全息螢幕上，名字跳動到最後那一格爭霸賽位時，整座體育場爆發出了如海嘯般震天動地的歡呼聲。

「巔峰之戰：天衡局教練·蘇晨，vs 臨海周家傳人·周家明！」

蘇晨站在被晚霞染紅的擂台正中央，他冷漠地看著對面那雙目赤紅如血、渾身血管暴突且明顯已經快要因為藥力副作用而失去理智的周家明，緩緩抬起手。

他看似隨意地摸了摸左手那枚漆黑的戒指，指間的一抹流光稍微鬆動了一絲縫隙，一小部分三階大修的威壓，在這一刻悄然洩露。

「周公子，你之前在賭局裡欠我的那塊所謂的『上古古碑殘片』，現在準備好親手交給我了嗎？」
