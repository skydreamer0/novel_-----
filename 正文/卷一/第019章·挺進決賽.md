# 第019章 · 挺進決賽

臨海市，潛龍大賽的比習區域，此刻已經成了全城靈壓最為狂暴的核心點。

原本整潔、現代化的訓練場地，在這一刻卻瀰漫著一股若有若無的藥劑甜味。這是不正常的，蘇晨站在備賽通道的陰影中，他的指尖正無意識地劃過腰間的那個不起眼的快遞包。在他的系統全景視野中，整個賽場地下的管網結構，正有規律地流動著一種深紫色的脈衝光，那是他在醫院太平間負三層見過的——源質汙染。

「果然，這所謂的選拔賽，從一開始就是周家為了篩選最合適的『進化體』而設下的陷阱。」
蘇晨冷笑一聲，他輕輕抿了一口剛開啟的可口可樂，碳酸氣泡在舌尖炸裂，帶給他短暫的舒爽感。

「蘇晨！到你了！你到底在墨跡什麼？」
外面傳來裁判有些不耐煩的催促聲。在他們眼裡，這個憑藉著「關係」混進淘汰賽的D級快遞員，不過是走個過場的炮灰。

然而隨招蘇晨一步跨入擂台。

那一瞬間。原本顯得有些嘈雜、甚至是帶著幾分輕蔑笑聲的觀眾席，竟然在一秒鐘內陷入了某種詭異的死寂。

蘇晨並沒有釋放出那種排山倒海般的靈壓。恰恰相反，他將自己那剛突破到三階巔峰的狂暴劍意，以一種現代快遞員特有的精確打包方式。死死地壓縮在了自己的每一寸毛細血管之中。在外人看來。他此時就像是一尊毫無修為波動、卻在烈日下卻顯得無比深邃的黑色深潭。

「嘿，快看那小子，他是不是被嚇傻了？連靈能外衣都撐不開了嗎？」
對面站著的，是來自省城周家的遠親子弟、實力已達三階初期的火系覺醒者周浩雲。

周浩雲此時滿臉傲橫，眼神中透著一股赤裸裸的殺意。他手中的那柄鑲嵌了三枚火晶石的長劍。在高溫下正劇烈顫動：
「姓蘇的，如果你現在跪下來，當眾承認你那所謂的『導航技術』是從我們周家偷來的，我也許能大發慈悲，只廢了你的雙手。否則……」

「否則？」蘇晨歪了歪頭。他的嘴角勾起一抹危險的弧度，「別廢話了。我的快遞配送時間，可是精切到秒的。」

「找死！！」

周浩雲發出一聲暴喝，整個人身形一閃，化作一道熾熱的火龍殘影。一招「烈焰灼心」，帶著足以將鋼筋熔化的數千度高溫，直取蘇晨的眉心！

在那一刻，周圍的學員們紛紛發出了驚嘆聲。在他們看來，三階初期的周浩雲，在年輕一代中絕對算得上是佼佼者，這霸道的一劍，那D級快遞員拿什麼擋？

然而。

在蘇晨那被系統強化的視野中，周浩雲這看似迅猛絕倫的一擊。簡直比一隻正在逆風爬行的蝸牛還要破綻百出。

「第一式：瞬殺。精確座標修正——左移三公分。切斷其火脈流向。」

蘇晨連眼皮都沒眨一下。
腳尖在特製的合金台面上輕輕一旋，一個輕盈到近乎優雅的側身。原本看似必中的火龍。就那樣險之又險、卻又毫釐不差地擦著他的紅色運動裝袖子。精準地落在了空處。

「什麼？！」周浩雲臉色劇變，他感覺自己這全力一劍。竟然像是砍在了一團毫無著力點的虛空雲霧之上。那種力道落空的巨大慣性。讓他胸口一陣煩悶欲吐。

也就是在那剎那。

蘇晨動了。那隻修長的、平時用來簽字寫單據的手。竟然以一種不可思議的頻率。悄無聲息地貼在了周浩雲那覆蓋著厚厚火盾的後心之上。

「嘭！」

一聲極其沈悶、如同布帛被撕裂的聲音在擂台之上輕輕一響。

原本看似強悍無比的周浩雲。整個人竟然在一聲不可思議的慘叫中。像是被一輛時速兩百公里的重型卡車正面撞中般。直接橫飛出了五十米開外，重重撞碎了場邊那號稱能抵禦四階衝擊的鋼化觀賽牆，昏死過去時，那身上原本璀璨的火甲。竟然已經徹底崩解析。

「蘇晨……勝！晉級八強決賽！」
裁判目瞪口呆，聲音甚至帶著幾分明顯的顫音。

整場選拔賽就在這種一種極度壓抑、卻又無比震撼的節奏中。被蘇晨以一種近乎「暴力送貨」的方式快速推向前進。

無論是那些家世顯赫的各路豪門子弟。還是那些自負天才的武道覺醒者。在碰到這穿著紅衣服的少年的時候。無一例外。全都被蘇晨用看似最樸實無華、實則精準到了魔幻程度的普通一擊。給毫不留情地清出了戰場。

……

決賽前夕，臨海市的夜。安靜得有些讓人感到背脊發亮。

「這枚『神廟殘片』，震動得越來越厲害了。」
蘇晨獨自坐在雜貨店的露台上。他看著手心中那块原本平平無奇、此刻卻隱約透著一絲邪惡紫光的殘片。這件東西，是在他擊破最後一名周家核心子弟時「順手牽羊」繳獲的。

```
  【系統檢測：偵測到跨維度座標糾纏，當前穩定度下降。】
  注意：這不是單純的修行物，這是一個定位導航器。
  它正在……呼喚外面的那些怪物。
```

「果然，這所谓的決賽擂台底下。根本不是靈脈，而是一座正在甦醒的血祭大陣。」
蘇晨冷冷地盯著燈火輝煌的賽場方向。這座城市，這場大賽，所有人都以為是在通往榮耀。唯有他知道。那是在通往地獄的嘴邊。

「想拿這座城的命給你換路？那我就把這條路。給你們拆個稀巴爛。」

蘇晨隨手捏碎了一個空的雪碧罐。
他的背影。在夕陽的餘韻下。顯得孤絕而冷峻。一場真正決定命運的高峰對決。正在那看不見的暗流中。全面引爆。
