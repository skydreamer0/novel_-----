# 第十九章 · 挺進決賽

東北區裂縫動亂之後的第三天。

儘管臨海市還未完全從那毀天滅地的震盪中恢復，但「潛龍賽」——這場旨在選拔全市最強青年俊傑的賽事，依然由天衡局總部下令，如期在中心演武場開幕。

這不僅僅是一場比賽，更是臨海市各大勢力在災後重新分配權力、展示武力的戰場。

「聽說了嗎？東北區戰場上出了一個神祕的快遞員，一招冰封了三哩地，幫林隊長擋住了青鸞的一擊。」
「拉倒吧，我聽說是總部派來的特種支援隊幹的。快遞員？送外賣砸死妖獸嗎？」

演武場休息區，議論聲此起彼伏。

蘇晨坐在一個角落裡，鼻樑上架著一副墨鏡，身上披著一件寬大的運動服。手中的戒指「隱氣環」正安靜地散發著冷光，將他的氣息精準地壓制在一階巔峰。

在他身後，几个被他救過命的新人覺醒者正一臉恭敬地站著，儼然已經把他當成了偶像。

「蘇教官，這第一輪您打算幾招解決？」那个寸頭男生小聲問道。

沒錯，蘇晨現在的身分是：天衡局臨海分局特聘——備戰教練。

這是秦老親自簽的手令。雖然他是教練，但根據「潛龍賽」的規矩，二十五歲以下的基層特勤人員，皆有參賽權。

「招數不重要。」蘇晨推了推墨鏡，語氣淡然，「重要的是別弄壞我的衣服，這件挺貴的。」

「切，裝模作樣。」

一道不和諧的聲音傳來。

周家明坐在不遠處的貴賓席上，此時的他，全身上下散發著一股令人不安的紫色氣息。雖然他在東北區丟了面子，但這兩天似乎得到了周家禁藥的加持，修為竟然也跨進了三階，而且氣息異常狂暴。

「蘇晨，我知道你在東北區耍了點小手段。但在這公平的擂台上，沒有地形可以給你利用，也沒有林若溪能幫你擋刀。」

周家明伸出手指，在脖子下做了一個切割的手勢，「決賽見。到時候，我會親手把你撕碎。」

蘇晨連頭都沒抬。

```
  【解析偵查啟動】
  ─────────────
  目標：周家明。
  狀態：源質深度侵蝕。
  修為：三階初階（偽）。
  隱患：靈氣流動不暢，爆發力極強但持久力差。
  ─────────────
```

「源質侵蝕……看來周家真的跟蛻變者走得太近了。」蘇晨在心底冷笑。

「第一輪，第一場，天衡局蘇晨，對陣，城防部趙猛！」

裁判的聲音響徹全場。

蘇晨緩緩起身，懶洋洋地走上了一號擂台。

對面的趙猛是一名二階巔峰的體修，渾身肌肉隆起，活像一尊古銅色的鐵塔。他看著蘇晨那瘦弱的身影，冷哼一聲：「小子，聽說你是走局長後門進來的？今天我就讓你看清楚，什麼叫實力差距！」

「一。」

蘇晨站在原地，動都沒動，只是伸出一根手指。

「什麼一？你嚇傻了？」趙猛怒吼一聲，雙拳灌注土黃色的光芒，猶如兩柄重錘瘋狂砸下。

蘇晨身形一矮，就像是一抹黑影穿過了重錘的空隙。

他沒有出劍，甚至沒有動用三階的肉身力量，只是精準地一側掌，切在了趙猛氣門的節點上。

「嘭！」

趙猛那巨大的身軀像是被放了氣的皮球，整個人失控地倒飛出擂台，重重鑲嵌在了遠處的水泥牆裡，昏死過去。

全場，死一般的寂靜。

一招。

甚至沒人看清蘇晨是怎麼出手的。

「這……這就是特聘教練的身手？」
「太快了！那趙猛連他的衣角都沒碰到！」

二樓看台上，林若溪靠在欄杆上，看著台下那個收手入兜、走得雲淡風輕的少年，嘴角微微上揚。

「這傢伙，真是越來越會裝了。」

接下來的三個小時裡，蘇晨完美詮釋了什麼叫「絕望」。

第二輪：vs 周家護衛部高手。蘇晨一指點在對方劍脊上，劍斷人飛。耗時：3秒。
第三輪：vs 凌海大學最強天才。蘇晨一掌拍散對方的冰球，反擊一記貼山靠。耗時：5秒。
準決賽：vs 蛻變者潛伏刺客（偽裝身分）。蘇晨眼神中閃過一抹冷冽，一式「瞬殺」的起手式，直接震碎了對方的經脈。耗時：2秒。

他就像是一台精密的收割機，無論面對什麼樣的對手，通通都是一招解決。

夕陽西下。

當大螢幕上的名字跳動到最後一格時，整座演武場爆發出了海嘯般的歡呼聲。

「決賽：天衡局·蘇晨，vs 臨海周家·周家明！」

蘇晨站在擂台中央，看著對面雙目赤紅、已經快要失去理智的周家明，緩緩抬起手。

他摸了摸戒指，指間的一抹流光稍微鬆動了一絲。

「周公子，欠我的那塊古碑殘片，準備好交出來了嗎？」
