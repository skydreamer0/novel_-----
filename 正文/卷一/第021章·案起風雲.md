# 第二十一章 · 案起風雲

「全城戒嚴，天衡局接管所有防務！」

秦老的聲音還在臨海演武場上空震盪，整座城市便已在瞬間切換到了戰爭模式。

原本慶祝比賽結束的煙花被無數紅色的警戒燈代替，天際之上，十幾架天衡局的靈能戰機呼嘯而過，投射出巨大的禁錮光網，將周家大宅所在的一整片山頭封鎖得水洩不通。

大雨，傾盆而下。

蘇晨站在一片狼藉的擂台中心，抹了一把臉上的雨水。他的手揣在兜裡，死死握著那塊剛到手的青銅殘片。

那東西涼得出奇，卻在他的指尖引發了一陣又一陣的神祕震顫。

「蘇晨，沒受傷吧？」

林若溪踩著瓦礫飛奔而至。她的制服上沾滿了碎石屑，冰劍歸鞘，清冷的眸子中透著一抹連她自己都未察覺的緊張。

「還行，就是有點費體力。」蘇晨裝出一副虛弱的樣子，晃了晃手裡斷掉的樹枝，「隊長，局長說的證據確鑿……是真的？」

「你剛才毀掉周家明體內的源質核心時，數據已經被後台防禦陣法截獲，同步發送到省城總部了。」林若溪嘆了口氣，看向遠處被秦老一掌拍廢、正被兩名特勤隊員拖走的周天嘯，「周家這次，是澈底完了。但這只是個開始。」

「開始？」

「周家在臨海市經營幾百年，盤根錯節。他們倒了，原本由他們鎮守的幾個『空間裂縫節點』現在全部失控。更糟糕的是……」她壓低聲音，「根據內應的消息，蛻變者的『大主教』親自帶隊進入了臨海市，目標就是你剛才拿到的那塊神廟殘片。」

「大主教……」蘇晨心中微動，腦海中浮現出那個地下實驗室裡紫色肉瘤的扭曲影像。

```
  【系統冷卻完畢】
  ─────────────
  目標定位：神廟殘片（一階權限）
  功能：開啟通往『萬界神廟』的鑰匙殘次品。
  數據回饋：持有此物超過 24 小時，將會引發附近 100 公里內所有源質生物的共鳴追蹤。
  ─────────────
```

蘇晨心中一沉。這殘片簡直就是一個超功率的定向GPS。

「走，先回總部避難。」林若溪拉起蘇晨的胳膊，語氣強硬。

然而，就在兩人準備上越野車的瞬間。

「轟隆隆——！」

遠處市中心的標誌性建築——臨海電視塔，在漫天大雨中發出了一聲慘烈的哀鳴。塔尖那巨大的避雷針竟然被一隻從雲層中探出的巨大紫色利爪生生折斷。

緊接著，一股比剛才「焚天青鸞」還要狂暴數倍的紫色光柱衝天而起。

「那是……」林若溪的臉色煞白。

「有人從內部開啟了最後一道防禦陣法。」蘇晨接過話頭，他的眼神變得極其凌厲，「林隊，不對勁，那裡是天衡局的『物資儲備庫』。」

「該死！他們不是在撤退，是在搶奪！」林若溪猛地跳上車，瘋狂啟動引擎，「蘇晨，抓穩了！」

越野車在積水四濺的街道上狂飆。

蘇晨看著螢幕上的解析地圖，發現無數代表「蛻變者」的紫色紅點，正從臨海市的各個下水道、暗渠、老舊廢墟中噴湧而出，目標直指天衡局分局。

「內鬼。」

蘇晨腦中蹦出這兩個字。

能精準避開秦老的排查，在全城大掃蕩的關頭精準突襲物資庫，天衡局的高層裡絕對出了叛徒。

「林隊，停車。」蘇晨突然說道。

「你瘋了？現在外面全是蛻變者的瘋子！」

「他們想要這塊石頭，對吧？」蘇晨舉起青銅殘片，「如果我跟你回總部，那就是在把所有的火引向秦老身後。我們得換條路。」

「換哪兒？」

「舊城區，蛇巷。」

蘇晨看著遠方那片淹沒在黑暗與暴雨中的老弄堂，「那兒有個老頭，欠了我一瓶龍井，該是讓他還債的時候了。」

越野車一個甩尾，調轉車頭，朝著那片被時代遺忘的蛇巷疾馳而去。

而在他們身後，三台改裝過的重型越野車，正無聲無息地從陰影中浮現，車頂的旋轉機槍，已經鎖定了蘇晨的車牌。

一場席捲整座城市的追逐與密謀，在雷電交錯的夜晚，拉開了最慘烈的序幕。
