# 第二十章 · 巔峰對決

決賽的鐘聲，沉悶地敲響。

臨海演武場的中央擂台，被一層特製的防禦光幕籠罩。數萬名觀眾屏息凝神，甚至連一些隱居多年的老怪也紛紛投來了神識。

周家明站在擂台的一端，他的黑色戰服下，血管正呈現出詭異的紫色，不斷跳動。那股原本屬於二階、三階的靈壓，在此刻竟然瘋狂膨脹，隱隱有突破四階的趨勢。

「蘇晨……你死定了……你一定會死……」

周家明的聲音聽起來像是兩塊鏽金屬在摩擦，那是神智被「源質藥劑」蠶食的徵兆。

「周公子，為了贏我，你連人都打算不做了嗎？」

蘇晨站在對面，語氣平靜如水。他背負著雙手，腳下的步伐隨意而散亂。但在秦老這種層次的高手眼裡，蘇晨此時的一呼一吸，都與這片天地的節奏合而為一。

「殺——！」

周家明發出一聲野獸般的嘶吼，整個人化作一道紫色的電光。他五指成爪，空氣中響起了一陣令人毛骨悚然的撕裂聲。

「周家絕技：裂空爪！」

這原本是三階頂級的武技，在源質的加持下，竟然帶上了一絲「破空」的屬性。

蘇晨身形不動，只是右手輕輕一拂。

「歸墟，瞬殺。」

這一次，他沒有出劍。

他以手代劍，指尖吞吐出一抹漆黑的劍意。

「鐺！！」

指尖與利爪相撞，竟然發出了金屬交擊的巨響。原本信心滿滿的周家明感覺自己像是撞上了一座萬年不化的寒冰。

「怎麼可能！」

「沒什麼不可能的。你的力量，本就不屬於你。」

蘇晨眼神一冷，體內三階鍛體的強橫肉身全面爆發。他肩膀一抖，一記樸實無華的「鐵山靠」直接撞在周家明的胸口。

「噗！」

周家明噴出一口混著紫色粘液的鮮血，整個人倒飛出數十米。

「家主！快開啟陷阱！」看台上的周家長老急了。

周天嘯臉色鐵青，指尖在遙控器上猛地按下。

「嗡——！！」

擂台下隱藏的紫色管道瞬間爆裂，一股濃稠的紫色霧氣迅速籠罩了整個光幕。那不只是毒素，更是能干擾靈氣運行、強行引發覺醒者內息紊亂的「源質誘餌」。

「蘇晨，在源質的海洋裡，我看你還怎麼施展劍意！」周家明在霧氣中瘋狂大笑，他的傷口在紫色霧氣的滋養下竟然迅速癒合，氣息再次暴漲。

擂台外的林若溪臉色大變，冰劍出鞘：「周家！你們竟敢公然投毒！」

「那是特殊演練環境！林隊長請坐下！」周家的私兵瞬間將林若溪圍住。

「秦老……」林若溪焦急地看向主位上的秦滄海。

秦老依舊端坐著，但那雙扶著藤椅的手，已經將精鋼鑄造的扶手捏成粉末。他死死盯著霧氣中的那個身影。

他在等。

等蘇晨那最後的底牌。

就在紫色霧氣即將吞噬蘇晨的一瞬間。

蘇晨長長地吸了一口氣。

他的識海中，系統發出了瘋狂的律動。

```
  【解析完成：目標源質頻率已鎖定】
  【劍修世界·葉孤楓劍感全開同步！】
```

「區區殘次品，也敢稱進化？」

蘇晨的手中，不知何時多了一截普通的樹枝。

那是他在上場前隨手從路邊折的。

但在這一刻，這截樹枝卻吞噬了周圍所有的紫色光芒。一股寂滅、荒涼、足以凍結時空的劍意，從蘇晨身上沖天而起，直接擊碎了擂台頂部的防禦陣法！

「第一式：瞬殺，寂滅三千！」

蘇晨身形一晃，化作了漫天黑羽。

沒有爆炸，沒有特效。

唯有劍氣劃開血肉的微弱聲響。

當紫色霧氣散去時，全世界都安靜了。

蘇晨站在擂台中央，手中的樹枝已經化為齏粉。

而周家明，全身上下每一處關鍵經脈，都被精準地切斷。他體內那狂暴的紫色源質像是找到了洩洪口，瘋狂地從傷口湧出，最終化作一灘腥臭的液體。

周家明，修為全廢。

而在他的面前，一塊散發著古老、玄奧氣息的青銅殘片，正靜靜地躺在瓦礫中。

那是周家的傳家寶，也是「蛻變者」夢寐以求的鑰匙——神廟殘片。

蘇晨彎腰，不緊不慢地將殘片揣進兜裡。

他抬頭看向看台上一臉呆滯的周天嘯，露出一抹燦爛的笑容。

「周家主，這彩頭，我收下了。」

「蘇晨！！我要你死！！」

周天嘯徹底瘋狂，他一身四階巔峰的修為轟然爆發，竟然不顧身分，直接從看台跳下，五指籠罩著黑光，朝著蘇晨的首級狠狠抓去。

「你動他試試？」

秦老的身影猶如瞬移，擋在了蘇晨面前。

老爺子單手負後，另一隻手只是輕輕一揮。

「轟——！！」

周天嘯像是一隻蒼蠅般被直接拍飛，撞碎了半座看台。

「周家勾結境外邪教，殘害同胞，證據確鑿。」

秦老的聲音在靈氣加持下，如同驚雷響徹臨海市，「全城戒嚴，天衡局接手臨海市防務！所有周家子弟，束手就擒！」

這一夜，臨海市的天，徹底變了。
