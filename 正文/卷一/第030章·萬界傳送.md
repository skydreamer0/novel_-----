# 第三十章 · 萬界傳送（卷一完）

臨海市那悶熱卻又帶著雨後清涼的盛夏之夜。在經歷了那一場足以毀滅周邊幾個行省文明的動盪與戰火之後。終於在那夕陽餘暉的收斂下。重新恢復了一種如深潭之水般祥和、寧靜且透著幾分滄桑感的夜色。

蘇晨獨自一人站在老趙雜貨店三樓那處有些漏風、且掛著幾件晾曬衣物的小破露台上。極目遠眺，遠處是重歸繁華、再度霓虹閃爍的摩登都市天際線。近處則是充滿了親切市井呼吸、瓦片層疊、偶爾傳來幾聲犬吠的老舊弄堂。這兩種截然不同、甚至如同兩個位面般衝突的氣息在此刻交織相融，卻莫名地讓此時的蘇晨感到了一種前所欲有的踏實、厚重，以及一種身為「守護者」的微弱自豪感。

「系統。傷疤也差不多快長好了，咱們該去賺那份跨位面的大錢了。確認傳送目標座標：仙俠高階位面——劍修星域·第一界·紫霄宮萬劍崖原址。」

```
  【最終星際傳送程序正式啟動中...】
  ─────────────
  當前起始方位：地球·大夏國·臨海分局管轄區特定錨點。
  目標位面能級評定：S+ 級（傳說中的高階仙道、多重靈氣重疊位面）。
  單次跳躍消耗結算：1000 塊標準極品中品靈石（已從亞空間儲物空間精準扣除，當前剩餘量：警戒級）。
  位面因果穩定性係數：99.999%（完美降臨狀態）。
  ─────────────
```

一道淡淡的、最初時幾乎肉眼不可見的透明時空漣漪。在此刻以蘇晨腳尖下踩著的那塊有些鬆動的紅磚為中心點。像是投入了寂靜湖面的沉重石子般。以一種優雅、神聖且極其有力地頻率向著四周的虛空擴散開來。周圍的空氣在此刻瞬間變得扭曲且粘稠了起來，隱約中，竟然能透過那扭曲的縫隙，看到裂縫另一端那正閃爍著萬千星辰、且有無數柄巨劍懸浮在空的宏偉仙山倒影。

蘇晨半側過身。最後深情地看了一眼安靜躺在身後那張舊木桌上、被一塊磚頭壓住的那一大疊新印好的、還散發著淡淡刺鼻油墨清香的「萬界快遞」專屬物流單。那每一張單子的封面上，都筆走龍蛇地印著八個足夠讓神靈震撼的燙金大字：【萬界快遞·縱橫諸天·使命必達】。

「呵……從這第一張正式跨越次元的單子派送開始。我這原本只想混日子的小小快遞生意，可就真的要做大做強。真正走向那諸天萬界的巔峰博弈了啊。」

他低頭輕笑了一聲，抬手緊了緊背後那個沉甸甸的、裝滿了各類「異界暢銷戰略物資」的黑色合金背囊。那是他接下來立足仙俠界的本錢。正當他深吸一口氣、準備毅然踏入那道正散發著神聖光芒、且伴有龍吟之聲的金色空間裂縫之中的那一秒。

「蘇晨——！！你這混蛋！給我等一下！」

一聲帶著明顯急促、且因為劇烈奔跑而顯得有些破碎、甚至還帶著一點點委屈鼻音的嬌呼。突然從他背後那陰暗潮濕的樓梯口處猛地傳來。

林若溪氣喘吁吁地衝上了露台。她那一向整潔神氣、代表了天衡局榮譽的白色軍用披風此時顯得有些凌亂地掛在肩膀上。臉上還掛著一絲分不清是雨水還是驚嚇過後的晶瑩汗珠。她那一雙平日裡清冷得如同萬年不化、生人勿進的冰川眼眸。此時正死死地盯著蘇晨身旁那正劇烈扭曲、甚至發出嘶鳴聲的空間旋渦。那其中一閃而過的複雜驚愕、掙扎與深深的擔憂痛苦。是這輩子她都不曾對任何外人、哪怕是家族長輩展露過的。

「你……你這傢伙……這就要走了嗎？不等傷口徹底好全了嗎？那個連聽都沒聽說過的『劍修界』……真的有那麼重要嗎？」

「只是去出個小差，幫一位挺耿直的客戶送幾箱積壓已久的急件罷了，這就是我的職業操守啊。」
蘇晨嘴角依舊掛著那副讓她又愛又恨的、沒心沒肺般的輕鬆微笑。他瀟灑地側過頭，對著林若溪揮了揮那隻因為搬貨而磨出了一層薄繭的手：
「去看看那個天天隔空喊話問我要可樂、甚至想用仙劍換可樂的劍仙哥們。放心吧，我的林大局長，我現在的物流信譽度評分可是爆表的 100%。我保證，只要單子一簽收，我一定會第一時間平平安安地滾回這條蛇巷。」

林若溪死死地咬著蒼白的薄唇。在蘇晨那隻腳即將跨入光亮中心圈的那一瞬。她像是下了某種極大的決心，突然從自己纖細的脖頸上。一把摘下了那枚從小佩戴、代表了林家核心嫡系地位、甚至能調動一支私軍的、內部隱約有冰鳳鳴叫的玉質命牌。她大步邁出，閃身衝到蘇晨面前，在那空間乱流的風壓中。強行將那枚依舊帶著她體溫與少女芬芳的玉鎖，塞進了這根大木頭佈滿了汗漬的粗糙掌心裡。

「這是我林家最核心的護身命牌，亦是我的本命魂石。如果有朝一日。你在那個什麼諸天世界裡，真的遇到了連你那破系統、或者是你自己都解決不了的絕對死局。記得，毫不猶豫地捏碎它！無論你身在哪一個次元。無論路途有多遙遠。我……我林若溪也一定會帶領天衡局最精銳的部隊。想盡一切辦法去那個世界……把你這混蛋給救回來！聽清楚了嗎？！不准給我死在那邊！」

林若溪的聲音雖然在顫抖，字數也不多，卻透著一股破釜沉舟、甚至是打算與諸天神靈硬碰硬的絕然霸氣。

蘇晨低頭看著手心中那枚晶瑩剔透、還帶著女孩淡淡幽香與熾熱心跳溫度的玉墜。心中某一處堅硬的防線，像是被某種巨大的、名為牽掛的溫暖洪流給猛地撞開了。他終究是沒忍住，伸出了那隻原本打算握住劍柄、卻在此刻微微顫抖的手。有些笨拙、卻極其溫柔地摸了摸林若溪那如冰川冰霜般清涼柔滑、卻又在微風中顯得有些零亂的長髮。

「好啦，我都記住了。等我這次把這趟大買賣做成了，再給你帶點那邊世界的『土特產』回來。到時候，我一定請你去全臨海市最貴的、那個能看見海風的頂層景觀餐廳。請你吃這輩子最好吃的炸雞套餐。管飽的那種。」

說完。蘇晨在林若溪那既羞惱又失落的目光中。瀟灑地轉過身。
他背負著那跨越了不同文明與因果的重擔，身形在那一刻義無反顧地、甚至帶著幾分豪邁地沒入了那道璀璨奪目的、此時已然完全展開的金色空間大門之中。隨即與那漫天的星光殘影一同，在林若溪的視線盡頭。徹底消失得無影無蹤。

林若溪癡癡地、安靜地站在那空蕩蕩、此刻顯得格外清冷落寞的露台上。她仰起那張精緻的臉龐，看著那繁星點點、卻彷彿比平時要高遠了無數光年的夜空，良久之後，才發出了一聲悠長、且帶著無盡思念與一絲微酸感的嘆息。

「蘇晨你這個大木頭……誰稀罕吃那種一點都不高級、還會長胖的炸雞油炸食品啊……我想喝的，是你以前親自遞給我的。那種帶著甜味、還能讓人臉紅臉心跳加速的氣泡仙露啊。你可一定要……早點回來。」

……

與此同時。另一片完全不同的星辰之下。
另一個位面能級高到讓所有覺醒者都會瞬間崩潰、靈氣充盈到實體化的神祕世界。

高階劍修大世界。中州第一聖地——紫霄宮遺址。

曾經那片在那場諸神黃昏戰役中，變得荒廢凋零、慘遭滅門且被雜草覆蓋的宗門遺蹟。在此刻竟然煥發出了一種類似於「超時代科技」與「古樸仙俠風格」高度重疊、甚至顯得有些賽博仙俠感的復興生機。在原本應該是跪拜祖師爺的莊嚴主殿龍首。此時竟然威風凜凜地矗立著一座高達五十米。由神祕的天外銀色合金與古老禁咒陣法基石完美融合而成的、充滿了機械冷冽感的巨大建築。

建築的最頂端。一塊由幾萬顆靈石粉末驅動、巨大的 LED 全息投影光學牌。在此刻正閃爍著足以亮瞎所有自詡清高之劍修長老狗眼的。那種金光燦燦、充滿了商業氣息的絕世大字：

【萬界快遞·高階劍修分部 / 第一批發中轉站 / 24 小時自助提貨區】。

在下面的人群中。葉孤楓此刻正穿著一套蘇晨專門為其私人定製的、裁剪得極其合身、且胸口和背後都繡著顯眼的「萬界物流·使命必達」圖案的黑紅雙色半身運動裝。他依舊背著那柄在外界看來價值連城的、卻早已斷裂的歸墟傳奇長劍。此時卻像個在菜市場吆喝的小販。站在高高的靈石堆上。對著下方那幾千名、平時在外面都是跺一腳能讓大陸震三震的劍修長老、首領們。正扯著那早已沙啞的嗓子大聲咆哮著：

「都給我排好隊！那個御劍宗的那個什麼勞什子四長老！說的就是你！把你的那破飛劍給我麻溜地收起來！別想在老子這兒插隊！沒帶夠那種閃閃發光、純度達標的極品靈石原礦。你就算天王老子來了。也別想提走這最後半箱『五倍勁爽版·康師傅紅燒牛肉麵』！」

「還有你！那個什麼飄渺閣的帶頭掌門老太。也給我往後邊稍稍站！我蘇兄在交貨手冊上說得清清楚楚。這兒的規矩大過天道。我們是誠信經營。童叟無欺。只要靈石到位。哪怕你是一隻修煉成精的地穴豬。我也能讓你今天喝上最正宗的、帶著冰碴子的冰鎮大可樂！」

就在這充滿了各種違和感、卻又顯得生機盎然、甚至有些熱血沸騰的奇特時刻。

整個物資驛站最中央的那座巨大的、散發著凜冽劍氣的跨界傳送主陣法。突然間在一聲震撼了附近幾百里山脈的爆鳴聲中。發出了一陣通天徹地的、讓整座萬劍崖上萬柄沈睡寶劍都為之劇烈共鳴顫抖的燦爛金光！

場內所有原本還在為了一包辣條而爭得面紅耳赤的劍修長老們。在此刻全部如遭雷擊。整齊劃一地屏住了呼吸。他們驚受地感受到了一股前所欲有的、強大到讓他們連拔劍的勇氣都提不起來的。那種夾雜著異文明法則的恐怖降臨氣息。

「蘇兄——！！哈哈哈！我就知道。你這小子的物流承諾和那恐怖的『加急派送』速度。從來就不會讓我葉老二失望一次啊！」

葉孤楓在看清那金光中的虛影後。發出了一聲興奮到近乎發瘋的快意狂吼。他甚至直接在那幾千名長老目瞪口呆的注視下。毫無風度地一把丟失了那疊珍貴無比的快遞簽收清單。整個人化作一道足以割裂空氣的銀色劍芒。瘋狂地、甚至有些急不可耐地衝向了陣法緩衝中心。

金色的空間塵埃在山谷的微風中漸漸散去。

蘇晨依舊是那副平日裡穿著運動裝、胸前掛著快遞掃描儀、看起來有些慵懶且平凡的亞子。他背著那個巨大的、如同黑色小山般的實體化亞空間背囊。左手拎著整整兩箱剛剛在地球上推廣的『萬界新春爆款——衛龍地獄級加辣版辣條』。右手則抓著一瓶還在冒著讓人垂涎欲滴寒氣的、特大號冰鎮肥宅快樂水。他就那樣慢悠悠地。像是在自家老宅後花園遛彎一般。帶著一身位面跨越後的凌厲且神聖的氣息。從光圈中穩穩地走了出來。

他眼神微瞇。有些好笑地看著滿地那些正眼巴巴地對著他手中那箱散發著異界香味的辣條、正瘋狂吞嚥口水的所謂「劍仙」們。看著眼前這座由他提供設計圖紙。卻由葉孤楓那幫人一磚一瓦、用靈石強行堆砌起來的、充滿了科幻動感的宏偉中轉驛站。蘇晨深深地。對著長空吸了一口。那空氣中濃郁到幾乎要化作液態流動、讓他每一個沉寂已久的毛孔都在瘋狂戰慄歡鳴的高階純淨靈氣。

他丹田深處那枚早已饑渴難耐的「無垢劍種」。在這一刻。感應到了這絕世的修行土壤。終於發出了前所欲有的、如同萬龍升天般的歡快長鳴。

「久等了，葉兄。不好意思啊，今天臨海市那邊在龍池搞了點『大氣層級別的交通管制』。路上稍微有點堵車和幾隻蒼蠅擋路。耽誤了你這第一批加急件的提貨時間。這箱地獄辣條。就算是我這個代理人對你這大客戶的額外超時補償贈品。拿去給長老們壓壓驚。」

蘇晨對著葉孤楓咧嘴灿爛一笑。露出了那一排潔白、整齊且充滿了無限野心的牙齒。

「好啦。下一單。咱們這對跨位面的黃金拍檔。該往哪一個還沒開拓的的神祕世界……去送那足以震撼諸天的、帶著咱們自己LOGO的加急快遞了？」

天空之上。方圓百里內那上萬柄原本沈默了幾千年的塵封神劍。在此刻感應到了那股劍中王者的君臨氣息。
同時發出了響切雲霄、久久不散的萬劍齊鳴之音。

至此。這名為「萬界交易商」的傳奇成長篇章。在此刻。才正式在那由無盡星輝與跨維度文明交織而成的、浩瀚如煙海的諸天背景板之下。正式揭開了它那最為壯麗、也最為瘋狂的……星際派送序幕。
