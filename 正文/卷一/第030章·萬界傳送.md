# 第三十章 · 萬界傳送（卷一完）

臨海市的夏夜，終於恢復了如水的寧靜。

蘇晨獨自一人，站在老趙雜貨店的露台上。遠處是霓虹閃爍的現代都市，近處是充滿市井氣息的老弄堂。兩種截然不同的氣息在此刻交織，卻讓他感到一種前所未有的踏實。

「系統，確認傳送座標：劍修世界·萬劍崖。」

```
  【傳送程序啟動】
  ─────────────
  偵測當前方位：地球·臨海市。
  目標位面：劍修世界·第一星域。
  消耗：1000 中品靈石（已從儲物空間扣除）。
  穩定性：99.9%。
  ─────────────
```

一道淡淡的、肉眼不可見的透明漣漪，以蘇晨為中心擴散開來。

蘇晨回頭看了一眼放在桌上的那一疊新印好的快遞單。上面寫著幾個燙金的大字：【萬界快遞·使命必達】。

「從這張單子開始，這生意可就真的做大了。」

他笑了笑，正準備踏入那道裂縫。

「蘇晨！」

一聲急促的呼喊從樓梯口傳來。

林若溪氣喘吁吁地跑了上來，披風有些凌亂，臉上還帶著一絲未乾的雨水（或許是汗水）。她看著蘇晨身邊那扭曲的空間，眼神中閃過一抹複雜的痛苦。

「你要走了？」

「去出個差。」蘇晨語氣輕鬆，對著她揮了揮手，「去看看那個天天問我要可樂的劍修哥們。放心，我會回來的。」

林若溪抿了抿唇，從脖子上摘下一個精緻的、雕刻著冰霜花紋的玉墜，快步走到他面前，強行塞進了他的手裡。

「這是我林家的命牌。如果有危險，捏碎它，無論你在哪，我……我都想辦法去救你。」

林若溪的聲音很小，卻透著一股破釜沉舟的決然。

蘇晨看著手心中帶著餘溫的玉墜，心中微微一震。他伸出手，輕輕拍了拍林若溪的腦袋，感受著那如冰霜般清涼的髮絲。

「好。等我回來，請你吃全臨海市最好的炸雞。」

說完。

蘇晨轉身，身形沒入那道金色的裂縫中，徹底消失不見。

林若溪站在空蕩蕩的露台上，看著繁星點點的夜空，良久，悠悠地嘆了一口氣。

「大木頭。誰要吃炸雞啊……我想喝你那種帶著氣泡的仙露。」

……

與此同時。

另一片星空下，另一個位面。

劍修世界，紫霄宮。

曾經荒廢的宗門此時已經煥發了生機。在原本的主殿位置，矗立著一座高達五十米、充滿了科幻與仙俠融合風格的巨大建築。

建築的頂端，一塊巨大的全息投影牌匾正閃閃發光：

【萬界快遞·劍修分部 / 24小時自助提貨區】。

葉孤楓穿著一身裁減得體的……快遞員制服（改裝版），背著那柄斷劍，正對著下面幾千名排隊領取「特供泡麵」與「冰鎮可樂」的劍修長老們大聲吼著：

「排隊！排隊！說你呢！那個御劍宗的長老，沒帶靈石別想提貨！」

「還有你！別以為你是掌門就能插隊！我蘇兄說了，誠信經營，童叟無欺！」

就在這時。

驛站中央的傳送陣突然發出一陣通天徹地的金光。

所有劍修同時停下了動作，驚駭地感受到一股強大、陌生且充滿了生機的氣息降臨。

「蘇兄！」

葉孤楓發出一聲興奮的狂吼，丟下手中的快遞單，瘋狂地衝了過去。

金光散去。

蘇晨背著一個巨大的、漆黑的儲物背包，手裡拎著兩箱最新的「萬界爆款——衛龍辣條」，慢悠悠地走了出來。

他看著滿地的劍修，看著前方那座宏偉的快遞驛站，深深吸了一口氣。那空氣中濃郁到實體化的靈氣，讓他體內的無垢劍種發出了前所欲有的歡鳴。

「葉兄。不好意思，路上有點堵，耽誤了幾分鐘。」

蘇晨咧嘴一笑，露出一口潔白的牙齒。

「下一單，送往哪裡？」

天空之上，萬劍齊鳴。

萬界交易商的傳奇篇章，在此刻，才剛剛揭開真正壯麗的序幕。
