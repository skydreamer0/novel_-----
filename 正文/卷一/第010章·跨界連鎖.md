# 第十章 · 跨界連鎖

「蘇大哥，這、這又是什麼上古神器？它發出的光，竟然連我的蔽息術都能照穿！」

靈草大世界。

李小苗正縮著身子趴在一塊佈滿青苔的巨石後，瞪大了那雙黑溜溜的水靈眸子，驚奇地仰視著蘇晨剛剛穿透虛空縫隙遞過去的工業級強光探照燈。那道筆直、銳利、帶著金屬質感且足以刺破迷霧的冷白光束，在靈氣氤氳卻常年昏暗的深山深處，簡直如同白晝真神降臨，驚得遠處幾頭守護靈藥的猴妖紛紛發出不安的尖叫。

「這玩意兒叫『破魔真光鏡』。」蘇晨在手機屏幕這端，臉不紅氣不喘地胡說八道，語氣誠懇得像個傳道受業的宗師，「只要你按下側面那個銀色的法紋按鈕，管它什麼萬年陰魂草還是劇毒沼氣，在它面前都得無所遁形。小苗，看到對面那座白骨皚皚的鷹愁澗了嗎？你念叨了好幾次的地心聖乳就在半山腰那個閃著微光的水簾背後，有這神器加餐，你敢不敢拚一把？」

「有蘇大哥給的這條『捆仙索』保命，我、我也敢去拚个前程！」

李小苗用力拉扯了一下身上那套由高強度碳纖維編織、能輕鬆承受五噸拉力的專業戶外防墜落攀爬吊帶。這種帶著各種防護卡扣與滑輪的現代工業結晶，在小苗眼裡，簡直比她那位元嬰期師父整天掛在嘴邊的本命長巾還要堅韌百倍、玄妙千倍。

蘇晨一邊喝著溫熱的茶水，一邊隔著屏幕看著那個像機靈的小猴子一樣、在千米懸崖自如攀爬的娇小黑影，嘴角忍不住勾起一抹得逞的弧度。

這就是他為自己升級後的系統規劃出的「連鎖升維交易鏈」。

他先用廉價卻高效的現代科技裝備武裝李小苗，讓她具備原本境界根本不可能擁有的採集能力，獲取極品資源；接著，他會用李小苗採集來的頂級溫養靈藥去救治劍修位面那個落難的絕世天才葉孤楓；最後，他再利用救命之恩，從葉孤楓手中換取那個殺伐位面最頂尖的戰鬥功法。

這才叫真正的位面交易商——低成本引入、高效率運作、中間商賺取足以讓人瘋狂的跨界差價。

「蘇、蘇大哥！我拿到了，我真的拿到了！」

屏幕裡，李小苗滿臉通紅，原本髒兮兮的小臉上滿是抑制不住的狂喜。在她那雙布滿細微傷痕的手心，一團約莫雞蛋大小、散發著乳白色神聖光暈、正如同心臟般有節奏跳動的人形液體，正靜靜地懸浮著，將周圍黑暗的洞穴映照得如夢似幻。

```
  【物資解析】
  名稱：[至純·地心聖乳]
  品質：SS（靈草世界特產）
  特效：生基重塑。蘊含極其恐怖且溫和的先天生機能量，可活死人肉白骨，對遭受毀滅性破壞的經脈具備「基因級」的重組與疏通奇效。
```

「好樣的小苗！立刻手心向上，心裡默念傳送。這包口感絕佳、在我們這兒只有皇室能享用的『五彩神仙糖』（其實就是家樂福超市十塊錢一包的水果硬糖）就是你應得的勞務報酬了。」

蘇晨強壓住內心的暴富心態，迅速完成了這筆跨越次元的「暴利交易」。

看著那團液態黃金般的地心聖乳安靜地躺在自己的亞空間儲物間內，蘇晨沒有半點耽擱，修長的手指輕輕一劃，窗口瞬間切換，鎖定在了那個黑暗死寂、充滿了肅殺之氣的萬劍崖底。

……

劍修大世界，萬劍崖，禁靈魔窟。

儘管已經過去了數日，葉孤楓依舊感覺自己的身體像是被千刀萬剮後又丟入冰窖中反覆碾壓，每一寸骨骼每一根髮絲都在痛苦地呻吟。

他曾是那片大陸劍道同輩中公認的第一天才，曾單手握劍，一人一馬橫壓帝國三千里，引得無數天驕盡折腰。可現在，他雙腿被廢、脊椎微駝，全身上下密密麻麻地被插滿了宗門特製、專門用來鎮壓邪修與叛徒的「鎖靈蝕骨針」。

「咳……咳咳……」

葉孤楓在幽暗中咳出一口混雜著內臟碎塊的暗紫色淤血，但即便身陷這種不見天日的泥淖，他的眼神依舊銳利如同一柄出鞘的寒劍，不帶絲毫求饒的軟弱。

「想讓我交出《歸墟劍典》的最後一卷？你們這些躲在幕後放乾我血的蛇鼠之輩，簡直是癡人說夢。」

就在他心存死志、打算燃燒靈魂深處最後一絲殘存劍意與這魔窟玉石俱焚的剎那，眼前那道曾經出現過的神祕虛空裂縫，再次無聲無息地裂開了。

那個穿著一件明黃色「神祕道袍」（快遞短袖）的奇怪男人，又一次毫無危機感地探出了頭。

「喲，哥們，這環境可不怎麼利於傷口癒合啊，還硬撐著呢？」

蘇晨看著螢幕裡葉孤楓那慘不忍睹、簡直像個刺蝟一樣的悽慘模樣，眼皮也不禁狂跳了幾下。這傢伙渾身散發出的那股「老子就算死也要咬掉你一塊肉」的寧折不彎之氣，哪怕是隔著厚厚的屏幕與維度障礙，都刺得蘇晨的皮膚隱隱生疼。

「滚。不管你是什麼位面的邪魔外道，若想來看葉某人的笑話，那你找錯人了。」葉孤楓的嗓音嘶啞如磨砂，那是刻在靈魂深處的孤高。

「看笑話？不不不，我是個商人，商人眼裡只有雙贏。」蘇晨手腕一翻，一隻通體透明的高強度玻璃試管出現在他指尖，裡面晃動著那團剛剛到手的地心聖乳。
「重塑破裂經脈，徹底洗髓伐骨，重新點燃萎縮的生機。這東西，你現在缺不缺？」

葉孤楓原本如同死灰般的雙瞳，在觸及那團綻放著神聖白光的液體瞬間，瞳孔驟然收縮到了針尖大小，一股名為「渴望」的情緒瘋狂地在眼底翻湧。

「地心聖乳……不，這股精純的生命波段，這是傳說中誕生自世界原胎的『聖乳之王』！在我們這兒，那是仙門領袖都要為之打破頭的禁忌神物。」

葉孤楓到底是頂尖劍修，一眼就認出了此物的價值：「你一個連靈氣都沒凝實的異鄉人，從哪兒弄來的？你背後到底是哪方勢力？你想要我付出什麼？」

「生意人，只講究爽快。」蘇晨咧嘴一笑，露出一副自認帥氣的表情，「我要一門真正的劍招。要求只有三點：第一，修煉速度要快；第二，殺傷力要夠狠；第三，最好是那種能讓我以二階（凝氣境）的實力，能在出其不意之下瞬間切開三階強者喉嚨的保命絕學。」

葉孤楓陷入了長久的沉默。
他修煉的《歸墟劍典》是凌雲劍派的立宗之本，是整個劍修位面最霸道、最專注於「毀滅」二字的殺生大術。

「宗門規矩，不傳外人。即便我被棄如敝履，但劍典是我的武道尊嚴。」葉孤楓咬牙道，這是他最後的堅持。

「兄弟，你那是愚忠。」蘇晨循循善誘，語氣中帶著一種長輩般的『關懷』，「你那宗門現在把你關在這兒放血、拔骨，想盡辦法逼你交出功法，你覺得你還有義務去守護他們的什麼基業？再說了，我所在的世界和你不在一起，中間隔著幾萬個平行時空，我學了也沒人認識。你若死在這兒，這神妙的傳承才是真的成了絕響，你甘心嗎？」

見葉孤楓的手指在劇烈顫抖，蘇晨又加了一枚重磅籌碼。

「這樣吧，劍典的全本我不要，我也沒那個時間去慢慢磨練。你只需要把這卷功法中威力最大、最不需要長時間蓄勢的那一招死招給我，再加上你這二十年來對劍道的一點點心得體會。

作為回報，我不僅把這管起死回生的聖乳給你，我還額外送你一樣我們世界的『護法神器』——它能讓你在目前靈氣全無、經脈未復的虛弱狀態下，也具備擊殺那些尋常探子、自我防護的能力。」

說著，蘇晨從儲物空間裡掏出了一個造型硬派、黑色磨砂塗裝、頂端跳動著藍色電弧的沉甸甸物體。

那是他昨晚在機電市場買的、私自拆解並強化了電容輸出、足以釋放二十萬伏特高壓的重型電擊槍，且後面還貼心地黏著一塊輕量化太陽能便攜充電板。

「這玩意兒，只要你對準敵人按下去，一瞬間就能爆發出如同天劫雷霆般的毀滅性能量。你們那兒的修仙者，雖然能御劍飛行，但應該也很怕被雷劈吧？」

葉孤楓死死盯著那個充滿了冷酷工業美感、雕刻著神祕黃色三角形危險符號的奇異法器，心中竟然莫名其妙地升起了一股前所未有的心驚肉跳。

雷霆之力……這凡人手中，居然能掌握如此精純且穩定、不需要任何手印引導的雷法？這也太離譜了。

「好。既然這天要亡我宗門，那我葉孤楓便也做一回叛逆。我換！」

葉孤楓原本虛弱的右手猛地抬起，一點刺目至極的金色劍芒從他眉心激射而出，穿透了層層虛空阻礙，直接沒入了蘇晨的識海深處。

```
  【跨界交易成功：史詩級大賺！】
  ─────────────
  您失去：[地心聖乳]、[強力電擊槍+電源件]
  您獲得：[歸墟劍意·序章：瞬殺]（完整肌肉記憶傳承）
  您獲得：[劍修悟道殘卷·凡靈篇]
  系統積分：+500（當前總覽：800）
  ─────────────
```

「嗡——！」

一股足以炸裂腦髓的刺痛感席捲而來，蘇晨感覺有無數柄透明的飛劍在自己並不寬廣的海識中瘋狂開天闢地。

那一式命為「瞬殺」的劍招，在傳承中極其簡單，只有兩步：拔劍、收劍。

但在這短暫的、連零點一秒都不到的時間差裡，所蘊含的速度曲線與暴虐殺意，卻讓蘇晨這現代人的心臟開始瘋狂搏動。他有一種強烈的錯覺，即便現在周家那位老祖宗站在面前，只要他手握一柄稱手的鐵條，就能在對方反應過來施法之前，俐落地削掉對方的鼻尖。

「合作愉快。葉兄弟你趕緊喝藥，先把那些『毒針』給逼出來。下次如果有搞不定的『快遞任務』，隨時記得呼叫我，二十四小時在線。」

蘇晨強忍著虛脫感關閉了光幕，隨後整個人像是脫水一般仰面倒在大床上，大口喘氣。

他的外在境界雖然依舊卡在二階凝氣，但有了這一式恐怖的殺伐劍招，以及手頭這波瘋狂掠奪來的跨界差價物資，他感覺臨海市那什麼自命不凡的「周家」、還有那個神祕莫測的「蛻變者」，在他的劇本裡，已經開始變得有些……『可愛』且無害了。

「叮鈴鈴——叮鈴鈴！」

一陣刺耳的手機鈴聲突然打破了屋內的寂靜。

蘇晨掃了一眼屏幕，居然是黑市那個不著調的二道販子趙德厚（老趙）打來的。

「喂，蘇小子。你沒給人煮了吃吧？」老趙那特有的煙嗓聲傳來，但這一次，語氣裡卻透著一絲難得的清醒與嚴肅。

「活得挺潤的，剛賺了筆大錢。老趙，大半夜的你這是有何觀點？」

「天衡局那幫雖然守規矩、但嘴不嚴的傢伙，已經把你在東港倉庫的戰鬥數據同步到地下黑市的懸賞網了。你現在在黑道上的報價，可是整整三萬顆靈石外加一條晉升渠道。」
老趙狠狠啐了一口痰，壓低聲音道，「聽說有一幫『蛻變者』的瘋子，正盯著你手裡那塊從幽冥犬身上挖出來的黑色源質結晶。這幾天臨海不太平，今晚你別在那破爛公寓待著，誰敲門都別開，聽懂沒？」

「趙叔，你看我蘇晨這一年多送快遞，什麼樣的惡犬沒見過？你看我像是那種會縮在被窩裡發抖的人嗎？」蘇晨看著掌心那截若隱若現、散發著冰冷青芒的拔劍印記，嘴角勾起一抹極度危險的弧度。

「……媽的，老子就知道你這小子骨子裡透著一股沒人性的狠勁。想找死別死在我們蛇巷門口，收拾起來晦氣！」

老趙啪地一聲掛斷了電話。

蘇晨站起身，緩步走到窗邊，拉開一絲窗簾往下窺視。此時夜色已深，繁華的臨海市依舊霓虹閃爍，但在那璀璨燈火照不到的陰影角落，激流已經開始由暗轉亮。

而這一次，他不再是那個只能在窄巷裡靠著一身蠻力逃生的底層快遞員。

他是遊走於諸天萬界、掌控著降維資源的——最強交易商。
