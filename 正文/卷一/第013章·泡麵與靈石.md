# 第十三章 · 泡麵與靈石

回到天衡局分配的臨時宿舍，蘇晨二話不說，先給房門落了三道鎖。

昨晚的公寓爆炸案雖然讓他損失了一堆家具，但也讓他看清了一件事：在這都市暗面混，沒有足夠的「硬通貨」打底是行不通的。符籙、丹藥、法寶，哪一樣不要錢？

天衡局那八萬塊獎金，在普通人眼裡是鉅款，但在覺醒者的物資清單裡，可能連一塊稍微像樣點的靈石都買不起。

「系統，連線劍修世界，葉孤楓。這哥們吃了我的地心聖乳，現在應該生龍活虎了吧？」

```
  【連線中...】
  ─────────────
  交易對象：劍修世界·葉孤楓
  對象狀態：狂暴、復仇中、靈氣飢渴
  位置：萬劍崖外圍（剛剛殺出重圍）
  ─────────────
```

光幕中，葉孤楓的身影再次浮現。

比起之前的垂死掙扎，現在的他簡直換了個人。原本乾涸的身體重新煥發出生機，破損的白衣隨風狂舞，背上竟揹著一台畫風極其不和諧的高壓電擊槍，而他的右手中，則握著一柄從追兵手裡奪來的青鋼長劍。

此時，在他的腳下，橫七豎八地躺著十幾具同門弟子的屍體。

「蘇兄！」葉孤楓聽見蘇晨的聲音，手腕一轉，劍尖斜指地面，冷冽的眸子裡閃過一抹狂熱，「你那『雷霆神器』當真神妙！剛才那內門長老想用『鎖雲手』拿我，被我一槍捅過去，整個人電得外焦裡嫩，連元神都麻了三息！」

蘇晨聽得一陣牙疼。能把電擊槍發揮到跨界戰鬥的水平，這葉孤楓也真是個人才。

「電得好。不過葉兄，你現在靈氣似乎消耗得很厲害？」

「不錯。」葉孤楓抹了一把臉上的血跡，神色有些陰沉，「這地心聖乳雖救了我命，但此間天地靈氣貧瘠。我剛殺出宗門，身上的靈石已消耗殆盡。若再無補給，怕是逃不出這三千里劍域。」

蘇晨心中一動，果然，機會來了。

「靈石？我這兒有的是好東西能讓你快速恢復實力，而且……還能讓你飽口腹之欲。」

蘇晨說著，從儲物空間裡掏出了一樣壓箱底的傳說級物資——

【老罈酸菜牛肉麵·加大加量裝】。

而且這不是一包，是整整十箱。

「這叫『五行聚靈麵』。」蘇晨面不改色地吹噓，「其中蘊含的百年陳醋酸意，能瞬間洗滌你的經脈疲役；那烘培過的麵餅，更是濃縮了大地精華。葉兄，你看這交易……」

葉孤楓看著那五顏六色的包裝袋，尤其是聞到蘇晨當場泡開一碗後散發出的那種霸烈、辛辣、酸爽的絕世香氣，肚子竟不爭氣地發出了一聲雷鳴。

他這輩子修仙，辟穀吃了幾十年，哪見過這種刺激五感的神物？

「好香……這氣味，竟比回天丹還要誘人！」

葉孤楓眼神熾熱，他能感覺到這「聚靈麵」中雖然沒有多少靈氣，但那種能讓人精神一振的神祕調味物質，簡直是他現在這種疲命天涯者的救星。

「蘇兄快人快語，你需要什麼？我這剛抄了外門長老的私庫，除了靈石，我也沒別的了。」

葉孤楓說著，隨手一揮，地面上瞬間堆起了一座亮晶晶的小山。

那是下品靈石，每一塊都切割得整整齊齊，散發著誘人的乳白色光澤。

「我這兒有三千塊下品靈石，二十塊中品靈石。蘇兄若是不嫌棄，這全都給你！只要你把這『聚靈麵』和之前的『仙露可樂』再給我一些，足矣！」

蘇晨眼角抽動，心臟險些停跳。

三千塊下品靈石？

在地球的覺醒者黑市，一塊最垃圾的下品靈石碎渣都要賣到上萬人民幣，而且還常常有價無市。這三千塊品相圓滿的靈石，如果換算成人民幣……那得是多少個零？

「成交！葉兄果然爽快！」

蘇晨強壓住內心的狂喜，快速將十箱泡麵、五箱辣條、三箱可樂全部傳送了過去。

```
  【交易成功】
  ─────────────
  您獲得：下品靈石*3000、中品靈石*20
  您獲得：劍修世界通用防護內甲（軟質）*1
  評級：★★★★★★★（絕對暴利！）
  ─────────────
```

看著瞬間塞滿了儲物空間的靈石小山，蘇晨只覺得有一種暴發戶的眩暈感。

「發財了……房東太太，您那房租我能買下你整棟樓！」

蘇晨隨手抓起一塊靈石，運轉《長青吐納術》。

「嘶——」

一股精純到恐怖的靈氣順著手心瞬間湧入丹田。如果說平時吸收空氣中的靈氣是抽水，那這靈石簡直就是直接往肚子裡灌營養液。

僅僅用了三分鐘，蘇晨體內的靈氣就徹底回滿，甚至原本二階初階的境界都在這一刻變得圓潤無比。

「有此物助陣，二階中階……指日可待。」

就在蘇晨沉浸在財富自由的喜悅中時，識海中的系統突然叮的一聲，發出了紅色的警報。

```
  【跨界連鎖觸發預警】
  ─────────────
  交易夥伴『李小苗』遭遇危險。
  地點：靈草世界·百藥谷。
  原因：因採摘地心乳被同門嫉恨，現被藥谷執法隊圍堵，生命危險中。
  ─────────────
```

蘇晨的眼神瞬間冷了下來，方才的喜悅被一股護短的殺意所取代。

李小苗是他最重要的供應商，更是他在這孤寂修仙路上的第一位「小朋友」。想動他的「快遞專員」？門都沒有。

「系統，連線李小苗，強制開啟虛擬投影投射！」

蘇晨猛地站起身。

「小苗，別怕，蘇大哥給你送『大規模殺傷性武器』來了！」

---

*章末提示：在百藥谷隱祕的山洞口，李小苗正瑟瑟發抖地抱著那台現代高強光探照燈，對面則是十幾個滿臉貪婪的藥谷弟子。而就在這時，一道裂縫在半空開啟，蘇晨冰冷的聲音迴盪在山谷之中。*
