# 第十三章 · 泡麵與靈石

回到由天衡局安排的那間位於分局後方的臨時特勤宿舍，蘇晨第一件事不是脫掉那身沾滿灰塵的外賣服，而是二話不說，反手給那扇厚重的防彈防核級金屬大門連落了三道電子鎖與兩道物理栓。

昨晚那場險些讓他粉身碎骨的公寓爆炸案，雖然讓他損失了一堆辛辛苦苦掙來的二手家具和家電，但也讓他徹底認清了一個生存法則：在這波譎雲詭、靈氣復甦的都市暗面混跡，如果沒有足夠數量的「跨界硬通貨」打底，是絕對行不通的。符籙、丹藥、法寶，甚至是一門像樣的躲避身法，哪一樣不需要海量的資源去置換？

天衡局發放的那八萬塊獎金，在普通工薪族眼裡或許是筆不得了的鉅款，但在覺醒者的地下物資清單裡，這點錢可能連一顆最低品質、稍微像樣點的靈石碎渣都買不到，更遑論那些能跨境殺伐的神器。

「系統，立刻跨位面連線劍修大世界，葉孤楓。這哥們之前生吞了我送過去的地心聖乳，按理說現在應該已經在那是是非非的宗門裡生龍活虎了吧？」

```
  【跨位面通訊連線中...】
  ─────────────
  連線對象：劍修大世界·落難劍子·葉孤楓
  對象當前狀態：極度狂暴、復仇執行中、體內靈氣嚴重枯竭。
  當前座標：凌雲劍派·萬劍崖外圍三里處（剛剛以一敵眾，殺出重圍）。
  ─────────────
```

伴隨著系統提示音，光幕中，葉孤楓的身影再次在蘇晨窄小的單間內立體投射而出。

這一次，葉孤楓比起之前的垂死慘狀簡直像是換了個靈魂。他原本乾涸枯敗的身體在聖乳的滋養下重新煥發出了驚人的生機，那一頭沾滿血跡的黑髮在狂風中肆意凌亂。

最讓蘇晨感到畫風炸裂的是，這位英氣逼人的大劍仙，此刻手中提著一柄從追兵手裡奪來的溢血青鋼長劍，可側腰上卻背負著那台充滿了科幻工業冷硬感的現代化重型電擊槍，而那塊閃閃發亮的太陽能充電板正被他當作一塊奇異的「戰甲掛件」固定在後背心。

此時的葉孤楓正站在一片冰冷的雪地上，他的腳下，橫七豎八地躺著十幾具昔日同門長老的屍體，鮮血染紅了大片雪原。

「蘇兄！竟然是你！」葉孤楓聽見虛空中傳來蘇晨的聲音，手腕猛地一轉，劍尖斜斜指向地面，一滴殘血順著劍身滑落。他那一雙原本死寂的眸子裡，此時閃爍著一種近乎狂熱的復仇快感：
「你贈予我的那件『雷霆神器』當真神妙無比！方才宗門那眼高於頂的內門執法長老想用『鎖雲擒拿手』強行拿我，可誰知我一槍（電擊槍）捅過去，那人全身僵直，整個人被電得外焦裡嫩，連本命元神都當場麻痺了足足三息時間！正是這三息，讓我一劍取了他的狗命！」

蘇晨聽得一陣牙疼，眼角忍不住微抽。能把區區一台改裝電擊槍，強行開發出跨位面越級戰鬥的套路，這葉孤楓也真是個被修行耽誤了的「戰術天才」。

「電得好，這種貨色就該有這種死法。不過葉兄，我看你氣息浮躁，面色慘白，你體內的靈氣似乎消耗得相當慘烈啊？」

「不錯。」葉孤楓有些吃力地抹了一把臉上的粘稠血跡，神色陡然陰沉下來：
「這地心聖乳雖救了我一命，並重續了我的經脈，但此間天地早已被宗門的鎖靈大陣禁錮，靈氣稀薄如水。我剛才拼老命殺出宗門，身上的儲備靈石已在那場血戰中消耗殆盡。若再無後續補給，怕是根本逃不出這連綿三千里的劍域追殺。」

蘇晨心中猛地一動，隱藏在快遞服下的瞳孔微微收縮——果然，對於商人來說，任何危機都是最高價值的機會。

「靈石補給？這在別處或許難辦，但我這兒恰好存了一批能讓你快速恢復巔峰戰力，且……還能讓你飽受這幾十年來口腹之欲缺失的好東西。」

蘇晨說著，嘴角勾起一抹神祕的笑意，從亞空間儲物間裡緩緩掏出了一樣在他看來是「降維打擊」級別的絕世物資——

【老罈酸菜牛肉麵·特製加量大桶裝】。

而且這不是孤零零的一包，是整整十箱，整齊得像是一座小型堡壘堆在宿舍的地板上。

「這玩意兒在我們這兒叫『五行混元聚靈麵』。」蘇晨面不改色地開始了他的跨界忽悠，「看到那包紫色的料包沒？其中蘊含著採集自深海的百年陳醋酸意，能瞬間洗滌你經脈中因為戰鬥累積的疲勞毒素；而那烘培過的金黃麵餅，更是濃縮了大地土系的渾厚精華。葉兄，你看這筆交易，能否解你燃眉之急？」

葉孤楓看著那包裝袋上印著的、色彩繽紛且極具現代美工誘惑力的牛肉圖案，尤其是當他那靈敏的嗅覺捕捉到蘇晨當場為了示意、而撕開包裝泡開一碗後，散發出的那種辛辣、濃郁、極具化學誘惑力的酸爽香氣時，他那這輩子只吃過幾粒辟穀丹的胃，竟極其不爭氣地發出了一聲雷鳴般的共鳴。

他這輩子修的是「無欲劍道」，可哪見過這種直接攻擊人類生物本能的神祕食物？這香味，簡直比宗門禁地裡那幾顆所謂的長生桃還要動人心魄。

「好香……這氣味中，竟透著一股能喚醒靈魂的霸道力量！」

葉孤楓眼神熾熱如火，在修仙者的感知中，這「聚靈麵」中雖然沒有什麼高濃度的天地靈氣，但其中那種被稱為「味精」和「脫水蔬菜」的神祕物質，產生的強烈多巴胺反饋，簡直是他這種亡命天涯、心神俱疲之人的救贖。

「蘇兄果真是神人也！你需要什麼？儘管開口！我葉某人剛才抄了外門幾位死鬼長老的隨身私庫，除了這些世俗靈石，我也沒什麼拿得出手的身外物了。」

話音剛落，葉孤楓不帶半點猶豫地隨手一揮，光幕對面那冰冷的雪地上，瞬間堆起了一座亮晶晶、散發著聖潔光芒的小山頭。

那是靈石。整整三千塊下品靈石，以及夾雜在其中的二十塊品質極高的中品靈石。每一塊都被切割得稜角分明，內部有無數乳白色的靈氣絲線在緩緩流動。

「我這兒目前只有這麼多。蘇兄若是不嫌棄，這全都給你！我只要你把這『聚靈麵』和之前的『快樂仙露』多給我一些，供我在這冰原逃亡途中撐過去，足矣！」

蘇晨站在宿舍中間，腳指頭都忍不住激動地摳了兩下地板，心臟險些從嗓子眼蹦出來。

三千塊下品靈石啊！
在現在地球那亂成一鍋粥的覺醒者黑市上，一塊指節大小、充滿雜質的品級靈石碎渣，都能被那幫土豪們炒到上萬人民幣，而且還常常是有價無市。眼前這三千塊品相圓潤、無暇級別的靈石，如果真的兌換成世俗的財富……那得是多少個零？那簡直是富可敵國！

「成交！葉兄你我一見如故，這點『聚靈麵』權當是在下給哥們墊肚子用的！」

蘇晨強壓住內心的狂喜，動作麻利地將十箱泡麵、五箱爆款辣條、以及三箱原本用來解渴的可樂罐，全部通通過跨位面通道傳送了過去。

```
  【交易圓滿達成】
  ─────────────
  您失去：現代生活物資一堆（成本：約RMB 1200元）。
  您獲得：下品靈石*3000、中品靈石*20。
  附加贈品：[劍修界特製·蠶絲軟甲]（防禦中級以下物理殺傷）。
  當前宿主財富值：系統預估已達成臨海市首富初步標準。
  ─────────────
```

看著瞬間塞滿了整整一立方米亞空間儲物間的靈石小山，蘇晨直覺得兩腿發軟，腦袋裡嗡嗡作響，那種暴發戶的眩暈感簡直比喝了假酒還要猛烈。

「發財了……真的發財了！周家，你們這幫想回收我的老狐狸，老子現在一塊靈石砸過去，估計都能埋了你們的祖墳！」

蘇晨顫抖著手抓起一塊通體冰涼的下品靈石，試探性地運轉體內的《長青吐納術》。

「嘶——呼！」

一股精純到近乎粘稠的靈氣，順著他的手心勞宮穴，如同暴雨決堤一般瞬間湧入了他的丹田之中。如果說平日裡吸收外界那稀薄到可憐的空氣靈氣叫「抽水」，那現在這靈石簡直就是在直接往他的四肢百骸裡灌高能營養液。

僅僅用了不到三分鐘的時間，蘇晨體內原本乾涸的靈氣就徹底回滿，甚至原本卡在二階初階的那個門檻，都在這一刻被那股澎湃的靈力沖刷得搖搖欲墜。

「有此海量資源助陣，二階中階境界……今晚我就能強行破掉！」

然而，就在蘇晨正準備沉浸在這種「用錢砸修為」的頂級快感中時，識海中的系統光幕突然毫無徵兆地轉為了刺眼的深紅色，發出了刺耳的等級一警告音。

```
  【跨界連鎖·緊急危難警報】
  ─────────────
  重要交易夥伴『靈草大世界·李小苗』遭遇生命威脅！
  座標定位：藥草位面·百藥谷幽林深處。
  危機詳情：因先前採摘「地心聖乳」消息走漏，引起同門嫡系弟子的極度嫉恨與貪婪。現被執法小隊以「盜取宗門至寶」的名義圍堵在禁地山洞，存活率正在急速下降。
  ─────────────
```

蘇晨原本喜色洋洋的臉龐，在看完這行文字的剎那，瞬間變得冷若寒霜。一股前所未有的護短殺意，從他這平日裡看起來吊兒郎當的快遞員身上肆意散發。

李小苗不但是他目前最重要的初級資源供應商，更是他在這孤寂、冰冷的修仙升級路上遇到的第一個對他報以純真信任的「小朋友」。

想動他的「專屬快遞員」？問過他手裡的現代化火力沒有？

「系統，別廢話！立刻連線李小苗，我要強制開啟全景投影投射到她身邊！」

蘇晨猛地站起身。

「小苗，別怕，把探照燈關了，躲到石頭後面閉上眼睛！蘇大哥這就給你傳送幾箱跨次元的『大規模殺傷性禮包』來救場！」
