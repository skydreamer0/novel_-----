# 第018章 · 狂暴升級

雨，越下越大，像是在為這座即將陷落的城市提前舉辦一場盛大的葬禮。

蘇晨的身形在空曠、支離破碎的東北區大道上疾馳。每一腳踏在積水坑中，都會激起數米高的、帶著淡淡血色的浪花。他此時的身體，正處在一種極端危險、也極端震撼的奇異狀態。

「噗、噗、噗——！！」

他全身上下的每一個毛孔。此時都在向外噴射著細微卻密集的血霧。那是因為體內那五百枚中品靈石在一瞬間被強制粉碎、化作了如同狂暴海嘯般的純淨能量，超出了他目前那具三階肉身所能承載的極限，正在進行著最為原始、也最為慘烈的「擴容」與「重組」。

「唔……啊！！」

蘇晨發出一聲低沈而壓抑的悶吼。他感覺像是有幾千根燃燒著的紅燙鋼針，正順著他的脊髓、順著他的血管，每一寸、每一毫地在瘋狂鑽探。而在他的識海深處。系統那原本清脆的提示音，在此刻也變得忽遠忽近，帶著一種電子信號被干擾後的扭曲感。

```
  【升級度：96%... 97%... 98%...】
  - 警告：宿主脊椎骨密度已達到 300% 受壓上限，即將發生晶質化轉變。
  - 當前心率：240次/分，血氧飽和度驟降，正切換至位面供能模式。
```

就在那如同靈魂被生生撕碎的劇痛到達頂峰的一秒鐘，蘇晨猛地抬頭。

迎面而來的。是一頭雙翼展開足有十米寬。渾身長滿了黑色倒鉤羽毛、實力已達三階中期的「高空獵隼」。這頭怪物顯然是被蘇晨身上那股由於突破而散發出的、濃郁到了極致且夾雜著鮮血香氣的靈壓所吸引，牠正張開那足以切開裝甲板的金屬利爪，帶著刺耳的音爆聲，直取蘇晨的咽喉！

「區區雜毛畜生，也想攔我蘇某人的快遞路？！」

蘇晨眼神中暴射出兩道長達寸許的、代表了極致憤怒與劍意的銀白色精光。

他沒有躲，甚至連停頓的意思都沒有。他的右手食指中指併攏。在那怪物抓到他臉皮的前零點一秒，以一種近乎瞬移的頻率。自下而上地輕輕一划。

「歸墟——第一式：瞬殺。」

「噗——！」

沒有華麗的光影爆炸，只有一種極致純粹的物理與靈能的雙重切割。

那頭剛才還不可一世的三階獵隼。甚至連一聲哀鳴都沒來得及發出，牠那巨大的、覆蓋了源質防禦層的軀體。竟然就那樣在那一道黑色劍氣中，整整齊齊地、毫無滯礙地被一分為二。鮮紅的妖血在雨幕中綻放出一朵淒美到了極點的血色花朵，隨即被狂風吹散。

蘇晨看都沒看那具正打著旋墜落的屍體。

「破——！！」

他的身體內傳出一聲清脆、如同某種古老封印被強行敲碎的玻璃碎裂聲。

在那一瞬。他腳下的柏油馬路。以他為中心點。像是被重磅炸彈正面命中般，瞬間塌陷出一個方圓十米的巨型凹坑，裂痕瘋狂向四周蔓延。一股呈現象牙白色、透著一股令人心底發寒的寂滅劍意的狂暴靈壓，宛如一道平地驚雷。在臨海市東北區的天空下正式炸響。

三階巔峰……達成！

甚至。隱約觸碰到了那層傳說中的、連秦老都夢寐以求的四階門檻。

蘇晨感受著體內那股脫胎換骨後。正緩緩平靜下來、卻變得更加凝實厚重的「歸墟劍氣」，他這才緩緩吐出一口帶血的廢氣，伸手抹了一把臉上的雨水。

此時的這座城市，早已失去了原本的祥和。
遠處。那頭身披青色地獄火、正將數棟摩天大樓當成棲息枝椏的「焚天青鸞」，正感受到了這股來自蟻穴般的挑戰之氣，牠那雙足有磨盤大小、散發著熔岩溫度的赤色妖瞳。在此刻緩緩轉動。死死地鎖定了正在街道中心、顯得極其單薄的少年身上。

「蘇晨……你居然沒死？」
林若溪此時帶著幾名渾身是傷的調查員部下，狼狽地從一輛翻倒的裝甲車後探出頭。當她看到那個在火光與雷暴中。渾身散發著陌生且可怕氣息的少年時。她那一向冷傲的臉上第一次露出了見了鬼般的驚駭表情。

「林隊長。帶著你的人去保護平民撤離。這頭大鳥。」
蘇晨緩緩從儲物格中再次取出一張通體冰藍色的。散發著足以凍結大地的高階符籙。「交給我這家快遞公司負責人……親自簽收。」

他手中拿著的。是剛從萬界系統中。用剩餘點數緊急兌換的、曾鎮壓過汪洋巨獸的「極寒冰蛟符」。

一場真正具備了神話色彩、甚至能載入這座城市史冊的凡人與領主之戰。在那一抹如血的殘陽與暗紅色的雨幕下。正式拉開了那壯麗且慘烈的、決死衝鋒的序幕。
