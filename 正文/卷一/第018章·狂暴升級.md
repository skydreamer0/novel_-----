# 第018章 · 狂暴升級

雨。越下越大。此刻的天空像是被某種巨大的利刃強行剖開了無數道猙獰的口子。在那漆黑且破碎的雲層深處。暗紅色的「源質雨」像是無窮無盡的、帶著怨氣的瀑布。在為這座即將陷落在異位面巨獸爪下的、平凡卻又倔強的臨海城市。提前舉辦一場最為宏大且沈悶的葬禮。

蘇晨的身形。此時正化作一道模糊、甚至帶點肉身撕裂感的殘影。在那條早已支離破碎、遍佈了金屬廢墟與焦黑瓦礫的東北區大道上。迎著那震天動地的雷鳴。瘋狂疾馳。他每一步重重地踏在那些灌滿了腐蝕性紅雨的積水坑中。都會在那驚人的爆發力下。激起數米之高。且帶著淡淡血腥味的妖異浪花。

他此時的身體狀態。正處在一個極端危險。也極端震撼、甚至是足以讓任何這方世界的生物學家感到瘋狂的「因果重組」狀界。

「噗、噗、噗、噗——！！」

這不是心跳聲。而是他全身上下的每一個毛孔。此時都在像是壞掉的高壓閥門一樣。瘋狂地向外噴射著細微、卻密集到了讓人窒息程度的深紅色血霧。那是因為在他體內。那整整五百枚高品質中品靈石。在一瞬間被強制粉碎、化作了如同恆星坍塌般的、純淨且狂暴到了極致的原始靈能海嘯。這股能量遠遠超出了他目前這具三階初級肉身所能承載的物理與生理極限。正在他的體內。進行著一場最為原始、也最為慘烈的「暴力擴容」與「基因層面的強行重組」。

「唔……嘔……啊——！！！」

蘇晨喉嚨深處發出一聲低沈、壓抑。卻透著一股子不服輸狠勁的悶吼。
他感覺。像是此刻正有幾千根。甚至是幾萬根被燒得紅透、且帶電的鋼針。正順著他的脊髓。順著他的每一根毛細血管。在一寸。一毫。一微米地。在瘋狂地。瘋狂地向內鑽探。要把他的靈魂都給攪碎了。

而在他的識海最深處。系統那原本冷靜、清脆的提示女音。在此刻。也早已變得忽遠忽近。忽左忽右。帶著一種強烈的電子信號被異位面干擾後的滋滋扭曲感。

```
  【升級權限——瘋狂解析中：當前進度 97.4%... 98.9%...】
  - 核能級預警：宿主全身上下 206 塊骨骼。其鈣質密度已達到正常人類極限之 450%！
  - 檢測到異常：脊椎骨正在發生不可逆之『晶質化』轉向轉變。
  - 當前極限心率：275 次/分！血氧飽和度已正式歸零——系統已強制切換至[本源因果流供能]。
```

就在那如同靈魂被生生撕碎成千萬片零散碎屑的劇痛。到達了某個臨界飽和點的那一微秒。蘇晨。猛地、以一種如同惡狼般的狠辣。抬起了那張早已鮮血淋漓的臉。

迎面而來的。是一頭雙翼展開足有十二米寬、渾身上下長滿了倒鉤狀、且每一根都閃爍著源質幽光的「高空獵隼」。這頭孽種怪物顯然是被蘇晨身上。那股因為暴力突破而溢散出的、濃郁到了極致且夾雜著頂級洗髓丹香氣的血肉精華。給徹底吸引了所有的本能食慾。
牠發出一聲刺耳的尖嘯。正張開那對足以輕易切開坦克正面裝甲板的、充滿了死氣的金屬利爪。帶著刺破音障的淒厲咆哮。直取蘇晨的咽喉要害！

「區區一隻被污穢氣息餵肥的。沒腦子的雜毛畜生。也特麼。也敢來攔我蘇某人的快遞路？！」

蘇晨眼神在那一瞬。徹底化作了兩道長達寸許的、代表了極致憤怒、劫火與毀滅劍意的。銀白色刺眼精光。

他沒有躲。甚至在那極限的速度中。連哪怕一丁點停頓或減速的意思都沒有。
他的右手。食指與中指並攏如刀。在那怪物的利爪距離他那脆弱的眼球前。不到零點零一公分的那百分之一秒內。以一種超越了這片世界時間邏輯的頻率。自下而上地。在那狂風中。輕輕。卻又重若千鈞地一划。

「歸墟——第一極意：瞬殺之界。」

「噗嗤——！」

場中。並沒有發生預期中的那種華麗、爆炸、或者是驚天動地的靈能對撞。
有的。只有一種極致到了極點、純粹到了極點的——物理學與靈能玄學混合後的。那種對物質結構的毀滅性橫向切割。

那頭剛才還不可一世、甚至在天衡局數據中被標記為『小區毀滅級』的高空獵隼。在此刻。甚至連那一抹代表了恐懼的哀鳴都來不及發出。牠那碩大的、覆蓋著足有三公分厚度源質裝甲羽毛的強橫軀體。竟然就那樣在那一道一閃而逝的黑色細微劍氣中。整整齊齊地。毫無滯礙地。被從鼻樑中線。一分為二。鮮紅、熾熱的妖血。在那漫天的血雨雨幕中。綻放成了一朵淒美、卻又帶著濃厚死亡氣息的超巨型。血色薔薇花朵。隨即在大風中。化作了無數瓣破碎的殘英。

蘇晨。連正眼看一眼那具正在身後打著旋墜落的兩半殘屍。都沒有興趣。

「給老子……破啊——！！」

他的身體內部。此時此刻。竟然傳出了一聲。如此清晰、如此低沉。如同某種從太古時期就流傳下來的神聖封印。在這一秒鐘。被強行用重錘敲碎的。那種晶體崩裂之音。

「轟——！！！」

在那一瞬。他腳下那原本堅硬耐磨的。特種柏油馬路。以他為中心點。像是被一顆攜帶著核聚變威能的重磅炸彈。正面命中了一般。在一微秒內。瞬間向下塌陷出了一個方圓十米、深達兩米的焦黑巨型下陷坑洞。無數猙獰的、如蜘蛛網般的裂痕。瘋狂地向著四周每一棟建築物的根基蔓延而去。

一股呈現象牙白色、中心處卻透著一股令人靈魂都要為之冰凍、感到無盡寂滅意境的。狂暴靈壓柱。宛如一道從地獄深處衝上的平地驚雷。在臨海市東北區那慘紅色的天空下。正式、狂暴且不容置疑地。徹底炸響！

三階巔峰……正式。達成！！

甚至。在蘇晨那充滿了歸墟劍意的感官中。他隱約。已經觸碰到了那層傳說中。連秦老奮鬥了整整三十年、也才剛剛摸到的。那個代表了『超凡入勝』的。四階超脫位門檻。

蘇晨感受著此刻體內。那股宛如脫胎換骨、如江河決堤後的沈穩平靜。卻變得更加凝實、更加厚重。隱約中帶著某種位面法則味道的「歸墟命脈劍氣」。他這才緩緩地、顫抖著。吐出了一口夾雜著內臟碎片的、溫度高得驚人的廢氣。隨後。他那隻滿是傷痕的手。重重地抹了一把臉上混合了那妖異紅雨的熱汗。

此時的這座城市。早已徹底失去了它那原本雖然嘈雜、卻安穩祥和的現代化外殼。
遠處。在那被鮮血染紅的視線盡頭。那頭身披著足以焚毀維度、焚燒靈魂之青色地獄火的。正將三棟摩天大樓當成自家棲息枝椏隨意踩碎的——「焚天青鸞」。也在此刻。在那敏銳到了極點的領主感知中。感應到了這股來自那「蟻穴」深處的、渺小。卻足以讓牠感到一絲威脅。敢於當面挑戰牠那至高無上之天威的。卑微氣息。

牠那對足有磨盤大小、正散發著令人心顫的熔岩光澤的赤色妖瞳。在此時。帶著一種像是發現了有趣獵物般的、那種殘忍與毀滅交織的冷酷玩味。緩緩轉動。死死地。跨越了數千米的空間。鎖定在了那個正孤零零地站在街道塌陷坑正中心、身軀顯得極其單薄、卻透著一股能刺破苍穹之銳氣的——紅衣少年身上。

「蘇晨……你……你這小混蛋……居然……居然在這種等級的空間震盪下。還沒死？！」
林若溪此時。正帶著幾名渾身是傷、靈能幾乎乾涸的調查員部下。狼狽不堪地從一輛被巨大風壓給掀翻的裝甲運鈔車後。有些艱難地探出頭。

當她那雙平日裡高不可攀、此時卻充滿了疲憊與驚懼的眸子。看清楚了那個。在前方火光與雷暴交織的廢墟中心。那個正緩緩站直身體。渾身散發著一股陌生、強悍。且透著一股子讓她這個二階強者都感到腿軟之寂滅殺氣的。那個小快遞員少年的時候。林若溪那一向冷傲、自負天才的精緻臉龐上。第一次露出了如見了鬼魅般、無法用語言去形容的。那種極度驚駭與崩潰的表情。

「林隊長。聽好了。帶著你的人。立刻去保護後方那幾個安置點的平民撤離。這裡……這頭自詡神鳥的、實則只是隻大一點的雜毛大鳥。」
蘇晨嘴角緩緩勾起一抹瘋狂。他從儲物格中。不緊不慢地。再次取出了一張通體冰藍色。散發著足以在一瞬間。將這方圓十里地的大地。徹底冰封、凍結成萬年冰川的高階古老符籙。

「這一件派送費極高的貨。就交給我這家『萬界加急物流』的負責人……親自去辦理。這最後的。血肉簽收手續好了。」

蘇晨手中緊握著的。是剛從萬界商城的清倉專場中。用光了所有剩餘這輩子信譽點。才勉強兌換出來的。傳說中曾一擊鎮壓過遠古汪洋巨獸的——禁忌之物：【SSS 級·極寒冰蛟隕落符】。

一場真正具備了神話悲劇色彩。甚至足以在幾百年後。依然被這座城市的幸存者們。在那酒館中反覆傳頌、歌咒的。凡人對神靈、蝼蚁逆伐蒼天的。那場壯麗、慘烈至極的、決死衝鋒的一戰。在那一抹如鮮血般艷麗的殘阳。與暗紅色的絕望雨幕下。正式。全面。揭開了它那不見血、便不回頭的血色帷幕。
