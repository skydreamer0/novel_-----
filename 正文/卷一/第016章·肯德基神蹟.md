# 第十六章 · 肯德基神蹟

深夜，臨海第一醫院 402 單人病房。

蘇晨躺在病床上，聽著門外巡邏護士漸遠的腳步聲，熟練地在被窩裡開啟了系統光幕。

「系統，我卡裡還有六萬多塊。現在能開啟那個傳說中的『萬界拍賣實行版』嗎？」

```
  【萬界拍賣功能解鎖中...】
  ─────────────
  消耗積分：500
  當前等級：Lv.2
  狀態：可啟動（限時拍賣，持續 30 分鐘）
  ─────────────
```

「啟動！」

蘇晨眼神發亮。這是系統升級後他一直沒敢亂動的底牌。

就在光幕閃爍的一瞬，數百個奇形怪狀的窗口在蘇晨識海中轟然炸開。

獸皇大陸的妖王、符籙仙域的長老、甚至還有渾身機械構件的廢土機師……這小小的病房，在此刻彷彿成了萬界交匯的中心。

「諸位，在下臨海交易商，初來乍到，沒什麼好送的，先給大家上一道『開胃菜』。」

蘇晨說著，輕輕點擊了下午回醫院途中偷偷去買的那份——【全家桶·豪華增量版】。

而且，他還特意讓服務員多加了兩份香辣沾醬和三包胡椒粉。

「此乃：九天神靈脆皮金鳳。經過三十六道神火淬煉，外焦裡嫩，更佐以『魔都秘制黑水』（可樂），食之可讓人身心通透，戰意倍增！」

蘇晨將金黃酥脆的炸雞高高舉起，在全息投影的加持下，那色澤竟然比異世界的聖藥還要奪目。

拍賣窗內。

「這……這金黃的色澤，難道是傳說中的三足金烏幼體？」一位修為深不可測的火系大能眼珠子都快突出來了。

「快看那黑色的靈泉！居然還在冒著神祕的氣泡……嘶，難道是修補神魂的『九幽幽泉』？」

「那刺鼻卻讓人精神一振的香氣……老夫修煉千年，竟從未聞過如此特殊的藥香！」

一時間，萬界頻道炸開了鍋。這群平時高高在上的強者，哪見過裹著麵粉炸過的面衣？哪喝過含糖量超標的碳酸飲料？

```
  【拍賣火熱進行中！】
  ─────────────
  - 出價 1：符籙仙域·白符師，出價『隱形符*50』。
  - 出價 2：妖獸大陸·赤炎虎王，出價『三階火晶*10』。
  - 出價 3：靈丹界·九轉藥老，出價『三階·洗髓丹*1』。
  ─────────────
```

蘇晨的呼吸變得有些急促。三階洗髓丹？那可是能讓二階覺醒者直接洗去全身雜質，提前觸摸三階門檻的神藥！

「藥老，你那洗髓丹，藥效如何？」蘇晨強忍激動，語氣平穩地詢問。

「小友放心。我這『天靈洗髓丹』，乃是用三十二株玄階靈藥煉製而成，凡人服之可易經伐髓，若你已經到了突破邊緣，此丹助你一步跨入三階，絕無副作用！」藥老的聲音中充滿了急切，「那九天金鳳……快與老夫交易！老夫已經嗅到了突破的契機！」

（蘇晨內心嘀咕：你那是嗅到了香料的契機吧……）

「成交！」

蘇晨直接點擊確認。

白光閃過，一份全家桶消失在虛空中，取而代之的，是一顆龍眼大小、散發著淡淡青色氤氳、其上刻有三道丹紋的璀璨丹藥。

這就是連秦老那個層次的人看到都會眼紅的——三階靈丹。

「系統，這藥……我現在能吃嗎？」

```
  【分析結果】
  建議配合《長青吐納術》服用。
  當前環境安全性：中下。建議配合解析偵查，清除潛伏目標後再行突破。
```

蘇晨收起丹藥，眼神變得凌厲起來。

他走出病床，手指輕輕在指間的「隱氣環」上轉了三圈。

「既然裝備齊了，先去地下三層看看那個『內應』到底是誰。」

他穿好外套，悄無聲息地推開病房門。在解析偵查的視野下，漆黑的走廊上散佈著一條條若隱若現的紫色絲線，那些絲線的源頭，無一例外指向了負三樓的停屍間方向。

「大半夜引我去停屍間？這劇情，還真是夠老套的。」

蘇晨冷笑一聲，手中不知何時已經握住了那柄換來的中品靈石長劍（已用白布包裹）。

他沒有坐電梯，而是像一抹黑色的幽靈，直接翻進了樓梯間的通風管道。

與此同時。

地下三層。

那尊穿著白大褂、正對著幾具「失蹤者」屍體進行縫合的「主刀醫生」，緩緩停下了手中的動作。他轉過頭，那雙瞳孔竟然橫向裂開，透著一股不屬於人類的妖異紫光。

「聞到了……是鮮活的、充滿了變革氣息的味道……」

他嘴角咧開一個誇張的角度，露出了滿嘴細密的尖牙。

「蘇晨……歡迎來到……進化的現場。」

---

*章末提示：蘇晨輕巧落地。在他面前，並非陰冷的停屍間，而是一個被擴張了數倍的空間——那裡漂浮著數以百計的透明器皿，裡面浸泡著的，全都是那些原本應該出現在「失蹤名單」上的覺醒者。而在最中央的水槽裡，一個巨大的、像心臟般律動著的紫色肉瘤，正在緩緩跳動。*
