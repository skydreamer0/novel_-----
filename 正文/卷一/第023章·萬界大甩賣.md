# 第二十三章 · 萬界大甩賣

「系統，開啟因果置換模式。這一次，我豁出去了！」

蘇晨站在趙記雜貨店的地底密室中，四面牆壁上掛滿了鏽跡斑斑的覺醒者裝備。外面的雷聲隱約傳入這深達三十米的地底，讓他的一顆心跳動得極快。

```
  【萬界拍賣行：特別緊急專場啟動】
  ─────────────
  當前連線：位面 142 個
  宿主籌碼：中品靈石*500、神廟殘片數據源、蘇晨的『快遞商』信譽值。
  需求：禁忌級大規模殺傷性武器。
  ─────────────
```

光幕上，無數貪婪或威嚴的身影在閃動。

「我要那種能瞬間氣化三階領主、重創四階位面保護層的東西！」蘇晨的聲音在識海中嘶吼，「誰有？代價好談！」

「嘿嘿，小友。我這兒有一顆『核能壓縮彈』，不知你那靈石換不換得起？」一位來自荒廢機甲位面的機械維修師發出陰冷的笑聲。

「不夠！那東西只能物理破壞，我要連靈魂和源質都能抹除的！」蘇晨拒絕。

「老夫這有一枚『寂滅死光符』。但這代價……」符籙仙域的一位紅袍長老眼神中透著一抹玩味，「老夫要你手中那塊青銅殘片的『神廟拓印圖』。」

蘇晨心中一震。原來這殘片在萬界眼中如此值錢！

「給！但你要給我兩枚！」

「成交！」

就在交易即將達成的順間。

「且慢。」

一聲清脆如玉、卻帶著無盡劍意的聲音，強行切斷了所有人的通訊。

劍修世界，葉孤楓。

他的影像出現在光幕中心。此時的他，衣衫有些凌亂，甚至胸口還帶著血跡，但他背後卻懸浮著一根散發著令人戰慄氣息的、長達十米的巨大銀色金屬棒。

那是他從劍修世界某個「上古科技遺蹟」中拼盡全力搶出來的。

「蘇兄，你救我兩次，這份人情，我葉孤楓還你。」

葉孤楓眼神堅定，指著那根金屬棒，「此物名為『上帝之杖·偽』。乃是我們位面曾有過的一段『科技修仙時代』留下的巔峰之作。它利用萬米高空的隕落動能搭配雷霆意境，足以在瞬間摧毀一座方圓百里的山脈。它不需要靈氣驅動，只需要定位。」

「這代價……」

「無需代價。」葉孤楓灑然一笑，「若你死了，誰還能給我提供那好喝的『仙露』和『聚靈麵』？蘇兄，活下去。」

```
  【交易成功】
  ─────────────
  您獲得：『上帝之杖·偽』（一次性定向軌道武器）
  您獲得：靈基鎖定裝置*1
  ─────────────
```

蘇晨手中沉甸甸的，出現了一個充滿賽博設計感的紅點激光器。

他深吸一口氣，睜開眼。

此時的老趙，正坐在一台滿是灰塵的通訊器前，手指瘋狂顫動。

「蘇晨，快！蛻變者的大軍已經進入蛇巷外圍了！天衡局那邊出了亂子，秦老被幾名四階潛伏者牽制住了，林若溪快撐不住了！」

「趙叔，幫我定位醫院地下三層的神廟核心座標。」

蘇晨眼神如刀，手中的紅點激光器散發著淡淡的微光，「我要邀請他們去見上帝。」

老趙愣了一下，隨即瘋狂操作。

「座標：東經 121.5，北緯 31.2。地下深度：150 米。那是祭壇的核心！」

「好。」

蘇晨轉身走上螺旋階梯。在他的身後，系統光幕發出了最後的警報。

```
  【升級提示：功能全開！】
  系統升級進度：100%。
  蘇晨身分變更：[萬界初級代理人]。
  解鎖權限：本體跨界傳送（每次消耗 1000 中品靈石）。
```

蘇晨推開雜貨店的大門。

暴雨中。

原本整潔的街道此時已經成了殘肢斷臂的屠宰場。林若溪正半蹲在垃圾桶後，手中的冰劍已經破碎了一半，她渾身是血，幾名身穿紫色機甲的「蛻變者」正獰笑著靠近她。

「蘇晨……快跑……」林若溪看見出口處的少年，虛弱地喊道。

蘇晨沒有跑。

他緩緩抬起右手，將那枚紅點激光器按下了開關。

一道細得幾乎不可見的紅線，在暴雨中穿過重重黑幕，精準地點在了臨海第一醫院那高聳的避雷針上。

「定位成功。」

蘇晨的聲音，在那一刻彷彿與天上的雷鳴合而為一。

「諸位。準備好，迎接天譴了嗎？」

天際。

原本厚重的雲層在那一瞬被一股強大到無法想像的慣性力量生生撕碎。一道長達十米、燃燒著劇烈白光、重達數噸的鎢金棒，猶如神祇投下的長矛，帶著摧毀一切的動能，朝著地表俯衝而下。

那一秒。

臨海市所有的電子儀器全部瞬間失靈。

唯有蘇晨，安靜地站在暴雨中，替受傷的林若溪擋住了那狂暴的風壓。
