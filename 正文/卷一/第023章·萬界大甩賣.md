# 第二十三章 · 萬界大甩賣

「系統，立刻強制開啟因果置換模式。這一次，哪怕是透支我未來三年的壽命，我也要豁出去了！」

蘇晨隻身站在趙記雜貨店那隱秘且壓抑的地底密室中。四周的石牆上掛滿了鏽跡斑斑、卻依然散發著微弱靈光的覺醒者舊式裝備。外面的雷聲透過厚厚的土層隱約傳入這深達三十米的地底，讓他那顆原本就緊繃的心臟跳動得前所欲有的快，像是要撞破胸膛。

```
  【萬界拍賣行：S級特別緊急避難專場——啟動成功！】
  ─────────────
  當前穩定連線位面：142 個（正在持續激增中...）
  宿主可提供籌碼：中品靈石*500、神廟殘片核心拓印數據、蘇晨的『跨界快遞商』信譽評級。
  核心需求：禁忌級別、針對亞空間祭壇的大規模殺傷性武器。
  ─────────────
```

光幕上，無數道或貪婪、或威嚴、或充滿了好奇的神祕身影在瘋狂閃動。這是一個瘋狂的維度交匯點，一個由數據和靈魂鑄造的黑市。

「我要那種能在一秒鐘內徹底氣化三階巔峰領主、甚至是強行重創四階位面保護層的高能殺器！」蘇晨的聲音在識海中低沉地嘶吼著，像是被逼到絕路的老虎，「誰有存貨？代價好談，哪怕是透支我這家公司的信用權限！」

「嘿嘿，小友，火氣別這麼大。」
一位來自荒廢機甲位面、半個身子都改裝成了精密液壓件的機械維修師，發出一聲令人極度不適的陰冷刺耳笑聲：「我這兒有一顆剛從反應堆裡拆出來的『反物質壓縮坍縮彈』，不知你那點可憐的靈石換不換得起？它能讓整條街區瞬間坍縮成一個微型黑洞。」

「不夠！那東西雖然物理破壞力強，但那是針對物質層面的。我要那種連靈魂、連異界邪神的源質汙染都能一併抹除的秩序武器！」蘇晨毫不猶豫地拒絕了。

「老夫這有一枚收藏了三百年的『大乘期·寂滅死光符』。但這份代價……」
符籙仙域的一位身披紅袍、眉心有一顆豎瞳的長老，眼神中透著一抹老辣的玩味：「老夫不要靈石，老夫要你手中那塊青銅殘片的『底層神廟數據拓印圖』。那東西，值這個價。」

蘇晨心中猛地一震。原來這塊在他手裡還在發熱的殘片，在萬界大佬眼中竟然如此具備戰略價值！

「給！但一枚不夠，你要給我三枚那種級別的符籙，我需要保險。」

就在這樁足以改變位面格局的交易即將達成的最後一秒。

「且慢。」

一聲清脆如玉、卻帶著一種能切開虛空無盡劍意的聲音，在大功率通訊鏈路中強行切斷了所有人的雜音。

劍修大世界。葉孤楓。

他的主影像出現在了光幕的最中心。此時的他，那一身原本乾淨的白衫顯得極其凌厲且破碎，胸口處甚至還帶著幾道尚未乾涸的深紅色血跡，顯然是剛從一場可怕的追殺中殺出重圍。但在他的背後，此時竟懸浮著一根散發著令人靈魂顫慄的、長達十米的巨大銀色金屬棒。

那是他從劍修世界某個被遺忘的「上古科技文明遺蹟」中，拼盡了最後一絲劍氣搶出來的。

「蘇兄，你救我兩次。這份恩情，我葉孤楓哪怕是散盡修為，也得還你。」

葉孤楓眼神中透著一抹跨越時空的堅定，他指著懸浮在身後的那根充滿了冷酷對稱美的金屬棒，緩緩說道：
「此物在我們這兒的古籍中名為『上帝之杖·偽』。乃是我們位面曾有過的那段短暫卻輝煌的『科技修仙時代』所留下的巔峰戰爭遺存。它利用萬米高空的極致隕落動能，搭配上雷霆與毀滅法則的雙重意境，一旦啟動，足以在三個呼吸內徹底摧毀一座方圓百里的巍峨山脈。它不需要你的靈氣驅動，它只需要——一個座標定位。」

「這代價……葉兄你現在處境也不妙，你想要什麼？」

「無需任何代價。若你今晚死在那異界的祭壇之旁，這世上還有誰能給我提供那好喝且帶氣的『快樂仙露』和那夠味的『聚靈麵』？」
葉孤楓灑然一笑，那是劍客獨有的孤高與豪邁：「蘇兄，拿好它。然後……活下去，下次來給我帶兩桶那個叫肯德基的鳥肉。」

```
  【跨位面硬核交易成功達成！】
  ─────────────
  您失去：[快遞商信譽積分 100 點]
  您獲得：『上帝之杖·偽』（一次性超視距軌道定向武器）
  附加物：[靈基高維鎖定裝置] * 1
  描述：由上古鎢合金鑄造。無視所有已知的靈能護盾，純物理與重力能混合打擊。
  ─────────────
```

蘇晨感覺手中猛地一沉，一個充滿了賽博龐克設計感、通體漆黑且帶著紅色激光指示器的控制終端，憑空出現在了他的掌心。

他深吸了一口氣，強忍住眼眶的濕意，猛地睜開雙眼。

此時的老趙，正坐在那台落滿了厚厚灰塵、正發出滋滋電流聲的舊式通訊器前，指縫間夾著的菸頭早已燒到了指尖。

「蘇晨，快！蛻變者那幫畜生的先遣大軍已經突破了舊城區的最後防線進入蛇巷了！天衡局那邊也出了天大的亂子，秦老被幾名隱藏在內部的四階巔峰刺客纏住了根本脫不開身，林若溪在外面……快要撐到極限了！」

「趙叔，別管我了。立刻幫我精準定位臨海第一醫院地下三層那個神廟祭壇的核心座標，要精確到小數點後三位。」

蘇晨的眼神中透著一股如同修羅般的冷酷，手中的紅點激光器在昏暗的地底散發著淡淡、卻致命的微紅幽光，「我要邀請這幫自詡進化的瘋子……去見真正的上帝。」

老趙先是愣了一秒，隨即雙手化作殘影，在老舊的鍵盤上瘋狂操作。

「座標確定：東經 121.579，北緯 31.226。地下核心埋深：152.4 米。那是整座城市防禦點的死穴，亦是祭壇心臟所在！」

「謝了，趙叔。」

蘇晨豁然轉身，大步走上那昏暗的螺旋階梯。在他的身後，識海中的系統光幕此刻竟發出了最後一聲清亮、優雅且莊嚴的升級警報。

```
  【升級提示：萬界交易權限全開！】
  當前系統升級總進度：100%。
  蘇晨正式身分變更：[萬界初等代理人]。
  新增恐怖權限：[本體跨位面肉身穿梭]（權限已解鎖，消耗巨大，請慎用）。
  ─────────────
```

蘇晨猛地一把推開趙記雜貨店那扇被雨水浸透的破爛大門。

暴雨狂風中。

原本整潔安詳的小巷街道，此時已經淪為了一片布滿殘肢斷臂與紫色機甲碎片的修羅屠宰場。林若溪正半跪在一個翻倒的垃圾桶後方，她手中那柄原本晶瑩剃透的冰藍長劍此時已經崩碎了一半，她原本白皙的手臂上滿是觸目驚心的血跡，幾名身穿厚重紫色機甲、呼吸聲如野獸般的「蛻變者」刺客正帶著猙獰的笑意步步逼近。

「蘇晨……別管我……快走……把東西帶給秦老……」林若溪抬起頭，看見出口處那個單薄少年的身影，用盡最後一絲力氣虛弱地喊道。

然而蘇晨並沒有跑，甚至連腳步都沒有挪動一下。

他緩緩抬起了顫抖卻穩定的右手，對準了遠處那籠罩在紫色雷暴中的地標塔身，將那枚紅點激光器毅然決然地按下了啟動開關。

「紅線已鎖定。上帝之杖，請求降臨。」

蘇晨那冰冷的聲音，在那一刻彷彿與九天之上的驚雷徹底合而為一，震懾靈魂。

「諸位自詡進化的怪物們。現在，準備好迎接來自另一個位面的……天譴了嗎？」

萬米高空之上。

原本厚重得如同永夜般的雲層，在這一瞬被一股強大到無法用科學解釋的、極致的慣性動能力量生生撕碎開來！一道長達十米、燃燒著如同白晝般劇烈白光、重達數噸的鎢金屬圓柱，猶如憤怒的神祇投下的長矛，帶著足以摧毀這世間一切物質的毀滅力量，正以五十倍音速的絕望姿態，朝著地表俯衝、墜落。

那短短一秒。

臨海市方圓百里內的所有精密電子儀器全部在一瞬間陷入了癱瘓失靈。

唯有蘇晨，始終安靜且屹立不倒地站在這場足以抹除城市的暴雨中。他伸出那寬大的墨綠色披風，替早已力竭受傷的林若溪，擋住了那迎面而來、足以切開大地的狂暴風壓。

*【章末番外】*
*在臨海第一醫院那鮮為人知的地底深處。蛻變者主教正虔誠地跪在那座血肉祭壇前，瘋狂地吟唱著企圖喚醒沈睡魔神的禁忌咒語。*
*突然間，他感覺到了地板發出了一聲清脆到極致、甚至有些悅耳的碎裂聲。*
*他下意識地抬頭看向空無一物的石製天花板，只見一道代表死亡的純白光束從天靈蓋直衝而下，他在這一秒鐘甚至沒感覺到痛苦，因為他的大腦與整座祭壇在大氣層外的撞擊力面前，瞬間蒸發。*
*他在這世上留下的最後一句遺言竟然是：「這……這不是靈氣波動……這特麼的是……純粹的物理撞擊？！」*
