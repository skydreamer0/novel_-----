# 第二章 · 可樂換仙草

交易連結建立的瞬間，蘇晨的意識像是被拽進了一條光的隧道。

不是真的移動——他的身體還好端端地站在巷子裡。但他的感知力彷彿被無限延伸，穿過了一層看不見的薄膜，接觸到了一個完全不同的世界。

確切地說，是接觸到了一個人。

「你、你是仙人嗎？」

一個怯生生的聲音在蘇晨腦海中響起。語氣裡帶著驚恐、興奮，和大量的不確定。

蘇晨愣了一下，隨即反應過來——這應該就是系統匹配到的交易對象。

光幕上浮現出一段文字：

```
══════════════════════════
  【交易通訊已建立】
──────────────────────────
  對方：靈草世界·青禾原
  身份：藥農·李小苗
  年齡：14歲
  等級：凡人（無修為）
  
  通訊模式：意念對話
  注意：對方可感知您的語意
  但無法看見您的世界。
══════════════════════════
```

十四歲的藥農。蘇晨在心裡琢磨了一下這個設定，決定先搞清楚狀況。

「我不是仙人。」他在腦海裡回應，「我是……呃，另一個地方的人。你那邊有個交易系統嗎？」

「系統？」李小苗的聲音更加困惑了，「我、我不知道什麼系統。我只是在挖草的時候，腦子裡突然冒出一個聲音，說有人要跟我做交易……然後、然後我就聽到你說話了。」

蘇晨看了一眼光幕。上面多了一行提示：

```
  提示：交易對象僅能接收到
  交易邀請和通訊功能。
  系統本體僅對宿主可見。
```

原來如此。對方只是被系統選中的交易目標，本身並不知道系統的存在。

「別怕。」蘇晨盡量讓語氣放鬆，「就是簡單的物品交換。你那邊有什麼？」

「我、我是藥農，我有草藥……」李小苗的聲音小了下去，「不過都是最普通的靈草，值不了幾個靈幣……」

光幕上同步出現了一個清單：

```
  【李小苗·可交易物品】
  ─────────────────
  ① 清心草 ×3（品質：普通）
    功效：淨化心神，微量增強感知
  ② 止血藤 ×5（品質：普通）
    功效：止血、促進傷口癒合
  ③ 醒神花 ×1（品質：良好）
    功效：提神醒腦，少量恢復靈力
  ─────────────────
```

蘇晨的目光落在第一項上。

清心草。淨化心神，增強感知。

如果這個「感知」和系統裡提到的「覺醒者一階·感知」有關聯的話……

「那個清心草，」蘇晨問，「能讓普通人感應到靈氣嗎？」

「靈氣？」李小苗想了想，「嗯……清心草確實能讓人的五感變得敏銳。我爺爺說過，有些有天賦的凡人吃了清心草之後，偶爾能看到靈氣的流動。不過這種草太常見了，在我們這裡滿山都是，不值錢的。」

滿山都是。

不值錢。

蘇晨嘴角緩緩上揚。

「那你想要什麼？」他問。

「你、你那邊有什麼？」

蘇晨低頭看了看手裡那瓶可口可樂。紅色的包裝在夕陽下反射著光芒。

他總不能說「我有一瓶碳酸飲料」吧？人家十四歲的孩子，在一個有靈氣的世界裡挖草，怎麼可能知道可口可樂是什麼。

但系統似乎比他想得周到。

光幕上出現了一行新的提示：

```
  提示：系統將自動轉化物品描述
  以匹配交易對象的認知框架。
  
  「可口可樂」→ 轉化描述：
  「來自異界的甘甜神液，
   入口清涼，氣泡如靈泉，
   飲之令人心曠神怡。」
  
  對方感知評級：★★★☆☆
```

三星。一瓶三塊錢的可樂在異世界居然有三星的評價。

「我這裡有一瓶異界神液。」蘇晨忍住笑說道，「味道很好，是我們這邊的珍品。」

嗯，從某種意義上來說，可口可樂確實是人類文明的珍品。

李小苗的聲音裡充滿了嚮往：「異界神液……聽起來好厲害……可是、可是我只有這些普通的草藥，會不會配不上？」

蘇晨看了一眼光幕上的交易建議：

```
══════════════════════════
  【系統建議交易方案】
──────────────────────────
  您提供：碳酸飲料（可口可樂）×1
  對方提供：清心草 ×3
  
  交易公平度：■■■■□（偏向宿主）
  
  備註：此交易對宿主有利。
  清心草在靈草世界雖為常見草藥
  但在宿主所在世界具有
  極高的修煉輔助價值。
══════════════════════════
```

偏向宿主。

也就是說，他佔便宜了。

蘇晨深吸一口氣。一方面，他確實需要這些靈草。另一方面，他不想佔一個十四歲孩子的便宜。

「三株清心草換一瓶神液。」他說，「但下次如果我再找你交易，你可以提高價格。畢竟我們這是長期合作，對吧？」

「長、長期合作？」李小苗的聲音一下子亮了起來，「你是說，以後還會找我？」

「當然。」

「太好了！」小女孩的聲音裡帶著純粹的開心，「那、那我們成交！我一定給你留最好的清心草！」

蘇晨正要確認交易，忽然想到了什麼。

「小苗，」他問，「你們那邊……是什麼樣的世界？」

李小苗奖嘿了一下：「呀？就是……就是很普通啊。我們青禾原是個小村子，周圍都是山。山上有靈草、也有妖獸。我和爪爪就靠採藥過日子。别的……别的沒什麼特別的。」

有妖獸。

蘇晨在心裡把這個信息記了下來。

「危險嗎？」

「山上的小妖獸還好，我們都會躺。但深山裡的大妖獸就很可怕了，上次有一隻三級的雷牙虎下山，把陰壁村的半條街都拆了……」李小苗的聲音小了下去，「還好閞門的修士來得快，不然的話……」

修士。

有修士的世界。有妖獸的世界。有靈草滿山的世界。

而那個世界裡滿山都是、不值錢的清心草，在他這個世界危危明明能讓普通人覓變為覆醒者。

這就是信息差。

這就是他的機會。

「好，我們成交。」蘇晨在光幕上確認了交易。

光幕閃爍。

```
══════════════════════════
  【交易完成】
──────────────────────────
  已送出：可口可樂 ×1
  已獲得：清心草 ×3
  
  交易積分 +10
  今日剩餘交易次數：2/3
  
  ！首次交易完成！
  ！系統正式綁定宿主：蘇晨！
  ！解鎖功能：物品鑑定（基礎）！
══════════════════════════
```

蘇晨手裡的可樂消失了。

取而代之的是三株翠綠色的小草。

它們靜靜地躺在他的掌心裡，每一片葉子都散發著淡淡的瑩光。微弱的，但確實存在的光芒。如果不仔細看，會以為是陽光的反射。

但蘇晨知道那不是陽光。

因為他能「感覺」到。

那三株草散發出的某種無形的波動——溫和的、清澈的、像是山間清泉一樣的東西——正在輕輕地撫過他的感知。

「這就是……靈氣？」

他沒有猶豫太久。

按照系統的物品說明，清心草可以直接服用。蘇晨拿起一株，放進嘴裡，嚼了兩下。

味道出乎意料地好。清甜中帶著一絲薄荷般的涼意，入口即化，像是一股清流從舌尖一路滑到了胃裡。

然後——

那股清流炸開了。

不是疼痛。是一種難以描述的「打開」的感覺。就好像他之前一直戴著一副厚重的墨鏡，而現在，墨鏡被摘掉了。

世界變了。

不，世界沒變。是他看世界的方式變了。

他能感覺到空氣中漂浮著某種細微的、肉眼不可見的粒子。它們無處不在——在風裡、在牆壁的縫隙裡、在地底深處。大部分地方很稀薄，但在某些方向，它們明顯更加濃郁。

不僅如此。他的五感都在急劇放大。

巷子盡頭那家小吃店的油煙裡混著什麼調料，他現在能分辨出來——花椒、八角、還有一絲微乎其微的桂皮。三百米外有個老人在咳嗽，咳嗽的節奏暗示著慢性支氣管炎。更遠處的街道上，某輛車的引擎聲帶著異響，左前輪的軸承可能要壞了。

這些信息像潮水一樣湧進他的腦海。不是雜亂的，而是有序的、分層的。他的大腦彷彿被重新格式化了，處理信息的能力提升了一個數量級。

而在所有這些感知之上，最清晰的，是那些無處不在的粒子。

「靈氣。」蘇晨喃喃道。

他真的能感覺到了。

光幕再次閃爍：

```
══════════════════════════
  【狀態更新】
──────────────────────────
  宿主已服用：清心草 ×1
  效果：感知力大幅提升
  
  ！恭喜宿主覺醒！
  ────────────────
  境界：一階·感知
  靈技：未覺醒
  體質：普通
  
  距離下一階（凝氣）：
  ████░░░░░░░░░ 15%
══════════════════════════
```

一階·感知。

蘇晨站在那條破舊的小巷裡，握著剩下的兩株清心草。

他是個覺醒者了。

用一瓶三塊錢的可樂。

他忍不住笑出了聲。從小聲的笑變成放聲大笑，笑到眼淚都快出來了。二十三年了，頭一次覺得老天爺沒有完全忘記他。

笑完之後，他做了一個測試。

他捏了捏拳頭，朝旁邊的磚牆輕輕打了一拳。

「咕」的一聲，沒用太大力氣，但磚牆上留下了一個清晰的拳印。灰層剝落，磚面出現了一條細小的裂紋。

蘇晨看了看自己的拳頭。一點都不疼。

這要是以前，他必定會把指骨節揍腫。但現在，他只是稍微一用力，就在磚牆上留了印子。

一階的體質強化，已經遠超普通人了。

而這僅僅是一株清心草的效果。他手裡還有兩株。跟李小苗的交易通道也已經建立——明天系統重置交易次數後，他可以繼續交換更多的靈草。

一瓶可樂換三株清心草。一株清心草就足以讓普通人覺醒。

這筆帳怎麼算都是暴利。

他小心翼翼地把剩下的兩株清心草用工作服的口袋包好——這是他目前最值錢的東西，比他半個月的工資都值錢。

然後他扶起倒在地上的電動車，把散落的快遞箱重新裝好。

快遞還是要繼續送的。

但生活，已經徹底不一樣了。

他重新跨上電動車，發動引擎。在駛出巷子的那一刻，他的感知力忽然捕捉到了什麼。

在城市的東北方向——大約三四公里遠的地方——有一股異常強烈的靈氣波動。

那種波動不像是自然存在的。它是尖銳的、不安的，帶著某種危險的氣息。

蘇晨的笑容慢慢收斂。

他看向東北方向。那裡是舊城區和新城區的交界地帶——一片正在拆遷的廢棄工廠區。

「那是什麼？」

他不知道答案。

但他的直覺告訴他，最好暫時不要靠近。

至少，在他變得更強之前。

他扭了扭手腕上被地面磨破的傷口。已經不疼了。

甚至——他低頭看了一眼——傷口似乎在以肉眼可見的速度癒合？

「感知境的覺醒者，連自愈能力都變強了嗎……」

他嘀咕了一句，把目光從傷口上移開，看向了手機上的訂單列表。

還有三單沒送。

電動車在夕陽下拖著一道長長的影子，消失在舊城區的巷尾。

而在他後方，那個廢棄工廠的方向，一聲沉悶、非人的嘶吼再次震顫了空氣。

在那片廢墟的邊緣，一個穿著白裙、手持冰藍色特製終端的年輕女人正靜靜佇立。她看了一眼終端上劇烈波動的數據，又看向蘇晨離去的方向，墨鏡後的雙眼微微瞇起。

「竟然有人先到了？」

女人的聲音清冷如雪，在寂靜的荒地裡傳出很遠。
