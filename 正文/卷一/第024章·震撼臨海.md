# 第二十四章 · 震撼臨海

那一刻，整座臨海市及其周邊幾百公里內的所有生靈，都真真切切地以為世界末日在那一秒鐘徹底降臨了。

「轟隆隆——！！！咚！！」

那並非任何雷鳴或爆炸所能發出的聲音，而是整片厚重的大地被某種來自外太空的、蠻橫到極點的力量生生貫穿、瘋狂攪動時發出的悲慘鳴叫。

那道長達十米的鎢金金屬棒，在萬米高空向下墜落的極短几秒鐘內，將動能加速到了恐怖的數十倍音速。它與空氣劇烈摩擦產生的白熾高熱，將漆黑的雨夜映照得如同烈日當空。當它精準命中第一醫院廢墟地心的那一瞬，所產生的恐怖衝擊波，直接將方圓兩公里內的所有摩天大樓玻璃窗震成了亮眼的晶瑩粉末，無數豪車的報警器在同一時間瘋狂尖叫，編織成了一首死亡的交響樂。

原本被粘稠如墨、足以干擾覺醒者內息的紫色源質霧氣死死籠罩的天衡局分局大門，在此刻竟然奇蹟般地恢復了清明。因為那股足以鎮壓跨位面靈氣的、純粹到極致的物理衝擊力，直接以暴力美學粉碎了空中漂浮的所有妖異源質顆粒，並將它們瞬間氣化蒸發。

蘇晨安靜地站在蛇巷那個陰暗狹窄的出入口，他感覺自己的五官在那一瞬間徹底失去了所有的聽覺。耳脈中唯剩下一片震耳欲聾的嗡鳴。

而在他寬大的懷裡，林若溪正指節發白、死死抓著他那件有些破爛的披風領口。那雙向來平靜如千年寒潭的清冷眸子中，此時正劇烈收縮倒映著遠處那根直插地心、久久不散的「白色死亡光柱」。

「那到底……是什麼層次的禁咒？是林家總部的支援嗎？」林若溪的聲音微弱得不可聞，整個人都在劇烈顫抖。

「沒什麼，一份跨位面的加急快遞，還是那種一旦簽收概不退換的特快件。」
蘇晨有些吃力地吐出一口濁氣。他能感覺到，體內的「歸墟劍意」因為近距離感應到了如此霸道、如此絕對的力量衝擊，竟然也隨之產生了某種前所欲有的興奮共鳴，原本的三階瓶頸似乎在那聲音中變得愈發鬆動。

```
  【上帝之杖·偽：打擊數據實時回饋中...】
  ─────────────
  打擊目標：[蛻變者]地下大祭壇、主教級生體內存、核心護衛隊（全滅）。
  掃描結果：該區域所有非法位面頻率已徹底抹除（達成率 100%）。
  源質精元高速回收中... 獲取[高階源質液] * 500 單位。
  系統經驗條暴漲提示：主升級進度額外補正 30%。
  ─────────────
```

「老趙，你看我這趟差送得到不到位？這五星好評，你可得幫我給總部那邊敲定了。」

老趙此時已經有些虛脫地癱坐在收銀台後的破地板上。他看著窗外遠方那朵冉冉升起、在暴雨中久久不散的、帶著藍白色電弧的「物理蘑菇雲」，笑得眼角的皺紋都擠在了一起。眼淚順著老臉流了下來：「行……真他娘的行。蘇晨，你這小王八羔子……今晚真的是讓老夫活了這輩子，開了最大的眼界了。」

幾分鐘後，天色依舊陰沈。

天衡局那幾部塗裝著特殊標誌的靈能垂直起降直升機，冒著殘餘的能量亂流低空掠過廢墟。秦老那道微駝的身影從百米半空中毫無防護地一躍而下。他那一身原本考究的灰色中山裝早已在刚到的那場激戰中破碎不堪，右臂還掛著幾道深可見骨的爪傷，顯然他在那幾名四階潛伏者的圍攻下，也拼到了底牌盡出的地步。

但當這位老辣的局長親眼看到，那被徹底夷為平地、甚至連地基都被氣化成了琉璃坑的醫院廢墟，以及那個正站在蛇巷口、手裡還百無聊賴握著一個奇怪外掛激光器的紅衣少年時，秦老爺子的神情從最初的極度驚愕，慢慢變成了某種深沉的欣慰，最後化作了一抹不容置疑、帶有最高保護意味的防備。

「小蘇晨。那驚天動地的『一棒子』，是你的那所謂的『祕密渠道』搞出來的動靜？」秦老龍行虎步地走到兩人面前，眼神如刀鋒般犀利地掃過蘇晨那張看似憨厚的老實臉。

「秦老，我一直都說，我就是個本分的萬界快遞商，不代購，不退貨。」
蘇晨面不改色，緩緩將那枚黑色的激光定位終端收進了儲物空間，「剛才正好我手裡有一份來自異位面老朋友的『特別訂購樣品』，本來是打算留著拆那種萬年玄冰包裝盒用的。沒想到今晚那幫瘋子吵得我頭疼，我就順手拿出來幫他們『物理降噪』了一下。」

秦老盯著蘇晨看了足足十秒鐘，隨即他那張嚴肅的臉龐舒展開來，轉頭看向林若溪，語氣威嚴得不容置疑：
「傳我手令。將蘇晨的所有保密檔案，從此刻起列入國家 SSS 級最頂級加密序列。對外通稿，就宣稱是天衡局總部最研發的新型『大氣層軌道打擊試驗』發生了局部座標偏移誤射，造成了遺憾的公共設施損毀。」

「是，局長！我這就派人著手封鎖消息鏈。」林若溪敬了一個標準得有些生硬的軍禮，看向蘇晨的眼神中透著一抹連她自己都沒察覺到的崇拜與複雜。

……

當晚。

臨海市在經歷了半夜的血與火後，終於再次迎來了久違的、甚至有些病態的寧靜。但所有幸存下來的高層都知道，這份安寧是建立在一個橫行百年的豪門徹底消失、以及一座三甲醫院被強行從地圖上抹除的血淚之上。

周家，正式宣佈在臨海市除名，所有名下的財產與靈能資產被天衡局全盤接管查封。

而作為今晚真正的幕後功臣，蘇晨卻在那份熱鬧且嘈雜的授勳邀請前，謝絕了所有入職「總部特派員」的提議。他一個人穿著那件有些破洞的快遞短袖，晃晃悠悠地回到了老趙那間依舊燈火昏暗的破舊雜貨店。

地底下，那間最深的秘密密室內。

「老趙，您之前說那塊殘片是誘餌發射器，那那位沈睡在我們地底深處的、所謂的魔神頭顱……它現在還能動彈嗎？」

老趙此時正坐在泛黃的燈光下，細心地擦拭著自己那一支曾經立下赫赫戰功的小口徑舊手槍。他的語氣顯得很輕鬆，甚至帶著幾分劫後餘生的調侃：
「都被你那一根重達幾噸的鎢金大棒子，從一萬米高空直接砸透了天靈蓋，你覺得它還能好到哪兒去？那魔神殘體本來就是靠著枯竭的地下靈脈在苟延殘喘、企圖借屍還魂。你那一頓物理操作，靈脈在那一秒鐘徹底崩斷，這老怪物也該徹底在歷史的塵埃裡消停下去了。」

蘇晨這才長長地鬆了一口氣。

但他心裡清楚，這件事絕對沒有那麼簡單就畫下休止符。

「系統，我現在身為代理人的權限，能實時觀測到葉孤楓那邊劍修世界的情況嗎？」

```
  【跨界全景監視窗口——啟動中...】
  - 由於宿主之前施展了因果置換，葉孤楓位面的時間流速目前為[異常加速狀態]。
  - 實時畫面：葉孤楓正一人一劍，右手持著那把改裝電擊槍，左手握著歸墟殘劍，傲然立於敵對宗門『紫霄宮』崩塌的廢墟塔尖之上。
  - 此時的他，似乎感應到了什麼，正對著天空，高舉著那一盒被雨水打濕的士力架，向著虛空行了一個怪異卻鄭重的劍禮。
  ─────────────
```

蘇晨看著那一抹跨越時空的劍意，第一次露出了發自肺腑的笑容。
跨界交易，救下來的可不僅僅是命，還有一份屬於強者的不屈孤傲。

就在這溫暖且寧靜的時刻，蘇晨身周的空氣中突然泛起了一陣令他感到極度不適的、粘稠的紫色漣漪。

原本已經被上帝之杖徹底消滅的、那位「蛻變者」主教的那種嘶啞嗓音，竟然再次如附骨之疽般在他的腦海深處瘋狂響起。

「蘇晨……呵呵……你真的以為……這一切都隨著那一棒子徹底結束了嗎？神廟的大門已經在震盪中出現了兩界裂縫……十天後的龍池選拔賽上……我們那真正復甦的主……會親自在那靈池底部接見你……」

蘇晨眼神在那一瞬間冷冽如霜。他並未使用實體劍，雙指併攏猛地一揮，一道無形的劍芒瞬間炸裂，直接斬斷了那股殘存在大腦皮層的源質精神波動。

「系統，給予最高權限支持。我要在十天內，完成權限的跨級衝刺。看來這臨海市的小水溝，已經徹底裝不下我這條長大了的魚了。」

```
  【系統 2.0 終極升級提示】
  ─────────────
  解鎖全新核心功能：[本體實體跨界傳送·試行版權限]。
  當前座標已通過上帝之杖反向鎖定：[高階劍修大世界]。
  傳送消耗預計：1000 塊高品質中品靈石。
  注意：跨界行為具備極高不可控風險，建議宿主攜帶足夠的『外賣保險』與戰備物資。
```

蘇晨看著亞空間儲物間裡剩下的一堆廉價零食與剛換來的、閃爍著誘人光芒的靈石。

「劍修大世界是吧？葉孤楓，兄弟別急。我這就把你那兒的座標填進我的物流單。十天後，我得讓你那幫沒見過世面的長老們親眼看看，我們大夏文明除了酸菜泡麵，還有更特麼震撼人心的物理真理。」

*【章末番外】*
*在臨海市一處早已被軍隊封鎖、無人察覺到的地縫最陰暗處，一顆如磨盤大小、呈現深紅色的心臟竟再次微微跳動了起來。*
*而這一次，它跳動的頻率與律動感，竟然與蘇晨口袋裡那枚青銅神廟殘片完全達到了一致。*
*下一次見面，或許就不再是簡單的衛星物理打擊……所能解決的了。*
