# 第二十四章 · 震撼臨海

那一刻，整座臨海市的生靈都以為世界末日到了。

「轟隆隆——！！」

不是雷聲，而是大地被生生貫穿的悲鳴。

那道鎢金棒在極短的時間內加速到了數十倍音速，所產生的動能衝擊波直接將臨海第一醫院周圍一公里內的所有玻璃震得粉碎。

原本被紫色源質霧氣籠罩的天衡局分局，在此刻竟然奇蹟般地恢復了清明。因為那股足以鎮壓靈氣的物理衝擊力，直接粉碎了空中漂浮的所有源質顆粒。

蘇晨站在蛇巷口，他感覺自己的五官在那一瞬間失去了所有的聽覺。

但在他懷裡，林若溪正緊緊抓著他的衣領，那雙清冷的眸子倒映著遠處那根直插地心的「白色光柱」。

「那是……什麼？」林若溪的聲音微弱得不可聞。

「快快遞，加急的那種。」蘇晨吐出一口濁氣，體內的「歸墟劍意」因為感受到如此霸道的能量衝擊，竟然也隨之發生了共鳴。

```
  【戰鬥回饋】
  ─────────────
  目標：蛻變者大祭壇、主教及護衛隊。
  結果：徹底抹除（100%）。
  源質回收中... 獲取高階源質*500。
  系統經驗暴漲：升級進度 30%。
  ─────────────
```

「趙叔，你看我這快遞送得還行吧？」

老趙此時已經癱坐在收銀台後的地上，他看著窗外那久久不散的蘑菇雲，笑得眼淚都流出來了，「行……真行。蘇晨，你他娘的……真是讓老子開了眼了。」

幾分鐘後。

天衡局的靈能直升機低空掠過。秦老的身影從半空中一躍而下，他那一身中山裝已經破爛不堪，顯然剛經歷了一場慘烈的激戰。

但當他看到被徹底化為灰燼的醫院廢墟，以及那個站在蛇巷口、手中還握著奇怪激光器的少年時，老爺子的神情從驚愕，變成了欣慰，最後化作了一抹深沉的防備。

「蘇晨，那是你的『祕密武器』？」秦老走到兩人面前，眼神如刀地掃過蘇晨。

「秦老，我說過，我就是個快遞商。」蘇晨面不改色，緩緩收起激光定位器，「正好我手裡有一份來自異位面的『特別訂購』，本來是留著拆快遞包裹用的，沒想到今天派上了大用場。」

秦老沉默了良久，隨即轉向林若溪：「把這小子的檔案列入 SSS 級加密。對外，就宣稱是總部的『軌道打擊試驗』發生了誤射。」

「是，局長。」林若溪敬了個軍禮，眼神複雜地看了蘇晨一眼。

……

當晚。

臨海市再次迎來了久違的寧靜。但所有人都知道，這份寧靜是建立在一片徹底消失的廢墟之上。

周家正式宣佈覆滅，所有的資產被天衡局全盤接管。

而蘇晨，卻謝絕了局裡邀請他正式入職「總部特派員」的提議，一個人回到了老趙那間破舊的雜貨店。

地底密室。

「趙叔，您說那殘片是發射器，那魔神的頭顱……現在還好嗎？」

老趙坐在燈下，擦拭著自己的舊槍，語氣輕鬆：「都被那一根大棒子砸穿了，還能好到哪兒去？那魔神本來就是靠著地底靈脈苟延殘喘。你那一攪和，靈脈斷了，他也該徹底消停了。」

蘇晨鬆了口氣。

但他知道，這件事沒那麼簡單。

「系統，我現在的權限，能看到葉孤楓那邊的情況嗎？」

```
  【跨界監視窗口啟動中...】
  由於因果置換影響，葉孤楓位面時間加速。
  現狀：葉孤楓手持電擊槍與歸墟殘劍，正站在劍修宗門『紫霄宮』的廢墟頂端。
  他似乎正對著天空，高舉著那一盒沒拆封的士力架，向你致敬。
  ─────────────
```

蘇晨笑了。

跨界交易，救的可不僅僅是命，還有一份孤傲。

就在這時，密室的空氣中突然泛起了一陣紫色的漣漪。

原本已經被消滅的「蛻變者」主教的聲音，竟然再次在他腦海中響起。

「蘇晨……你以為……結束了嗎？神廟的大門已經鬆動……龍池賽上，我們的主……會親自接見你……」

蘇晨眼神一冷，手中劍芒一閃，直接斬斷了那股殘存的源質波動。

「系統，全力衝刺升級。看來這臨海市的小水溝，已經裝不下我這條大魚了。」

```
  【系統升級提示】
  ─────────────
  解鎖功能：[本體跨界傳送·試行版]。
  當前座標鎖定：劍修世界。
  消耗：1000 中品靈石。
  注意：跨界有風險，建議攜帶足夠的『快遞保險』。
```

蘇晨看著儲物空間裡剩下的一堆零食和剛換來的靈石。

「劍修世界是吧？葉孤楓，等我。我得讓你看看，我們這兒除了泡麵，還有更震撼的東西。」
