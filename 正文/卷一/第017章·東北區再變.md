# 第十七章 · 東北區再變

醫院地下三層，原本陰冷潮濕的空氣在此刻彷彿徹底凝固成了不透明的膠質。

蘇晨孤身一人站在那尊巨大的、正如同心臟般強而有力律動著的紫色肉瘤面前，呼吸平穩得令人感到不安。即便是在這幽暗壓抑、充斥著死亡與畸變氣息的環境下，他手中那支裝著殘餘「地心聖乳」的特製試管，依然在黑暗中頑強地散發著最後一絲微弱卻神聖的乳白色光芒，將那醜惡肉瘤的血管脈絡映襯得格外清晰。

「看見了嗎？蘇晨，這就是主賜予凡人邁向進化的終點，亦是新生。」

那名被失蹤者家屬視為最後希望、此時卻成了惡魔化身的「院長」，緩緩脫掉了那件早已被汙血染得斑駁不堪的白大褂。伴隨著一陣令人牙酸的骨骼爆裂聲，他的脊椎骨處猛然刺破皮肉，猙獰地長出了六根長達兩米、類似於昆蟲節肢的鋼纖狀利刃。每當這些黑色利刃微微顫動，都會與大理石地面摩擦出讓人頭皮發麻的火星與刺耳噪音。

「將人類脆弱如紙的肉體，與妖獸那近乎不死的強橫生命力完美融合……這，就是主的旨意，亦是這個時代唯一的救贖。」

蘇晨環視著周圍那些浸泡在充滿綠色液體器皿中、面容扭曲且正在發生不可逆畸變的同僚與市民，眼中閃過一抹前所未有的冰冷殺意。

「主？你說的是那顆長得像是一顆壞掉腐爛、還在發臭的大腰子一樣的肉瘤？」

蘇晨不屑地冷笑一聲，手中那柄被白布包裹、實則散發著驚人劍氣的中品靈石長劍猛然斜指地面，劍尖輕顫，發出一陣微弱卻堅韌的蜂鳴。

「廢話少說。既然你把自己改造成了這副連親媽都認不出來的德性，那我就發發善心，替醫保局省下那份昂貴的術後維修開支吧。」

「無知的爬蟲……找死！」

院長發出一聲非人的厲喝，背後的六根鋼鐵節肢猶如六道紫色的閃電般瞬間劃破空氣，帶著尖銳的破空聲狠狠刺出。在一陣劇烈的轟鳴中，蘇晨原先站立的那塊厚實地磚瞬間被紮成了密密麻麻的孔洞，碎石飛濺。

然而，蘇晨的身影卻在這一刻快得猶如幻影。他在這狹窄侷促、布滿了化學儀器的空間內，施展出了《長青吐納術》所附帶的一套秘傳步法。

每當那鋼鐵節肢落下的剎那，他都能以毫米級的極限差距，險之又險地擦著鋒刃避開。

「瞬殺·寂滅歸墟！」

就在院長因為用力過猛而產生短暫僵直的剎那，蘇晨眼神猛然凝聚如針。他手中的長劍在半空中劃出一道肉眼根本無法捕捉的黑色流光，那是跨界學來的、足以切斷生機的絕世劍意。

「噗嗤——！」

伴隨著綠色腥臭血液的瘋狂噴濺，院長背後兩根最為粗壯的異化節肢竟然被齊根斬斷，重重地砸在地板上，發出沉重的金屬撞擊聲。

「啊——！！這不可能！你只是一個二階初期的雜魚廢物，怎會掌握這等連那幫劍瘋子都難以修成的純粹劍氣？」

院長狼狽地踉蹌退後，六根節肢只剩下了四根，那張被紫光徹底佔據的臉龐顯得愈發猙獰，「你到底隱藏了什麼實力？！」

「世界很大，你這輩子都縮在這暗無天日的地溝裡，沒見過的東西多了去了。」

蘇晨面無表情，手掌緩緩握緊劍柄，體內的長青靈氣正源源不斷地灌注進劍身，準備再次出劍徹底了結這場令人作嘔的鬧劇。可就在這千鈞一髮之際，他的識海中突然響起了一陣前所未有的、震耳欲聾的系統紅色警報。

```
  【區域級·滅世靈壓衝擊預警！！】
  ─────────────
  探測目標：臨海市·東北郊區遺跡核心。
  能量反應指數：呈幾何倍數爆炸式攀升！
  事件類型：超大型空間位面通道徹底開啟。
  警語：極高風險！檢測到三階以上、疑似領主級強大生物氣息正在橫跨界限降臨。
  ─────────────
```

「轟隆隆——！！」

就在文字刷屏的下一秒，整個臨海市的地皮在那一刻彷彿遭遇了十級大地震，開始了劇烈的顫抖。

醫院那堅硬的混擬土牆壁在瞬息之間裂開了無數漆黑的縫隙，原本維持空間重疊、屏蔽搜索的實驗室大陣在這種絕對的自然伟力面前，脆弱得就像是一張薄紙，瞬間徹底崩潰。

「什、什麼？那邊發生了什麼？！」院長顧不得斷臂之痛，驚駭欲絕地看向那扇被震碎的氣窗外。遠處東北方向的夜空，此時已被染成了一種詭異、壓抑且帶著不詳氣息的紫青色。

蘇晨也停下了即將刺出的一劍。

他感覺到了一股來自靈魂深處的微弱戰慄，那並非恐懼，而是一種位階上的絕對壓制。那股氣息是如此宏大、冷漠，遠超他見過的二階、三階生物，甚至是不屬於這個正在復甦的世界。

「蘇晨！你還活著就快接電話啊！」

通訊器裡傳來林若溪近乎崩潰、帶著一絲哭腔的急促喊聲，「撤退！所有在外的調查員、教官立刻無條件撤回分局地下堡壘！東北區的封鎖線徹底失控了……有一頭領主級的恐怖妖獸……它、它竟然強行擠進來了！」

蘇晨冷冷地掃了一眼縮在大肉瘤旁、狼狽不堪的院長，收劍入鞘，語氣森然：
「算你今天命大，我的頂頭上司喊我回去領更燙手的獎金。這肉瘤你先留著當棺材板吧。」

「蘇晨……別跑……大勢已至，誰也……誰也活不下來……」院長狼狽地蜷縮著，那顆巨大的紫色肉瘤在遠處裂縫爆發的感應下，竟然開始瘋狂地抽取空氣中原本溫和的靈氣，體積以肉眼可見的速度膨脹，表皮變得愈發鮮紅欲滴。

蘇晨沒有再耽擱一秒鐘。

物有先後，事有緩急。這「蛻變者」的地下實驗室雖然令人作嘔，但比之東北區那即將吞噬整座臨海市、甚至是毀滅一切的位面級威脅，後者才是真正的燃眉之急。

他轉身如獵豹般衝出地下室，在翻身跳上一輛停在路邊、原本用來運送物資的運鈔車頂時，他看見了這輩子都難以抹去的震撼一幕。

臨海市的東北方天際線，一個直徑高達數百米的巨大紫色雲霧旋渦橫跨了大半個蒼穹。

而在旋渦的最中心，一隻遮天蔽日、足以覆蓋數條街區的巨大蒼青色羽翼正緩緩從異次元探出。羽翼上流動著刺眼如岩漿般的神祕金色符文，每一次輕微的扇動，都會引發一場足以掀翻重型卡車的狂暴靈氣風暴。

「三階巔峰領主：焚天青鸞。」

蘇晨深深地倒吸一口涼氣，心跳快得驚人。

那是劍修大世界葉孤楓在談及「極北禁區」時，連聲音都會自覺顫抖的恐怖存在。

「系統，別廢話！如果我現在將剛才賺到的三千下品靈石全部燃毀換算成系統升級能量，夠我多久強行突破至三階·御物境？」

```
  【極速大數據運算完畢】
  - 消耗預算：1000 下品靈石（全額能量化）。
  - 當前突破成功概率：95.8%。
  - 所需最快護法時間：10 分鐘。
  - 終極警示：突破過程中產生的靈氣潮汐極其劇烈，極易引發高階領主生物的本能仇視與獵殺。
```

蘇晨跳下車頂，眼神中透著一股向死而生的瘋狂與狠辣。

「十分鐘是吧？好！那我就親自給自己找個最危險、也同樣是最安全的地方去突破。」

他在漫天尖叫的人群中逆行，攔住了一輛正頂著風暴往分局方向狂飆的黑色越野車。在林若溪那震驚、充滿了不可置信的目光中，他一把拉開車門，按住了她那雙正在顫抖的冰冷肩膀。

「林隊。掉頭，別回那什麼狗屁分局了，那兒保不住你。」

蘇晨指著那緩緩降臨、將整座城市都映照成末日慘狀的天際羽翼，語氣是前所欲有的沉重且決絕：
「跟我去東北區。既然這世道已經瘋了，那我們就直接去亂源的核心，給它這賊老天捅個真正的透明窟窿出來！」

「蘇晨……你知不知道你在說什麼？你徹底瘋了嗎？那可是領主！」

「瘋的不是我，而是這個不再講究資歷、只看誰拳頭更硬的世道。」
蘇晨矮身坐進了副駕駛，隨手捏碎了一塊散發著強大波動的中品靈石，將其粉末直接吸入臟腑，
「開車吧，林隊長。你說過的……那三萬塊錢的人頭獎金，我可是打算今天去現場簽收的。」
