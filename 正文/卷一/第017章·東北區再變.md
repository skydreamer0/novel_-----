# 第十七章 · 東北區再變

醫院地下三層的冷氣似乎凝固了。

蘇晨站在那尊巨大的紫色肉瘤面前，呼吸平穩。即便是在這幽暗壓抑的環境林下，他手中的那管地心聖乳殘液，依然在散發著最後一絲微弱的光芒。

「看見了嗎？蘇晨，這就是進化的終點。」

那名被稱為「院長」的怪物，緩緩脫掉了染血的白大褂。他的脊椎骨上刺破皮肉，長出了六根類似鋼纖的節肢，每動一下都發出令人牙酸的摩擦聲。

「將人類虛弱的肉體，與妖獸那強橫的生命力完美融合……這，就是主的旨意。」

蘇晨看著那些器皿中痛苦掙扎的同僚，眼中閃過一抹前所未有的冷色。

「主？你說的是那顆長得像壞掉的大腰子的肉瘤？」

蘇晨不屑地嗤笑一聲，手中的長劍猛然指地。

「廢話少說。既然你把自己改造成了這副德性，那我就替醫保局省點開支吧。」

「找死！」

院長大吼一聲，六根節肢猶如閃電般刺出，瞬間將蘇晨剛才站立的地磚扎成了篩子。

蘇晨的身影快如鬼魅，他在這狹小的空間內施展出了《長青吐納術》步法的極致。每一根節肢落下，他都能以毫米級的差距險之又險地避開。

「瞬殺·歸墟！」

蘇晨眼神一凝，手中的長劍在半空中劃出一道肉眼難辨的黑線。

「噗！」

兩根異化的節肢齊根斷裂。

「啊——！！」

院長發出一聲非人的咆哮，「這不可能！你只是一個二階初期的廢物，怎會有這等劍氣？」

「世界很大，你沒見過的東西多了去了。」

蘇晨正準備再次出劍徹底了結對方，突然，他的識海中響起了一陣尖銳的警報。

```
  【區域級靈壓警報！！】
  ─────────────
  探測目標：東北郊區。
  能量指數：爆炸式攀升！
  事件類型：超大型空間裂縫開啟。
  警語：極高風險，檢測到三階以上領主級生物氣息跨界。
  ─────────────
```

「轟隆隆——！！」

整個臨海市的地皮在那一刻劇烈震動起來。醫院的牆壁裂開無數縫隙，原本維持空間重疊的實驗室大陣在瞬間崩潰。

「什、什麼？」院長驚駭地看向窗外，那裡，東北方向的夜空已被染成了詭異的紫青色。

蘇晨也停下了動作。

他感覺到了一股來自靈魂深處的戰慄。那是遠超二階、三階，甚至是不屬於這個世界的絕對權威。

「蘇晨！快接電話！」

通訊器裡傳來林若溪近乎崩潰的喊聲，「撤退！所有在外的調查員立刻撤回分局！東北區的裂縫失控了……有一頭領主級的妖獸……它、它進來了！」

蘇晨看向地上的院長，冷聲道：「算你運氣好，我的雇主喊我回去了。」

「蘇晨……你逃不掉的……大勢已至……」院長狼狽地蜷縮在大肉瘤旁，那顆肉瘤卻在裂縫爆發的瞬間，開始瘋狂地吸收空氣中的靈氣，體積迅速膨脹。

蘇晨沒有再耽擱。

物有先後，事有緩急。這「蛻變者」的實驗室雖然噁心，但東北區那即將吞噬整座城市的威脅才是當務之急。

他轉身衝出地下室，在翻身跳上一輛停在路邊的運鈔車頂時，他看見了畢生難忘的一幕。

臨海市的東北方，一個直徑數百米的巨大紫色旋渦橫跨天際。

而在旋渦的中心，一隻遮天蔽日的巨大羽翼正緩緩探出，羽翼上流動著像是熔岩般的符文，每一次扇動，都引發一場席捲全城的靈氣風暴。

「三階巔峰領主：焚天青鸞。」

蘇晨倒吸一口涼氣。

那是劍修世界中，連葉孤楓提起來都要色變的恐怖存在。

「系統，我現在的靈石全部換算成經驗，夠我多久突破三階？」

```
  【計算完畢】
  消耗 1000 下品靈石。
  突破概率：95%。
  所需時間：10 分鐘。
  警示：突破過程中靈氣波動劇烈，極易引發高階生物敵意。
```

蘇晨跳下車，眼神中透著一股狠辣。

「十分鐘是吧？好，我給自己找個最危險、也最安全的地方去突破。」

他攔住了一輛正往分局狂飆的林若溪的越野車，在林若溪震驚的目光中，一把按住了她的肩膀。

「林隊。別回分局了。」

蘇晨指著那緩緩降臨的天際羽翼，語氣前所欲有的沉重。

「跟我去東北區。既然這世界要亂，那我們就去亂源的核心，給它捅個窟窿出來。」

「你瘋了？」

「瘋的不是我，是這世道。」蘇晨坐進副駕駛，隨手捏碎了一塊中品靈石，「開車。三萬塊獎金，我可是要去現場簽收的。」
