# 第十二章 · 老趙的壓箱底

清晨，當第一縷溫暖且細碎的金色陽光越過遠處鱗次櫛比的摩天大樓輪廓，慢吞吞地投射進天衡局高級特護宿舍那扇巨大的落地窗時，蘇晨猛地睜開了眼睛。

他的右手下意識地摸向那鬆軟的枕頭下方，直到指尖觸碰到那個冰涼、堅硬且帶著幾分鏽跡的黑色鋼管——那是他隨身攜帶、用來偽裝的「法器」——他緊繃的神經才終於在這一刻稍微放鬆了一些。

「呼……這一覺，睡得可真沉，連個夢都沒做。」

蘇晨緩緩坐起身，雙手向上伸展，全身的骨骼都在這一刻發出了如同炒豆子般輕微且連綿的脆響。那種昨晚因為強行催動劍氣而產生的靈氣透支與肌肉酸痛感已經消失了大半，取而代之的是一種脫胎換骨般的輕盈感。他甚至能感覺到，體內的長青氣息運作得比以往任何時候都要流暢、絲滑。

他低下頭看了看自己的胸口。昨晚那件披在身上、帶著淡淡清冷寒梅香味的白色披風已經不知何時被收走，取而代之的是一身裁剪得體、散發著淡淡皂香的整潔病號服。

```
  【系統檢測模塊自動運作中...】
  宿主生理狀態：優（細胞修復率98%）。
  靈氣充盈度：45.3% -> 62.1%（正隨著自然呼吸穩定回升）。
  總結：戰後突破帶來的生機重組已接近尾聲。
```

蘇晨看著識海中跳動的綠色數據，咧嘴露出一抹滿足的笑容。昨晚那一式「瞬殺」雖然險些抽乾了他的底蘊，但回報之豐厚也是超乎想像的。他能清晰地感覺到，原本那層阻礙他晉升的二階瓶頸已經徹底崩碎，現在他的根基如磐石般穩固，修為更是隱隱有向二階中階攀升的狂暴趨勢。

「既然有力氣對著鏡子傻笑，就說明死不了，趕緊起來把東西吃了。」

伴隨著一聲清冷如碎冰的嗓音，醫務室那扇厚重的防彈門被輕輕推開。林若溪手裡提著兩份用牛皮紙包裝、還在冒著騰騰熱氣的早餐走了進來。她今天的神色比起往常似乎更加冷峻，眼底甚至還帶著一絲淡淡的金青色倦容，顯然那些繁雜的戰後善後工作與審訊，讓這位冰山指揮官徹夜未眠。

「隊長，昨晚那個……關於戰鬥中損毀的快遞配備報銷，還有我個人的精神損失……」

「報銷的事情局裡已經在特事特辦走加急流程了，少不了你的那份。」林若溪沒好氣地白了他一眼，將兩大杯新鮮的豆漿和大大的肉包子布放在桌上，發出砰的一聲悶響，「另外，告訴你一個壞消息。你原本租住的那間公寓暫時回不去了。周家那幫老狐狸現在正動用所有關係滿世界找你，想私下詢問你關於那顆源質結晶的每一個細節。而且根據我們截獲的情報，『蛻變者』組織內部既然啟動了針對你的二號行動計畫，說明他們已經正式把你列入了『核心高價值回收名單』。」

「回收？我又不垃圾，也沒打算被誰回收。」蘇晨咬了一口油乎乎的大肉包，汁水四溢，他含糊不清地嘟囔著，「隊長，局裡既然現在限制我的行動，那我能不能請個假回趟蛇巷？我有個從小看著我長大的長輩在那兒，這兩天市區不穩定，我不親眼看他一眼，我這心窩子定不下來。」

林若溪端著豆漿的手在半空中沉默了片刻，那雙漂亮的眸子透出一抹深思：「你是說那個叫老趙的、在巷子尾開雜貨店的老頭？」

「喲，隊長你這記性可真好，連這種『小商小販』都注意到了？」

「臨海市境內所有有特別檔案記錄的覺醒者，都必須在天衡局的動態監控網之下。趙德厚，前任天衡局實戰組總教官，二十年前因重傷退役，目前實力評定等級為『三階廢棄』，且身患多種靈力反噬症。」林若溪語氣平淡地敘述著冷冰冰的檔案，隨後頓了頓，「如果你真的想去，正好我也要帶隊往蛇巷那邊區域進行例行排查。吃完跟我上車。」

……

半小時後，充滿了市井煙火氣與喧鬧聲的蛇巷。趙記雜貨店門口。

巷子裡的氣氛依舊如往常那般悠閒且安寧，路邊的煎餅果子發出的呲啦聲與小販的叫賣聲交織在一起，彷彿昨晚那場足以驚動半座城市高層的生死激戰，在這片古老的陰影裡從未發生過。

老趙依舊是那副邋裡邋遢的懶散模樣，翹著二郎腿晃悠著藤椅，手裡那份《臨海晨報》已經在手心裡攥得皺巴巴的，連拿反了這件事都顯然沒發現。

當他看見蘇晨跟在面無表情的林若溪身後，一臉憨笑地走進店門時，老趙原本鬆弛的身子猛地僵直了一下，隨即扯開嗓門破口大罵：

「臭小子！你還敢腆著臉回來！我這小破店一年的租金比你這條命都貴，要是你這惹禍精把那些瘋子引過來把店給炸了，老子非把你全身這二兩肉給颳下來燉湯不可！」

林若溪倒是很有禮貌，微微低頭向這位前輩示意：「趙老師，許久不見，打擾您了。」

老趙斜著眼瞟了她一眼，語氣不鹹不淡中帶著幾分排斥：「喲，這不是林家那位驚才絕豔的小天才嗎？不在分局大樓裡指揮抓妖，帶著我這不成器的侄子來這兒敘什麼舊？」

「蘇晨只是擔憂您的身體，想過來看看。」林若溪也不氣惱，她似乎很清楚這位老教官的脾氣，轉頭對蘇晨說，「我去周圍巷口巡邏一圈布置監測點，十分鐘後，我在巷子口拉紅燈等你。」

說完，她優雅地轉身離開，臨走前還順手拉上了雜貨店那扇玻璃上貼滿了膠帶、一拉就嘎吱作響的破爛大門。

隨著玻璃門合上的那一刻，老趙原本那副暴躁、市儈的神情瞬間消失得乾乾淨淨。他猛地從藤椅上站起，三步並作兩步衝到蘇晨面前，那雙原本渾濁不堪的眼睛裡，突然爆發出一股幾乎要凝為實質的凌厲精芒。

他像是看怪物一樣，從頭到腳將蘇晨反覆打量了三遍，最後目光死死定格在了蘇晨那垂在地側、正隱隱散發著一絲霸道劍意的指尖。

「二階凝氣？而且氣息連成都快圓滿了？」老趙倒吸了一口特有的旱煙味。

「嗯，昨晚情況緊急，僥倖突破了。」

「僥倖個狗屁！」老趙壓低了嗓子，嗓音因為激動而顯得異常沙啞，「昨晚那橫掃半條街的劍意……是你這毛都沒長齊的小子發出來的？我在蛇巷這頭都能感覺到那股子死絕、寂滅的恐怖味道。蘇晨，你老實給老子交代，你小子是不是在送快遞的時候，被哪個遠古遺蹟裡爬出來的老怪物給奪舍重生了？」

蘇晨有些無奈地聳聳肩，攤開雙手：「趙叔，您看我這熟練的拆包裹手法，像是被老怪物奪舍的樣子嗎？」

老趙沉默了，他死死盯著蘇晨的眼睛，試圖尋找一絲說謊的痕跡。良久，他才像是卸下了渾身的勁兒，悠悠地長嘆了一口氣。

「罷了，每個人都有自己的逆天造化。既然這賊老天選中了你，老子也懶得當那個攔路虎。」

他轉身走回堆滿了雜物的收銀檯，在那個擺滿了廉價「鹽漬話梅」的粗瓷破罐子後面摸索了老半天，最後按下了某個機關，掏出了一個落滿陳年灰塵的小暗格。

暗格中央，靜靜地躺著一枚造型古樸、通體呈現出一種如深淵般漆黑色的金屬戒指。戒指表面沒有任何珠光寶氣的裝飾，只有一圈像是瘋狂生長的黑色藤蔓般的暗紋在戒身表面緩緩流動，透著一股沉重的歷史感。

「趙叔，這戒指……是給我的？」

「廢話，趕緊拿著。」老趙一把抓住蘇晨的手，將那枚沉甸甸的戒指不由分說地塞了過去，「這玩意兒叫『墨隱環』，或者叫隱氣戒。別看它賣相醜，這可是老子當年拼掉半條命，從一處上古修士的戰場遺蹟裡帶出來的好貨。雖然這玩意兒現在沒了原本的防禦功用，但它有一個最核心的效果——能徹底鎖死你全身的靈氣波動曲線。」

老趙指了指窗外林若溪離去的方向，眼神凝重。

「你現在表現出的實力太過扎眼了。哪怕是那個背景通天的林家丫頭，只要你戴上這枚戒指，在她的靈力感知網裡，你永遠只是一個氣息雜亂、初入一階的普通菜鳥。只要你別在人前隨便拔劍，這世上基本沒人能看穿你真正的修為底細。」

蘇晨心中猛地一震。這簡直是為他這個「萬界大忽悠」量身定做的神器啊！

他最近一直在發愁，隨著他跨界交易次數的增多，實力提升的速度已經快到了離譜的地步，根本無法用地球現在的體系來解釋。有了這枚隱氣戒，他就能繼續披著那張「D級潛力快遞員」的保護色，在暗處悶聲發大財。

「趙叔，這東西太貴重了，這可是您的壓箱底寶貝……」

「少給老子在那兒矯情！老子早就想把它處理掉了，免得每天看著它，心裡就想起當年那些死在遺蹟裡的兄弟老夥計。」老趙不耐煩地擺擺手，重新坐回那張吱呀作響的藤椅，語氣變得有些深邃且擔憂，「蘇晨，以後臨海市的日子可就沒以前那麼好混了。周家那邊已經跟省城的『蛻變者』高層正式搭上線了，那幫自詡為新人類的瘋子丟了核心結晶，絕對不會善罷甘休的。」

蘇晨將戒指戴上。就在戒圈吻合的一瞬間，他感覺體內原本奔騰如大河的靈氣像是被一層厚實、溫暖的黑幕徹底籠罩，外界能感知到的靈壓強度立刻呈斷崖式下跌，最後穩穩地停在了「一階覺醒者」的邊緣。

「趙叔放心，我這人別的本事沒有，就是命硬，閻王爺暫時不收快遞。」

「命硬那也得腦子夠用才行。記住老子一句話，這世上最危險的從來都不是那些長毛的妖獸，而是那一層層人皮下的人心。」老趙再次裝模作樣地拿起報紙遮住臉，語氣恢復了那種懶洋洋的調子，「滾吧滾吧，林家的小妮子都在門口按喇叭催你了。下次來的時候，記得去市中心給老子帶兩瓶正宗的西湖龍井，天天喝這點破茶根，老子舌頭都喝木了。」

蘇晨深深地看了老趙一眼，對著這位隱居在市井中的老教官，極其鄭重地行了一個晚輩禮，隨後轉身大步走出。

雜貨店門口，林若溪正斜靠在那輛巨大的軍綠色越野車旁。她看著蘇晨走出來，那雙秀麗的眉毛微微一挑。

「老朋友談完了？看你這表情，他沒少罵你吧？」

「談完了。趙老頭身體倍兒棒，就是那脾氣簡直像個點著了的煤氣罐。」蘇晨又恢復了那副招牌式的憨笑，動作嫺熟地摸著後腦勺，「隊長，現在咱去哪兒？是直接回分局領那剩下的三萬塊戰鬥津貼嗎？」

林若溪狐疑地打量了蘇晨一圈。不知為何，她總覺得眼前的少年在進入雜貨店短短十分鐘後，整個人的氣息似乎變得更加和諧自然了，就像是一個再普通不過、整天為了油鹽醬醋奔波的「快遞小哥」。

「先回分局，去後勤部領一套你的個人初級武裝配備。局長親口點名了，後天上午有一場關於東北區上古遺跡外圍開發的聯合探索會議，要求你也列席參加。」

林若溪拉開沉重的車門，語氣比剛才緩和了不少。

「還有，蘇晨。關於你那招威力大到不可思議的劍法……既然你不想在檔案裡真實展示，等一會兒回去寫實戰報告的時候，你就把它統一表述為是臨時激發了某種一次性的、祖傳的『家族防護聖符』。我會在報告審核欄幫你簽字封存，列入最高保密權限。」

蘇晨愣了一秒，隨即露出一個會心的溫暖笑容。

「多謝林隊長美言，這份人情我記下了。」

隨著引擎的一聲低吼，越野車如同一頭鋼鐵巨獸，朝著臨海市繁華的中心區域疾馳而去。

蘇晨整個人放鬆地靠在厚實的椅背上，修長的手指輕輕轉動著指間那枚漆黑沉穩的戒指，嘴角勾起一抹極度危險且興奮的弧度。

「系統，立刻幫我再次定位劍修大世界。告訴葉孤楓，既然有了我送他的雷霆神器，就該讓那些想讓他含冤而死的廢物們見識見識，什麼叫真正的『九天玄雷劫』。」

一場更為宏大、足以撼動多個位面格局的連鎖交易與肅殺之局，正在這座看似平靜的臨海市上空，悄然成型。
