# 第二十八章 · 終極對決（下）

紫色的霧氣籠罩了龍池訓練營，那尊源質主教的身軀在虛空中瘋狂律動。他的每一條觸手都像是一根吸管，正瘋狂地抽取著火山口下方積蓄百年的地脈靈力。

「蘇晨……你毀了我經營十年的聖所……我要把你的靈魂，塞進這火山最深處，燒上一千年！」

主教的聲音在天際迴盪，宛如雷鳴。

「大話誰都會說。」蘇晨單手持劍，另一隻手卻在大口大口地往嘴裡塞巧克力，「系統，同步所有能動用的劍修精元。這一次，我不打算留手了。」

```
  【能量同步率：100%】
  【警告：身體負荷達到臨界點！】
  蘇晨身分變更：[萬界劍意承載者]。
```

那一瞬，蘇晨那一頭漆黑的碎髮，在此刻竟然以肉眼可見的速度轉化為了一股妖異的亮銀色。每一根髮絲，都吞吐著足以切斷視線的銳利劍氣。

「趙凌霄，看清楚了。這才是真正的……潛龍出淵。」

蘇晨腳尖在一塊斷裂的履帶上猛地一蹬。

「砰——！！」

地面下陷。蘇晨化作了一道刺破紫色天幕的白色流光，在那漫天落下的紫色毒雨中，逆流而上！

「找死！」主教怒吼一聲，數十條長達百米的觸手交織成一張巨網，帶著足以拍碎山岳的力道，朝蘇晨合圍而去。

「劍起，歸墟·開天。」

蘇晨在高空中，雙手合握那柄鐵片劍柄。

原本平平無奇的劍格處，突然噴湧出長達百米的漆黑劍芒。那是寂滅一切、吞噬生機的歸墟之意，與這世間最極致的黑暗撞击在一起。

「啦——！！」

空間發出了刺耳的碎裂聲。

那些無堅不摧的紫色觸手，在碰到黑芒的瞬間，全部像被高溫融化的蠟燭般消融、崩解。

蘇晨的速度不減反增。他就像是一枚燒紅的鋼針，直接刺入了源質主教中心那顆跳動的紫色心臟。

「不——！！這是什麼力量？你的背後到底站著什麼樣的存在？！」主教驚恐地發現，自己那半步五階的強橫生命力，在被這股力量瘋狂地……抹除。

「他背後站著的是……萬界。」

蘇晨在心底默默回應。

「爆！」

蘇晨鬆開劍柄，雙手直接按在了主教的核心源點上。

那一瞬，他腰間那十箱「紫色雷管」被同時引爆。

那是劍修世界的雷霆能量。

那是超越了這個低武世界理解範疇的、天劫級別的懲罰。

「轟——！！！！！」

一朵碩大無比的紫色蘑菇雲在龍池訓練營上空升騰。爆炸產生的光芒，方圓千里內的省城、臨海市，乃至更遠的城鎮都清晰可見。

狂暴的雷火將所有的紫色霧氣燒毀，連那不可一世的主教，都在這神罰之下化作了虛無。

火山口劇烈噴發，但噴出的不再是致命的岩漿，而是被雷火精煉過的、最純淨的靈氣甘霖。

足足過了半個小時。

煙塵散去。

蘇晨半跪在火山口邊緣，渾身血跡斑斑，銀髮恢復了黑色。他的呼吸微弱到了極點，手中的那柄鐵片劍已經化作虛無，唯有掌心裡握著一顆拳頭大小、流光熠溢的紫色結晶——那是主教最後留下的核心，也是進化到了極致的源質。

「蘇晨！」

林若溪不顧一切地衝破禁錮房的廢墟，跑到了他身邊。

「還……沒死呢……」蘇晨抬起頭，露出一個慘白的笑容，「隊長，你看……這場面，夠不夠抵消我的……補償費？」

林若溪看著他這副模樣，眼眶突然一紅。她沒說話，只是用力地、死死地抱住了他。

一旁的趙凌霄，還有那些倖存下來的省城天才，此時全都低下了頭。

這一劍，劈開的不僅僅是主教，還有他們心中那原本高高在上的傲慢。

在龍池之巔，這個被評定為 D 級潛力的少年，完成了一次足以載入整個人類史冊的——神級救贖。
