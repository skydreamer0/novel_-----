# 第033章 · 擺攤：靈石收購商

清晨。

當第一縷微弱的陽光試圖穿透萬劍崖終年不散的陰冷氣息時，青雲峰上已經響起了一陣陣充滿活力的喝彩聲。

蘇晨坐在一張由不知名靈木打造、雖然有些蟲蛀但還算結實的摇椅上，悠閒地晃動著。

他的面前，一塊顯目的、用噴漆在白布上噴製的招牌正迎風招展：

【萬界大甩賣：回收各類靈石、靈藥殘片。兌換比例：巧克力、打火機、冰鎮可樂。】

這幾個字寫得龍飛鳳舞，配上招牌下方那兩具散發著神祕感的光能路燈（淘寶買的太陽能草坪燈），在這一片死寂的禁地中顯得極其礙眼，卻又引人注目。

「蘇哥，真的會有人來嗎？」

葉依依站在一旁，身後背著那柄已經被酒精和細砂打磨得閃閃發亮的短劍。她此時換上了一身蘇晨送她的黑色運動套裝，雖然有些寬大，卻襯托得她的肌膚如雪，平添了幾分異位面的野性美。

「放心吧，這禁地的情報傳遞速度，可比你想像中快得多。」

蘇晨打了個哈欠，隨手撕開一塊德芙巧克力的包裝，塞進嘴裡。

「這空氣中都是貪婪的味道，那些躲在暗處偷偷觀察的『散修』，可憋不了多久。」

果然。

不到一刻鐘，一陣急促、卻又有些猶豫的破風聲從遠處的山脊處傳來。

三名身穿灰袍、看上去賊眉鼠眼的修士，正踩著飛劍在半空中盤旋。當他們看清這山頂上的景象時，那表情簡直比見了鬼還要精彩。

「那是……葉孤楓？他居然沒死？」

「等等，你看他手裡拿的是什麼？那是靈氣實體化後的黑玉（巧克力）嗎？他在舔……他在舔那玩意兒？！」

「還有那個穿著奇怪衣服的傢伙，他身邊那是……那是傳說中的『長明神燈』？居然不需要任何靈力波動就能發光？」

蘇晨瞇著眼，對著天空招了招手。

「上面的朋友，別飛了，下來歇歇。新店開張，前三位客人有贈品，過期不候啊！」

那三名散修對視一眼，眼底閃過一抹狠戾。在這禁地混久了，他們信奉的從來不是公平交易，而是弱肉強食。

「上！那小子看著沒什麼靈力波動，估計是靠那些寶貝撐場面。搶了這波，咱們這輩子就不用在魔霧裡吃乾草了！」

「嗖、嗖、嗖！」

三道飛劍化作流光，帶著一股子煞氣直取蘇晨的首級。

蘇晨依舊坐在摇椅上，連眼皮都沒抬一下。

「依依，這活兒交給你了。動作快點，別耽誤我做生意。」

「是，蘇哥！」

葉依依身形如電，在那三名散修驚駭的目光中，她腳下的運動鞋踩出一聲輕響，整個人竟然化作一道殘影消失在原地。

「青雲十三劍·歸一！」

隨著一聲嬌斥，一道清亮的、隱含著某種玄奧氣息的劍氣橫掃而過。

「鐺、鐺、鐺！」

三聲脆響。

那三柄看起來頗具氣勢的飛劍，竟然在空中被硬生生震斷了鋒刃。那三名散修更是像是被重鎚擊中，慘叫著倒飛而出，重重地摔在蘇晨面前。

「怎……怎麼可能？！你是葉依依？你不是經脈盡斷了嗎？！」為首那名散修驚恐地看著眼前的少女。

一晚上的時間，這小姑娘不僅痊癒了，甚至連修為都突破到了二階巔峰？那些「神露」到底是什麼神丹妙藥？！

「好了，談生意就要有談生意的態度。」

蘇晨此時慢悠悠地站起身，手裡拋著一個五塊錢一個的透明打火機。

「滋——！」

一道微小的、卻極其穩定的火苗在寒風中倔強地跳躍著。

那三名散修瞪大了眼睛。

在禁地中，想要生火是一件極其困難的事，尤其是在隨時會熄滅火焰的魔霧中。而這小玩意兒，竟然能隨心所欲地操控火焰？

「這個，叫『赤陽引火器』。裡面封印著南明離火的火種碎片。」

蘇晨說起謊來臉不紅氣不喘。

「一塊拳頭大小的下品靈石，換一個。」

散修們咽了咽口水，看著那跳動的火焰，又看了看蘇晨腳邊那箱散發著誘人香氣的巧克力。

「我……我換！我這裡有兩塊下品靈石，是昨天剛從廢墟里刨出來的！」

一名散修顫抖著手，從懷裡摸出了兩塊灰不溜秋的晶體。

蘇晨掃了一眼，眼神中閃過一抹喜色。

在地球，這玩意兒是無價的能源。在這裡，對於這些物資匱乏的人來說，這就是能夠換取「文明希望」的籌碼。

```
  【成功發起一筆跨界交易。】
  【支付物品：普通打火機 *2。】
  【獲取物品：下品靈石 *2（約含純淨靈能 50 單位）。】
  【當前利潤空間：15000%】
```

「不錯，你是識貨的。這是贈品，拿著。」

蘇晨隨手丟過去兩粒用錫紙包裹的德芙巧克力。

「吃下去，如果你覺得自己像是在吞噬這世間最美味的修煉精華，那說明你的體質很適合跟我長期合作。」

那名散修迫不及待地剝開巧克力，塞進嘴裡。

那一瞬間。

那種絲滑、濃郁、且帶著一絲甜膩的化學提煉感，在久經風霜的舌尖炸裂開來。他整個人像是被雷劈了一樣，眼神從迷茫轉向狂喜，最後竟然雙膝跪地，泣不成聲。

「這……這就是上界的味道嗎？這就是……神恩啊！」

在那名散修的瘋狂宣揚下。

原本寂靜的青雲峰，在短短不到半天的時間裡，竟然引來了這片區域內幾十名聞風而動的散修。

在一片「咔嚓、咔嚓」吃巧克力的聲音和「滋、滋」打火機的聲響中，蘇晨面前的那個原本空空如也的木筐，此時已經被五顏六色、形狀各異的下品靈石填滿了大半。

蘇晨看著那堆積如山的靈石，嘴角露出一抹狡黠的微笑。

什麼是壟斷？

這就是壟斷。

用全世界最廉價的工業邊角料，換取另一個世界最核心的高級能源。

而這個世界的人，還在為能得到一塊巧克力而對他感恩戴德。

這種降維打擊帶來的爽感，簡直比親自上場殺敵還要刺激一百倍。

「葉校長，你可以通知附近的那些小宗門長老了。」

蘇晨仰頭喝了一口冰鎮可樂，那氣泡在舌尖跳躍的聲音，宛如權力的序曲。

「這場萬界大甩賣……才剛剛進入高潮啊。」
