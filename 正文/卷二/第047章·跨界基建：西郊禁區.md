# 第047章 · 跨界基建：西郊禁區

臨海市西郊，這片曾經被遺忘的工業廢地，如今已經成了全球最神祕、也最讓人嚮往的「禁區」。

清晨，當第一縷陽光試圖穿過那層淡紫色的「靈能過濾網」時，整個園區發出了一陣類似於蜂鳴的低頻震動。幾百名由蘇晨從異界召喚回來的「機關傀儡」，正整齊劃一地在工地上忙碌著。

這些傀儡與地球常規的機器人完全不同。它們沒有繁複的電路板，只有銘刻在核心內部的、細如髮絲的靈力迴路；它們的關節靈活如真人，卻擁有著能輕易托起十噸重物的恐怖力量。

「蘇哥，這就是你說的『全自動裝卸港口』？」

葉依依穿著一身幹練的迷彩作戰服，手裡拿著一瓶冰鎮美年達，一臉好奇地看著眼前那座高達五十米、通體由玄鐵合金打造的黑色巨塔。

巨塔的頂端，一團巨大的雷電能量正在緩緩旋轉，形成了一個穩定的空間渦旋。

「這叫『亞空間物流接駁塔』。」

蘇晨戴著一副遮陽鏡，正靠在一輛嶄新的、通體漆黑的悍馬越野車旁，悠閒地嚼著口香糖。

「以後異界的貨物，不需要我親自來回跑，只要在兩端建好這種接駁塔，普通的物資就能實現自動化傳輸。靈石、丹藥、甚至是異界的廢棄礦石，都能像流水線一樣送進我們的倉庫。」

這就是蘇晨升級 Lv.3 系統後的真正底氣。

他不再僅僅是一個快遞員，而是一個跨位面的物流巨頭。

「但是，蘇哥，這動靜是不是太大了點？」

葉依依指了指圍欄外。

只見在園區的紫色障壁外，整齊地停靠著幾十輛掛著軍方牌照的吉普車，數百名穿著天衡局制服、全副武裝的精英戰士正神情緊繃地巡邏著。甚至在遠處的山頭，還能隱約看見幾座臨時搭建的防空導彈陣地。

他們不是來進攻的，他們是來「保護」這個聚寶盆的。

「動靜大才好。」

蘇晨冷笑一聲，「我要的就是讓全世界都知道，規矩是我定的。誰想來分一杯羹，就得按照我的步調走。林若溪那邊準備得怎麼樣了？」

「林姐已經去見那個『藍星科技』的代表了。」

葉依依撇了撇嘴，「那群洋鬼子，一聽說蘇哥你要用一項『基礎靈能壓縮技術』換他們在太空軌道上的三顆低軌通信衛星，眼睛都紅了。」

「三顆衛星換一項技術，他們賺翻了。」

蘇晨語氣平淡。對於他來說，所謂的靈能壓縮，不過是將異界那些隨處可見的「碎靈石」進行現代工業化的脫水處理罷了，成本幾乎為零。

但對於地球這些極度缺乏高效能源、且正在被「世界重疊」後的環境壓制得喘不過氣來的科研團隊來說，這簡直就是第二次工業革命的火種。

就在這時，蘇晨腰間的通訊器振動了一下。

「蘇晨，有人想見你。不是為了生意，是為了……老趙。」

林若溪的聲音透著一種前所欲有的嚴肅。

蘇晨的手指猛地收緊，眼底那抹慵懶瞬間消失不見。

「老趙？他怎麼了？」

「他在蛇巷的雜貨店門口，被幾個不明身分的黑衣人帶走了。對方留下了話，說想想要老趙活命，就拿你手中的那枚『破曉徽章』去換。」

林若溪急促地說道，「我已經派了一隊人馬趕過去，但對方的實力極強，我們的人……連一招都沒接住。」

「一招都接不住？」

蘇晨的眼神徹底冰冷了下來。天衡局的行動組雖然不算極致，但好歹也是二階、三階的覺醒者，能一招秒殺，說明對方的等級至少在四階巔峰，甚至是……

「不用追了。林若溪。」

蘇晨跨上了悍馬，引擎發出一聲如野獸般的狂吼，「讓所有人散開，別去送死。」

「依依，守好家。」

蘇晨扔下這四個字，黑色悍馬便化作一道殘影自，在那紫色障壁自動開啟的一瞬間，如同一枚黑色的利箭，瘋狂地衝向了臨海市那迷宮般的弄堂深處。

蛇巷。

這裡雖然距離外灘不遠，卻像是被時光遺忘的角落。

當蘇晨趕到雜貨店門口時，只見那扇漆黑的木門早已破碎不堪，店內原本擺放整齊的過期可樂和乾脆麵散落一地。老趙平時最愛躺的那張藤椅，此時斷成兩截，地面上有一灘極其刺眼的暗紅色血跡。

蘇晨俯下身，指尖在那血跡上輕輕一抹，隨後送入戰術風鏡的感測器中。

```
  【DNA 序列掃描中……】
  【匹配目標：趙德厚（老趙）。】
  【檢測到異常：血液中含有極強的靈力反噬痕跡。】
  【偵測到殘留氣息：大夏國·極光實驗室·特種封印術。】
```

「封印術？」

蘇晨的心臟劇烈地跳動著。

老趙一個開雜貨店、整天只會吹牛逼的中年油膩男，竟然會牽扯到 20 年前那場禁忌的實驗室封印術？

聯想到老趙平日裡那種看似圓滑實則深藏不露的處事風格，以及他對自己那種近乎「引導」的照顧……

「老趙，你到底是誰？」

蘇晨緩緩站起身，看著雜貨店深處那張掛在牆上的、雖然佈滿灰塵卻依然莊重的全家福。照片角落裡，一個身穿老式研究服、面容略顯生澀的年輕人，隱約能看見老趙當年的影子。

他的手中，拿著的正是那枚被三劍貫穿齒輪的……破曉徽章。

「當年的破曉小組……竟然還有你。」

蘇晨握緊了拳頭，就在這時，杂貨店破舊的電話機突然「鈴、鈴、鈴」地響了起來。

蘇晨接起電話，聽筒那頭傳來了一個沙啞、冷酷、卻又帶著幾分瘋狂的聲音：

「蘇晨，你來晚了。」

「想要救你的導師，那就來臨海市郊的『廢棄化工廠』。記得帶上那枚領主芯片。」

「否則，三十分鐘後，你只能看到一堆被封印炸碎的肉泥。」

「喀查。」

電話掛斷。

蘇晨冷冷地看著那一地的廢墟，背後的紫色巨劍發出一陣瘋狂的錚鳴。

「敢動我的人……」

「既然你們想看萬界交易商的真本事，那我就讓這臨海市的夜色……徹底染成血色！」

下一秒。

黑色悍馬化作一道黑色閃電，在那無人的小巷中劃出了一道足以切開空氣的恐怖音爆雲。
