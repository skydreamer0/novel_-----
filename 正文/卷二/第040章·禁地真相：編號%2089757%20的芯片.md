# 第040章 · 禁地真相：編號 89757 的芯片

萬魔谷的慘敗，僅僅發生在不到一炷香的時間內。

當毒骨真君那引以為傲的魔修肉身，在經歷了「極致低溫」與「高頻劍意」的交替轟炸後，如同脆弱的瓷器般在空中崩碎時，整片萬劍崖外圍的秩序，被徹底改寫了。

「蘇兄，全數清理完畢。」

葉孤楓落在蘇晨身側，手中的斷劍依舊散發著淡淡的熱氣。那些平日裡不可一世的骨船，此刻早已化作了一堆毫無靈氣的木渣。

但蘇晨卻沒看那些戰利品一眼。

他正低頭看著戰術風鏡上那瘋狂閃爍的紅色警報。

「依依，葉兄，守住這裡。如果我半個小時沒出來，立刻啟動驛站的『自毀程序』，帶著弟兄們撤回大門後面。」

蘇晨的語氣前所未有的凝重，這讓剛沉浸在勝利喜悅中的眾人心中一凜。

「蘇哥，你要進去那裡？」葉依依指著前方雷暴的核心——那座被紫色雷霆徹底封鎖的神祕山谷。

蘇晨沒有回答，只是拍了拍腰間那個裝滿了「科技裝備」的黑色掛包，身形一晃，便化作一道殘影沒入了那足以殺死四階修仙者的雷幕之中。

「嗡——！」

踏入雷域核心的一瞬間，蘇晨感覺自己像是進入了一個巨大的、無聲的微波爐。

周圍的空氣因為過度的電離而散發著一股刺鼻的臭氧味，每一寸皮膚都在這高密度的電荷中叫囂戰慄。

系統面板在他的視網膜上拼命跳動：

```
  【當前輻射值已超標 500%！】
  【防護服有效時間剩餘：15 分鐘。】
  【偵測到目標物脈衝增強……編號 89757 數據接口已準備連結。】
```

在那萬千雷霆的洗禮下，蘇晨終於來到了一座由幾十柄百米高的巨型神劍組成的「石陣」中央。

而在那劍陣的最中心，靜靜地躺著一個讓他心跳幾乎停止的物體。

那是一個只有巴掌大小、外殼由某種不知名黑色合金打造的圓柱形密封筒。雖然筒身已經佈滿了被雷電反覆轟擊產生的碳化痕跡，但那一排精緻的鋼印編號，在此刻卻如烈日般灼燒著蘇晨的眼睛：

【D-89757 · 跨位面量子黑匣子（備份）】
【歸屬地：大夏國·極光實驗室】

「這就是……這就是爸媽留下的東西？」

蘇晨顫抖著手，將黑匣子拿了起來。

就在這一瞬間，一道冰冷、卻又帶著幾分熟悉的機械化女聲，直接在蘇晨的腦海中響起：

『掃描到合法繼承人 DNA 序列……生物識別通過。』
『正在解鎖 20 年前的最後一封語音信件。』

雷暴，在這一刻詭異地靜止了。

蘇晨的意識進入了一片白茫茫的空間，在那空間的盡頭，一對身穿白色研究服、眼神疲憊卻溫柔的身影，正隔著時空對他微笑。

『晨晨……如果你看到了這個，說明你已經長大了。很抱歉，在你的童年裡，我們選擇了遠行，去修補這地平線上最大的裂痕。』

『聽著，這不是什麼神仙世界，這裡是一個早已崩崩潰、正在「收縮」的次維度垃圾場。那些所謂的靈氣，不過是高級文明洩漏的「熵能源」。』

『我們已經找到了關閉連通門的方法，但代價是……我們必須留在「那邊」。』

『代價是，我們可能永遠無法與你在同一個時空相遇。』

畫面開始劇烈顫抖，那個女人的聲音帶上了哭腔：

『別過來……蘇晨！千萬別過來！那個編號 89757 的芯片裡隱藏著兩界平衡的最後座標，也是這世界唯一能拯救地球的鑰匙。快去臨海市的……』

「滋啦——！」

語音信件再次斷裂，黑匣子在那強烈的雷霆衝撞下發出了一聲哀鳴，隨後徹底化作了一堆金屬廢屑。

蘇晨靜靜地站在原地。

雷暴重新降臨，瘋狂地傾瀉在他的避雷裝備上，但他卻像是毫無察覺。

他緊緊握著手中那枚發燙的、寫著 89757 的微小芯片。

這不是修仙。

這是一場跨越了兩代人、兩個維度的、關於拯救與生存的巨大陰謀。

「什麼狗屁狗屁修仙，什麼萬界交易……」

蘇晨緩緩抬起頭，眼底深處，原本那抹隨性的慵懶早已徹底消失。取而代之的是一種足以屠盡漫天神佛的冷冽與執著。

「鬧了半天，這整個位面，不過是個裝滿了垃圾的實驗室。」

「而你們，居然敢拿我的父母當『修補材料』。」

蘇晨嘴角勾起一抹極度瘋狂的弧度。

「系統，消耗剩餘的所有靈能點，解鎖[跨位面定位權限]。」

「不管那道門在哪裡，不管接下要面對的是什麼……」

「我蘇晨，都要把他們，平平安安地……接、回、家！」

```
  【警告：檢測到宿主精神波動異常。】
  【系統正在強力適應：[位面領主模式] 解鎖中……】
  【當前主線變更：從「萬界交易」改為「兩界歸一」。】
```

禁地深處，傳來了一聲驚天動地的劍鳴。

那不是求救，而是宣戰。

萬界交易商的傳奇，從這一秒開始，正式跨入了那波瀾壯闊、也最為血腥瘋狂的第二幕。
