# 第045章 · 「萬界拍賣行」首拍

三日後。臨海市西郊，原「極光工業園區」。

這裡在此刻早已改頭換面，原本破舊的廠房被一層極具科幻美感的、發光的納米外殼所覆蓋。园區周圍，不僅有天衡局最精銳的部隊層層把守，更有無數道看不見的、由蘇晨親手佈置的劍意鎖鏈在虛空中交織。

幾十架印有各國國旗的高級私人直升機，正排隊降落在臨時開闢的草坪上。

「那就是蘇晨的地盤嗎？簡直像是從科幻電影裡搬出來的。」

一名身穿手工西裝、在歐美金融圈呼風喚雨的白人大鱷，此刻卻像是剛進城的土包子，眼神中充滿了震撼與敬畏。

拍賣大廳內。

林若溪身穿一套剪裁得極其得體的銀灰色晚禮服，優雅地站在拍賣台中央。她的左手腕上戴著一個微型的全息控制器，身後則是那面巨大的、展示著各種物品參數的虛擬屏幕。

「歡迎各位來到『萬界拍賣行』。」

她的聲音清冷且自信，透過特製的靈力擴音系統，傳遍了整個會場。

「今天，我們不拍賣古董，也不拍賣土地。我們拍賣的……是未來。」

台下座無虛席。這裡聚集了全球最富有、最有權勢的一群人。他們中有為了續命而來的百歲財閥，有為了提升國力而來的參謀長，更有各國隱藏已久的強大覺醒者。

「廢話不多說。請看第一件拍品。」

隨著林若溪的手勢，拍賣台中央的一根圓柱體緩緩升起，上面放著一個透明的密封艙。艙內，一顆通體金黃、表面隱約有雷紋游走的果實，正散發著一種讓人體每一個細胞都為之歡呼雀躍的異香。

「五行靈果·鑠金。功效：強行優化人體基因序列，拓寬能量通道。根據我們的實驗室數據，服用此果後，普通人的預期壽命將增加至少三十年，且具備覺醒金系能量的機率。」

全場屏息。

「起拍價：二等靈石礦脈一條（或同等價值的稀有金屬探明儲量）。」

「我出兩條礦脈！」

一名來自澳洲的礦產大亨迫不及待地站起身，那一臉狂熱的樣子，彷彿手裡的兩條礦脈只是換一包可樂的零錢。

「三條！外加我們在非洲的核心稀土權限！」另一名歐洲的公爵咬了咬牙，大聲吼道。

……

蘇晨此時正坐在二樓的貴賓休息室內。

他一手抓著一袋剛拆開的衛龍辣條，一手握著平板電腦，看著後台不斷跳動的資源回報。

「嘖，這群人是真有錢啊。」

蘇晨搖了搖頭，「一袋子在萬劍崖沒人要的爛果子，洗乾淨了居然能換幾條礦脈。林若溪這女人，天生就是個奸商。」

```
  【成功建立全球貿易連鎖（初級）。】
  ─────────────
  當前獲取資源：
  - 稀有稀土元素：3,000 噸。
  - 特種航太合金技術：4 套。
  - 核心城市地契：12 份。
  ─────────────
```

「蘇哥，這僅僅是個開始。」

身穿一套改良版青雲劍袍、顯得英氣逼人的葉依依，此時正安靜地站在蘇晨身後。她是蘇晨特意從異界接回來擔任「迎賓大將」的，同時也是為了讓地球的科研人員能近距離觀察「完美修煉體質」。

「等會兒那顆『洗髓丹』上場，這群人估計連靈魂都想抵押給你。」葉依依抿嘴笑道。

果然。

當最後一件壓軸拍品——由蘇晨提純後的「九轉仙靈果」出現在台上時，整個會場的氣氛徹底沸騰到了頂點。

「這顆果實，能完全瓦解人體內的所有惡性腫瘤。是的，你們沒聽錯，是所有。」

林若溪的聲音帶著一絲誘惑，「並且，它能讓一名瀕死的三階覺醒者，瞬間突破至四階。只有一顆。」

「起拍價：不設限。我們只接受蘇晨先生指定的……『神祕遺蹟殘件』，或者是在座各位家族傳承中那些『無法解釋的山海古物』。」

這一招，正是蘇晨的計策。

他在地球不缺靈石，不缺錢。他缺的，是那些可能與父母失蹤、與編號 89757 設備相關的遠古線索。

全場陷入了短暫的寂靜。

隨後。

一名一直坐在角落、身披黑袍的老者缓缓站了起來。他的聲音沙啞，卻帶著一種讓空氣凍結的冷冽：

「我這裡有一份從崑崙深處取得的、散發著 89757 脈衝頻率的金屬殘片。我拿它，換這顆果實。」

蘇晨握著辣條的手，猛地僵住。

他透過二樓的單向玻璃，死死地盯著那名黑袍老者手中的東西。

那是一塊被高溫熔融過的、呈現出詭異波浪狀的黑色徽章。上面赫然刻著一個圖案——那是一個被三柄利劍貫穿的、正在燃燒的齒輪。

這圖案，他在父母的實驗筆記中看過。

那是二十年前，大夏國極密計畫中，被蘇晨父母親手負責的……「破曉小組」的標識！

「成交。」

蘇晨的聲音，透過林若溪耳邊的頻道，帶著一股不容置疑的果決落了下來。

這場名動全球的拍賣會，在這一刻，不僅確立了蘇晨「物資霸主」的地位，更正式拉開了那塵封二十年的、兩界陰謀的最後大幕。

夕陽西下。

當所有大佬帶著如獲至寶的表情離去時。

蘇晨站在倉庫頂端的露台上，看著手中那枚帶血的徽章，眼神深邃得如同暴風雨前的海面。

「破曉小組……」

他喃喃自語，「既然你們還有人活著，那就別怪我這快遞員……親自上門去『派送』正義了。」
