# 第042章 · 一劍斷江，震撼全球

臨海市外灘。

江風依然在咆哮，但那股讓人肝膽俱裂的暴戾氣息，卻已經隨著那頭被切成萬塊的九幽噬骨龜，徹底煙消雲散。

蘇晨站在堤壩邊緣，手中的可樂罐發出「滋——」的一聲最後的抽吸。

在他的腳下，林若溪正仰著頭，那張美得讓人屏息的臉龐上，此時寫滿了荒誕與震撼。她看著眼前這個男人，他背後的巨劍在吸收了巨獸的精血後，正發出一種類似於引擎運轉的低沉鳴叫。

「蘇晨……你……這半個月你到底去了哪裡？」

林若溪的聲音有些顫抖，不知道是因為傷口的劇痛，還是因為眼前這超越認知的場面。

「去了一個快遞費比較貴的地方，順便進了點貨。」

蘇晨隨手將可樂罐捏扁，丟進了旁邊廢墟中的垃圾桶。他緩步走到林若溪面前，不顧對方下意識的戒備，直接蹲下身，手掌覆蓋在了她被毒素侵蝕的左肩上。

「別動，會有點癢。」

林若溪剛想反駁，卻感覺一股充滿了那種……如春雨般潤物無聲、卻又蘊含著極致純淨生命能量的靈力，瞬間衝入了她的傷口。

那是蘇晨在劍修世界，利用「五行靈果」和「現代生物過濾法」提純後的極品靈露精華。

「嘶——！」

林若溪倒吸一口冷氣，她驚恐地發現，那些連天衡局醫療中心都束手無策的「九幽毒素」，在此刻竟然像是雪遇殘陽般迅速消融，粉嫩的新肉在不到十秒鐘內便重新生長出來，甚至連一丁點疤痕都沒留下。

「這……這是神蹟嗎？」周家明在一旁看得眼珠子都要掉出來了。

他以前一直覺得蘇晨只是個運氣好點的「覺醒者」，但現在，他感覺自己像是在看一位降臨凡間的神明。

與此同時。

天空中的全球直播畫面，正將蘇晨的近景特寫發送到世界各地的屏幕上。

紐約、倫敦、京都、倫巴……無數國家的避難中心內，原本絕望的民眾看著這個一臉淡定、治癒如神的東方青年，爆發出了前所欲有的歡呼與祈禱。

「發現了嗎？」

蘇晨此時卻抬起頭，看著遠處依然在不斷扭曲的暗紅雲層。

「什麼？」林若溪剛站起身，還有些不適應。

「這片雲層後面，還有幾隻大點的蒼蠅正準備過來。」

蘇晨眼神微凝，他的戰術風鏡上，正瘋狂跳動著一串串讓普通科研人員會當場崩潰的數據。

```
  【警告：次維度裂縫正在二次擴大。】
  【偵測到五型生命體反應（准神級）。】
  【建議：立即進行「大範圍能量擾動阻斷」。】
```

「大隊長，借你的兵用一下。」

蘇晨沒理會林若溪的疑惑，他身形猛地拔地而起，直接站在了江面中心的那根殘存的航道浮標上。

「萬劍歸宗·廣域共振切割！」

蘇晨單手持劍，猛地將紫色巨劍插向了腳下的黃浦江。

「轟——！」

整條寬闊的黃浦江，在這一刻竟然像是被神靈從中間硬生生地切開了一般！

兩側的水牆高達百米，露出了江底那深埋已久的淤泥與殘骸。而在那兩側水牆的中間，無數道透明的、由高純度靈能凝聚而成的劍氣網格，竟然在虛空中織就了一張遮天蔽日的巨網。

「他在幹什麼？他在切開江水？！」

「不！你看那天上！」

隨著一名市民的驚叫。

只見天空中那層厚重的、試圖再次投放怪物的暗紅色雲層，在那透明劍網的切割下，竟然像是被手術刀切開的膿瘡，大片大片的暗紅能量崩潰消散，露出了背後那清冷湛藍的、屬於地球的星空。

原本蠢蠢欲動的幾道恐怖氣息，在感受到蘇晨這「一劍斷江、切開雲層」的威懾後，竟然像是受驚的野獸，忙不迭地縮回了裂縫深處。

雲散，雨停。

金色的陽光，重新灑在了滿目瘡痍的臨海市外灘上。

蘇晨站在乾涸的江底中心，背對著陽光，手中的巨劍正在緩緩消融在虛空之中。那一副不可一世的、卻又極其疲憊的神情，透過攝像機鏡頭，深深地刻在了全球幾十億人的腦海裡。

「沒事了。」

蘇晨回頭，對著堤壩上的林若溪招了招手。

「大隊長，記得幫我跟局裡請個假。這次出差太久，我的快遞積分估計都要扣光了。」

他說完，身形一晃，竟然就這麼在那眾目睽睽之下，消失在了漫天金光之中。

外灘邊，林若溪呆呆地看著那道逐漸癒合的江水，以及那空無一人的浮標，良久，才露出了一個既無奈又充滿了好奇的苦笑。

「蘇晨……你這個混蛋……到底成了什麼樣的高手啊。」

這一天。

全球各國政府的最高機密檔案中，同時出現了一個最高等級的紅色條目：

【代號：降臨者·蘇晨。性質：不可觀測之神級戰力。建議對策：傾舉國之力招攬，嚴禁任何挑釁行為。】

而這一切的主角，此時正坐在蛇巷雜貨店的三樓露台上。

他手裡拿著一根剛從冰櫃裡翻出來的老冰棍，看著手中那枚寫著 89757 的芯片，眼神深邃得如同黑洞。

「第一階段，回隊。接下來……該是算總帳的時候了。」
