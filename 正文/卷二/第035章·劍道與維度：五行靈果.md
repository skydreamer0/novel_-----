# 第035章 · 劍道與維度：五行靈果

血刀門的寶庫。

這座位於半山腰、被陣法層層覆蓋的石室，此刻門戶大開。

封萬安戰戰兢兢地帶路，每一步都走得極其艱難。他親眼看著蘇晨手裡那個能噴火的「打火機」，隨意燒毀了他們珍藏幾十年的幾根「困靈鎖」，那種對規則的無視，讓他徹底崩潰了。

「蘇哥，這裡好像有很多被污染的靈果。」

葉依依走進石室，眉頭微皺。

只見石架上，擺放著幾十顆晶瑩剔透、內部卻隱約有黑氣纏繞的果實。這些果實散發著一種極其微弱、卻又充滿了生命力量的波動，正是萬劍崖這片區域特產的「五行靈果」。

「蘇兄，可惜了。」
葉孤楓嘆了口氣，眼神中閃過一抹肉痛。

「這些五行靈果本是劍修錘鍊肉身、拓寬經脈的聖品。但因為魔霧的滲透，裡面混合了太多的戾氣和濁質。若強行吞服，極易導致走火入魔。在這禁地附近，只有那些大型宗門的煉丹師，用珍貴的『清靈散』洗滌七天七夜，才能勉強取出一半的功效。」

蘇晨卻是眼睛一亮。

他快步上前，抓起一顆淡金色的靈果，在手裡掂了掂。

```
  【獲取高級材料：五行靈果（受污染狀態）。】
  【偵測到大量非靈子結構雜質，建議使用「高週波超聲波清洗儀」或「工業級離心分離機」處理。】
  【系統小撇步：地球的『強鹼性洗潔精』配比特定比例的生理鹽水，可初步中和其中的部分負能酸性。】
```

蘇晨咧嘴一笑。

「洗滌？那太浪費時間了。」

他從背後的包裡，慢悠悠地摸出了一瓶在超市只要九塊九、還贈送一塊洗碗布的藍色液體。

瓶身上，赫然印著【白貓洗潔精·強效去油】。

「蘇哥……這是什麼？」
葉依依好奇地歪著腦袋，看著蘇晨熟練地從儲物包裡倒出一個塑料臉盆，然後接了半盆清水，滴入這藍色液體。

「這叫『九轉濯靈液』，專門用來清理世間一切污濁。」

蘇晨隨口忽悠，一邊挽起袖子，將那幾十顆被視為雞肋的五行靈果直接倒進了臉盆。

「滋——！」

當洗潔精與靈果表面的魔霧殘留接觸時，原本頑固的黑氣竟像是遇到了天敵，發出一陣微弱的滋滋聲，隨後化作一圈圈白色的泡沫浮上水面。

葉孤楓和封萬安在一旁看得呆若木雞。

那是連金丹期強者都感到頭疼、甚至需要耗費本源真元才能驅散的魔氣。

結果，這個男人……他在幹什麼？他在搓？他在用手洗？！

「差不多了。」

蘇晨將洗乾淨的靈果撈出來，隨手遞給了葉依依。

「嚐嚐看，有沒有土腥味。」

葉依依咬了咬嘴唇，她絕對相信蘇晨。

她拿起那顆原本被黑氣纏繞、現在卻晶潤得宛如藍寶石般的鑠金果，輕輕咬開。

「咔嚓！」

一股清醇、霸道、且不帶任何雜質的純淨金系靈能，瞬間順著她的喉嚨直衝四肢百骸！

那一瞬間，葉依依原本就已經恢復了大半的經脈，像是被一股暖流徹底拓寬了一倍。她周身原本細碎的劍氣，在此刻竟隱隱有了一種沉穩、凝練的「重劍」氣息。

「這……這功效，簡直比宗門記載的完美品相還要高出三成！」

葉依依驚叫出聲，眼神中充滿了難以置信。

因為洗潔精的高分子活性劑，竟然在無意中剝離了靈果表皮那層阻礙吸收的植物纖維。

蘇晨看著那一臉呆滯的封萬安，轉頭對葉孤楓說道：

「葉校長，這些果子歸你們了。一人吃三顆，剩下的給那幾個小劍童分了。」

「另外……封長老。」

蘇晨看向縮在角落的封萬安，語氣變得有些陰森。

「剛才那招閃光彈，只是給你一個警告。如果你不想讓這座血刀門徹底從萬劍崖消失，那從明天開始，你要幫我在這附近搜羅這種『廢棄』的靈果和古籍，每一千斤，我可以換你一箱餅乾和兩瓶二鍋頭。」

「你有意見嗎？」

封萬安渾身一個激靈，猛地磕頭：

「沒意見！絕對沒意見！封某定當竭盡全力！」

一千斤廢棄果子換一箱餅乾？

這特麼簡直是白給啊！

在封萬安眼裡，蘇晨簡直就是個揮金如土、卻又掌握著神聖力量的大慈善家（邪魔）。

蘇晨看著石室內，那些欣喜若狂、正大口吞噬靈果的少年劍修們，心中卻在默默計算著：

這波收穫了這片區域的地圖，收編了一個地頭蛇勢力。

更重要的是，他發現了「地球工業化學品」對異界資源的奇效。

這哪裡是做生意，這簡直是開著挖土機在聚寶盆裡撈。

```
  【成功建立第一條區域供應鏈。】
  【當前聲望：萬劍崖外圍·物資霸主。】
  【下一步建議：解鎖系統 Lv.3，開啟「區域定位點自建系統」。】
```

蘇晨仰起頭，看著石洞外那翻滾的魔霧，嘴角噙著一抹極其欠扁的、優越感拉滿的笑容。

他知道，真正的跨界傳奇，才剛剛完成第一幕。

很快，這裡就不再是禁地，而是他的後花園。

「走，依依，下一站……咱們去看看那顆刻著地球編號的芯片碎片。」

在那石室的更深處，一個被血刀門當成普通鎮紙的奇異物體，正隱隱發出一陣只有蘇晨能感應到的、來自另一個文明的微弱脈衝。
