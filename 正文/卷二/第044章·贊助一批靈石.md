# 第044章 · 贊助一批靈石

這是一個註定會載入天衡局、乃至整個大夏國歷史的午後。

蘇晨留下的那一箱極品靈石，此刻正被幾十名全副武裝的「龍魂特種部隊」嚴密護送進了地下的超級保險庫。

但在這保險庫的外面，另一場風暴正在王震海的辦公室裡醞釀。

「你真的決定了？林隊長？」

王震海看著桌子上那份剛剛遞上來的辭職報告，手指不自覺地顫抖了一下。

「局長，這不是辭職，是轉換賽道。」

林若溪手裡握著蘇晨送她的那柄「萬年玄冰長劍」，冰涼的氣息順著她的掌心直衝腦門，讓她感到前所欲有的冷靜。

「蘇晨在西郊拿的那塊地皮，雖然名義上是『私人倉庫』，但以他手裡掌握的資源，那裡很快就會成為全球修煉資源的核心中轉站。如果天衡局只是想通過行政手段去管束他，結果只能是兩敗俱傷。」

「所以，你想去當他的……『物業管理員』？」王震海語氣有些複雜。

「不。我是去當他的『首席營營運官』。」

林若溪淡淡一笑，眼中閃過一抹銳利的精芒，「蘇晨雖然實力強大，但他對於地球現有的商業運作模式和官僚體系完全沒興趣。如果沒有一個懂行的人去幫他打理，那些聞到味兒的跨國財閥、野心家，會像蒼蠅一樣把他煩死的。」

「我去，就是為了幫他把這些蒼蠅拍死。順便……保證天衡局在未來的貿易序列中，永遠排在第一位。」

王震海沉默了良久，隨即無奈地嘆了口氣，拿起印章在報告上重重一蓋。

「既然如此……那就委託你了。若溪，天衡局給你保留一等調查員的所有權限。你跟蘇晨……一定要處理好關係。」

「合作愉快。局長。」

林若溪敬了個標準的軍禮，轉身離去。

……

臨海市，西郊。

原本是一片荒廢已久的工業園區，此時卻已經被一層淡紫色的、隱約可見的弧光屏障給徹底籠罩。

幾輛掛著「萬界快遞」牌照的黑色大貨車（其實是蘇晨利用系統魔改後的運輸設備），正安靜地停在寬闊的裝卸平台上。

蘇晨正穿著一件洗得發白的舊背心，指揮著幾個從系統招募出來的、面無表情的「自動木偶搬運工」（由靈石驅動的低級傀儡），正在搬運著幾箱散發著淡淡異香的綠色物資。

「這效率還是低了點啊……」

蘇晨摸著下巴，嘴裡啃著一根剛從冰櫃裡拿出來的夢龍雪糕。

```
  【亞空間中轉站狀態：營運中。】
  ─────────────
  當前庫存：
  - 污染五行靈果（已清洗）：30,000 單位。
  - 下品靈石原礦：100 萬 單位。
  - 基礎淬體丹藥：5,000 瓶。
  - 異界廢棄兵器材料：50 噸。
  ─────────────
```

「這麼多貨，光靠我一個人賣，得賣到猴年馬月去。」

就在這時，一陣清脆的高跟鞋踩地聲，從屏障門口傳來。

蘇晨抬起頭，看著那一身筆挺的職業套裝、戴著一副金絲眼鏡、顯得精明幹練到極點的林若溪，不由得愣了一下。

「大隊長，你這……制服誘惑？天衡局的任務升級了？」

「報告蘇先生。」

林若溪走到他面前，姿態優雅地遞上了一份包裝精緻的文件夾，「從今天起，我正式從天衡局辭職。如果您不介意，我將擔任您這座『萬界貿易行』的代理負責人。」

「辭職？」蘇晨張大了嘴，手裡的雪糕差點掉在地上，「不是，我說大隊長，你這公務員鐵飯碗說丟就丟？我這兒可沒五險一金啊。」

「您給出的那箱靈石，價值足以買下半個臨海市。」

林若溪推了推眼鏡，嘴角勾起一抹職業化的微笑，「而您手裡掌握的那些『廢物利用』的資源，如果經過適當的操作，將會重塑全球的貨幣體系。」

「作為回報，我只需要您提供足夠的修煉資源，保證我的實力不落後於這個時代。如何？」

蘇晨瞇著眼打量著林若溪，隨即發出一陣豪爽的狂笑。

「行！既然你這麼有眼光，那這攤子事兒就交給你了。」

他隨手一指身後那幾萬平米的倉庫區域，「那裡面全是貨，有治病的、有殺人的、還有能讓老太太跳广场舞的。你的第一項任務，就是在臨海市舉辦一場……讓全球都為之瘋狂的拍賣會。」

「我要讓那些所謂的財閥和強者知道，想變強？可以。拿靈石來，拿技術來，拿我想要的一切來！」

林若溪看著蘇晨那恣意狂傲的眼神，心跳不由得加快了幾分。

她知道，這個男人正在做的，已經不再是簡單的快遞生意。

他是在地球的西郊，親手打造了一個凌駕於所有國家、所有勢力之上的——萬界樞紐。

「明白。」

林若溪翻開文件夾，一支精緻的簽名筆在她指尖輕快地旋轉。

「名字我都想好了。就叫……『萬界首拍：靈啟時代』。」

「拍賣會的主打產品……」

蘇晨咧嘴一笑，從兜裡摸出了一顆在日光下散發著七彩光暈的、被洗潔精洗得發亮的五行靈果。

「就用這個吧。治癒一切癌症、重塑人體基因的……『九轉仙靈果』。」

這一天。

一則簡短卻如同深水炸彈般的邀請函，從臨海市西郊發出，瞬間出現在了地球上所有五百強企業CEO和各國首領的辦公桌上。

全球，在這一刻，屏息凝神。
